# 1. 常见别名、物理性质、CAS、溶解度
- 中文名：呋塞米
- 英文名：Furosemide
- 常见别名：速尿（Lasix）
- CAS号：54-31-9
- 分子式：C₁₂H₁₁ClN₂O₅S
- 分子量：330.74
- 物理状态：白色或类白色结晶性粉末
- 水中溶解度：极微溶（≈0.1 g/L，25℃）
- 有机溶剂：
  - 易溶于乙醇、甲醇
  - 微溶于丙酮
- pKa：约 3.9（弱酸性）
# 2. 化学性质、光热稳定性
- 化学类别：磺胺类袢利尿剂
- 酸性药物，在碱性环境溶解度↑
- 光稳定性：对光敏感，溶液需避光
- 热稳定性：常温稳定，高温长期可降解
- 注射液：需 pH 8–9 才能稳定存在
# 3. 生化特性
- 作用靶点：肾髓袢升支粗段
- 抑制转运体：Na⁺-K⁺-2Cl⁻ 协同转运蛋白（NKCC2）
- 结果：
  - Na⁺、Cl⁻、K⁺ 重吸收显著下降
  - 尿渗透压下降
  - 肾髓质高渗梯度被破坏
# 4. 适应症、药理毒理
- 适应症
  - 心力衰竭（急、慢性）
  - 各类水肿（肾病综合征、肝硬化、肾衰）
  - 高血压（尤其合并容量负荷者）
  - 急性肺水肿
  - 高钙血症（辅助）
  - 急性肾衰少尿期（选择性）
- 毒理特点
  - 容量耗竭型毒性
  - 电解质失衡主导
# 5. 药代动力学、起效时间
- 口服：30–60 min 起效，1–2 h 达峰，持续 6–8 h
- 静脉：5 min 起效，30 min 达峰，持续2–4 h
- 生物利用度：≈50%（个体差异大）
- 血浆蛋白结合率：>95%
- 代谢：少量肝代谢
- 排泄：肾脏（原形为主）
# 6. 常见剂量、给药方式
- 口服：20–40 mg / 次，1–2 次/日
- 静推：20–40 mg，可重复
- 最大剂量：
  - 口服：600 mg/d（重症）
  - 静脉：200 mg/次（ICU）
# 7. 副作用、药物过量
- 常见副作用
  - 低钾血症（⭐最常见）
  - 低钠、低镁
  - 低血容量、低血压
  - 高尿酸血症（诱发痛风）
  - 耳毒性（大剂量或静推过快）
- 过量表现
  - 严重脱水
  - 循环衰竭
  - 意识障碍
  - 心律失常
# 8. 常见同分异构体和类似物质
- 同类袢利尿剂：
  - 布美他尼（Bumetanide）
  - 托拉塞米（Torsemide）
  - 依他尼酸（非磺胺类）
- 生化差异：
  - 托拉塞米：半衰期更长，抗醛固酮效应
  - 布美他尼：效力更强，mg级
# 9. 在人体内的整体作用
- 快速、强效脱容量
- 降低前负荷、肺毛细血管压
- 改变肾髓质渗透梯度
- 长期使用激活 RAAS
# 10. 内分泌相关的激素
- ↑ 肾素
- ↑ 血管紧张素Ⅱ
- ↑ 醛固酮
- 间接 ↑ ADH
- 对性激素无直接作用
# 11. 对脂肪代谢的作用
- 无直接影响
- 间接：
  - 长期低钾 → 胰岛素抵抗↑
  - 水分减少 ≠ 脂肪减少
# 12. 对血压的作用
- 短期：显著降压（容量减少）
- 长期：
  - 单用降压效果有限
  - 常需联用 ACEI / ARB
# 13. 对消化系统的作用（急性）
- 恶心、呕吐
- 食欲下降
- 极少数：胰腺炎（继发电解质紊乱）
# 14. 对神经系统的调节机制和作用
- 低钠 → 意识模糊、抽搐
- 低钾 → 肌无力、腱反射减弱
- 耳蜗 NKCC 抑制 → 耳鸣、听力下降
# 15. 对生殖系统的作用
- 无直接抑制
- 间接影响：
  - 电解质紊乱 → 性功能下降
  - 长期低血容量 → 勃起功能受损风险↑
# 16. 对皮肤的作用
- 皮疹（磺胺结构）
- 光敏反应（少见）
- 紫癜（极少）
# 17. 过多或不足时的治疗药物
-  呋塞米过量
  - 等渗液体补充
  - 氯化钾 / 镁
  - 必要时螺内酯拮抗
- 呋塞米不足/耐药
  - 加用：噻嗪类（序贯阻断）、托拉塞米
- 女性非孕期差异：更注意低钠风险
# 18. 中医八纲辨证与五行归经
- 八纲：偏 寒、虚、下、阴
- 功效类比：利水渗湿、泻实水
- 五行归经：肾、膀胱
- 易伤：肾阳、脾阳
- 常配思路：黄芪、桂枝（防虚脱）

# 1. 常见别名、物理性质、CAS、溶解度
- 别名：尼克酸、烟碱酸、吡啶-3-羧酸、维生素B₃、Niacin
- 化学式：C₆H₅NO₂
- 分子量：123.11
- CAS号：59-67-6
- 外观：白色或类白色结晶性粉末
- 熔点：236–238 ℃（分解）
- 气味/味道：无臭，微酸
- 水中溶解度：易溶（≈15 g/100 mL，20 ℃）
- 有机溶剂：微溶于乙醇，几乎不溶于乙醚、氯仿
- pKa：≈4.8（羧酸）
# 2. 化学性质、光热稳定性
- 弱酸性芳香族羧酸
- 耐光性好
- 热稳定性强（烹饪损失小）
- 酸碱环境下较稳定
- 与强碱长期接触可发生盐化
# 3. 生化特性
- 是 NAD⁺ / NADP⁺ 的前体
- 参与：
  - 细胞呼吸
  - 氧化还原反应
  - DNA修复
  - 线粒体能量代谢
- 不可由人体大量合成（色氨酸→烟酸效率低）
# 4. 适应症、药理毒理
- 适应症：
  - 烟酸缺乏（糙皮病）
  - 调脂治疗（药用大剂量）
- 药理作用：
  - 抑制肝脏VLDL合成
  - 升高HDL-C
  - 降低TG、LDL-C
- 毒理：
  - 低毒
  - 长期大剂量可致肝毒性
# 5. 药代动力学、起效时间
- 吸收：小肠主动/被动吸收
- 血药峰值：30–60 min
- 代谢：肝脏甲基化
- 半衰期：≈1 h
- 排泄：尿液（N-甲基烟酰胺）
- 起效：
  - 潮红：15–30 min
  - 调脂：2–4周
# 6. 常见剂量、给药方式
- 营养补充：RDA（男）：16 mg NE / 天
- 药用调脂：
  - 起始：100–250 mg / 天
  - 常用：1–2 g / 天
  - 最大：≤3 g / 天
  - 口服（缓释制剂更常用）
# 7. 副作用、药物过量
- 常见副作用：
  - 皮肤潮红、灼热、瘙痒
  - 胃部不适、恶心
- 严重不良反应：
  - 肝功能异常
  - 高尿酸血症
  - 血糖升高
- 过量表现：
  - 剧烈潮红
  - 低血压
  - 肝损伤
# 8. 同分异构体与类似物
- 烟酰胺（Niacinamide）
  - 无潮红
  - 无明显调脂作用
  - 更偏向神经、皮肤修复
- 肌醇六烟酸酯
  - 潮红轻
  - 调脂作用弱、证据有限
# 9. 在人体内的整体作用
- 能量代谢核心维生素
- 维持皮肤、神经、消化系统完整性
- 改善脂蛋白谱
- 抗炎、抗氧化（间接）
# 10. 内分泌相关激素
- 间接影响 胰岛素敏感性
- 抑制脂肪组织 激素敏感性脂肪酶（HSL）
- 影响 肾上腺皮质激素代谢
# 11. 对脂肪代谢的作用
- ↓ 游离脂肪酸释放
- ↓ 甘油三酯
- ↓ LDL-C
- ↑ HDL-C（独特优势）
- 减少肝脂合成
# 12. 对血压的作用
- 急性：外周血管扩张 → 血压轻度下降
- 长期：无明确降压治疗作用
# 13. 对消化系统的作用（急性）
- 刺激胃黏膜
- 可致：
  - 胃灼热
  - 恶心
  - 腹胀
- 餐后服用可缓解
# 14. 对神经系统的调节
- 维持神经细胞能量代谢
- 缺乏可致：
  - 抑郁
  - 记忆障碍
  - 神经炎
- NAD⁺ 参与神经修复
# 15. 对生殖系统的作用
- 间接支持睾丸能量代谢
- 改善血脂 → 有利于性腺微循环
- 无直接促性激素作用
# 16. 对皮肤的作用
- 改善角化、皮炎
- 缺乏 → 糙皮病（三联征：皮炎、腹泻、痴呆）
- 潮红源于 前列腺素D₂释放
# 17. 过多或不足时的治疗药物
- 不足：
  - 烟酸
  - 烟酰胺（首选，副作用少）
- 过多（潮红/肝损）：
  - 阿司匹林（减潮红）
  - 停药 + 保肝药
- 与非孕产期女性对比：
  - 女性更易潮红
  - 肝毒性风险略高
  - 推荐更低起始剂量
# 18. 中医八纲辨证与五行归经
- 性味：酸、平
- 归经：肝、脾
- 八纲：偏补、非寒非热
- 功效类比：类似“益气养血、通络”
- 适合证型：气血不足、脾虚夹湿
- 不宜：肝火旺、湿热内盛者大量使用

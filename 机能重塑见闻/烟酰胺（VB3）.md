# 烟酰胺

- 另请参见：[烟酸（VB3）](烟酸（VB3）.md)

## 1. 常见别名、物理性质、CAS、溶解度

- 常见别名
  + 烟酰胺
  + 尼克酰胺
  + 维生素 B3（酰胺型）
  + Nicotinamide（NAM）
- 化学式：C₆H₆N₂O
- 分子量：122.12 g/mol
- CAS 号：98-92-0
- 物理性质
  + 白色或类白色结晶性粉末
  + 无臭，味微苦
  + 熔点：128–131 ℃
  + 不挥发
- 溶解度
  + 水：高度易溶（≈500 g/L，25 ℃）
  + 乙醇：可溶
  + 丙酮：微溶
  + 氯仿、乙醚：几乎不溶
  + 有机油脂中：不溶

## 2. 化学性质、光热稳定性

- 为 吡啶羧酰胺结构，化学性质温和
- 不具烟酸的游离羧基 → 无明显血管扩张
- 对光稳定
- 对热稳定（常规加工不易分解）
- 强酸强碱下可缓慢水解为烟酸

## 3. 生化特性

- 是 NAD⁺ / NADP⁺ 的直接前体
- 参与：
  + 氧化还原反应
  + ATP 合成
  + DNA 修复（PARP）
  + 线粒体功能
- 不激活 GPR109A 受体（区别于烟酸）

## 4. 适应症、药理毒理

- 适应症
  + 维生素 B3 缺乏（糙皮病）
  + 皮肤屏障修复（外用）
  + 炎症性皮肤病辅助
  + 高代谢状态（耗 NAD⁺）
- 药理特点
  + 抗炎
  + 抗氧化
  + 改善皮肤屏障
  + 维持细胞能量稳态
- 毒理
  + 安全窗宽
  + 长期大剂量可致肝酶升高

## 5. 药代动力学、起效时间

- 口服吸收快（小肠）
- 血药峰值：30–60 分钟
- 广泛分布于肝、肾、肌肉、脑
- 肝内转化为 NAD⁺
- 代谢物经尿排出

## 6. 常见剂量、给药方式

- 成年男性（口服）
  + 日常补充：15–30 mg / 日
  + 功能强化：100–500 mg / 日
  + 医疗剂量：≤1000 mg / 日（分次）
- 给药方式
  + 口服片剂 / 胶囊
  + 外用乳膏（2–5% 常见）

## 7. 副作用、药物过量

- 常见副作用（高剂量）
  + 恶心
  + 食欲下降
  + 轻度疲劳
- 罕见（>3 g/日，长期）
  + 转氨酶升高
  + 胰岛素敏感性下降（争议）
- 不产生
  + 潮红
  + 血管扩张  
- **与烟酸显著不同**

## 8. 同分异构体和类似物质

- 类似物质
  + 烟酸（Niacin）：强调脂代谢、潮红明显
  + NMN（烟酰胺单核苷酸）
  + NR（烟酰胺核糖）
- 生化差异
  + NAM → 直接参与 Salvage pathway
  + NMN / NR → 更偏 NAD⁺ 快速提升

## 9. 在人体内的整体作用

- 维持能量代谢
- 促进 DNA 修复
- 抗慢性炎症
- 抑制过度氧化应激
- 皮肤与神经系统获益明显

## 10. 内分泌相关激素

- 间接影响：
  + 胰岛素信号
  + 皮质醇节律（高剂量可能扰动）
- 不直接刺激性激素分泌

## 11. 对脂肪代谢的作用

- 不显著降脂（区别于烟酸）
- 高剂量可能：
  + 抑制脂肪动员
  + 轻度降低胰岛素敏感性（个体差异）

## 12. 对血压的作用

- 无明显升压或降压
- 不引起血管扩张
- 心血管刺激极低

## 13. 对消化系统的急性作用

- 对胃刺激小
- 空腹大剂量可能轻度恶心
- 明显优于烟酸耐受性

## 14. 对神经系统的调节机制

- 提供 NAD⁺ → 维持神经元能量
- 抑制神经炎症
- 有助于：
  + 抗疲劳
  + 抗脑雾
  + 神经修复支持

## 15. 对生殖系统的作用

- 间接支持精子能量代谢
- 高剂量 不提升雄激素
- 极高剂量可能抑制睾丸 NAD⁺ 平衡（动物证据）

## 16. 对皮肤的作用

- 外用核心作用
  + 增强角质层屏障
  + 降低经皮水分流失
  + 抗炎、抗痤疮
  + 改善色素沉着
- 口服：改善慢性炎症性皮肤状态

## 17. 过多或不足时的治疗

- 不足
  + 补充烟酰胺或烟酸
  + 女性（非孕产期）：剂量相同
- 过多
  + 停用
  + 肝酶监测
  + 不需特殊拮抗药

## 18. 中医八纲辨证与五行归经

- 性：平
- 味：淡
- 归经：脾、肝、肾（偏精微物质代谢）
- 八纲辨证：偏虚证调补，不助火，不伤阴
- 适合：
  + 气阴两虚
  + 久劳内耗型体质

## 一句话总结

- 烟酰胺 = 安全、稳定、偏“基础代谢维护型”的 NAD⁺ 前体不追求刺激性效果，但适合长期、稳态支持。

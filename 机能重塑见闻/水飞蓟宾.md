# 1. 基本信息
- 常见别名：水飞蓟宾、silybin、silibinin、奶蓟素主成分
- 来源：水飞蓟（Silybum marianum）种子提取物
- CAS号：22888-70-6（主要异构体混合物）
- 分子式：C25H22O10
- 分子量：约482.44
- 外观：淡黄色至棕黄色结晶性粉末
- 熔点：约163–165°C
- 溶解性：
  - 水：极微溶（<0.05 mg/mL）
  - 乙醇：可溶
  - 丙酮、DMSO：易溶
  - 脂溶性：中等（亲脂性较强）
# 2. 化学性质与稳定性
- 属于黄酮木脂素类化合物（flavonolignan）
- 含多酚羟基 → 强抗氧化能力
- 稳定性：
  - 光稳定性：中等，长期光照可降解
  - 热稳定性：较好（<100°C稳定）
  - pH稳定性：中性稳定，碱性条件下降解
# 3. 生化特性
- 强抗氧化（清除ROS、提高谷胱甘肽）
- 抑制脂质过氧化
- 调节细胞膜通透性
- 抑制炎症因子（TNF-α、NF-κB）
- 促进蛋白合成（肝细胞再生）
# 4. 适应症与药理毒理
- 适应症
  - 慢性肝炎（病毒性/酒精性/药物性）
  - 脂肪肝（NAFLD/NASH）
  - 肝硬化辅助治疗
  - 解毒（蘑菇毒素如鹅膏毒素）
- 药理作用：（肝保护核心机制）
  - 抗氧化
  - 抗炎
  - 稳定肝细胞膜
  - 抑制毒素进入肝细胞
- 毒性
- 毒性极低（LD50 > 2000 mg/kg）
- 长期使用安全性较高
# 5. 药代动力学
- 口服生物利用度低（20–50%）
- 吸收：肠道吸收有限（常制成磷脂复合物提高吸收）
- 分布：肝脏富集
- 代谢：肝脏葡萄糖醛酸化
- 半衰期：约6–8小时
- 排泄：胆汁为主
- 起效时间
  - 抗氧化作用：数小时内
  - 肝功能改善：2–4周
  - 明显结构改善：8–12周
# 6. 常见剂量与给药方式
- 常规剂量：140 mg，2–3次/日
- 高剂量（治疗性）：420–600 mg/日
- 剂型：
  - 口服片剂/胶囊
  - 磷脂复合物（如silybin-phytosome，吸收更高）
# 7. 副作用与过量
- 常见副作用
  - 轻度胃肠反应（腹泻、恶心）
  - 过敏（罕见）
- 过量
  - 极少严重毒性
  - 可能出现：腹泻、轻度头晕
# 8. 同分异构体与类似物
- Silybin A / Silybin B（主要活性成分）
- 相关化合物：
  - 水飞蓟素（silymarin，混合物）
  - 异水飞蓟宾（isosilybin）
- 生化差异
  - Silybin B：抗氧化更强
  - Isosilybin：抗肿瘤作用更突出
# 9. 在人体整体作用
- 核心靶器官：肝脏
- 系统性效应：
  - 抗炎
  - 抗氧化
  - 代谢调节
- 对慢性代谢疾病有辅助改善
# 10. 内分泌影响
- 对性激素：轻度抑制芳香化酶（弱）
- 对胰岛素：提高胰岛素敏感性
- 对皮质醇：间接降低炎症相关升高
# 11. 对脂肪代谢的作用
- 抑制脂肪生成（↓SREBP-1c）
- 促进脂肪氧化（↑PPAR-α）
- 改善脂肪肝
- 降低甘油三酯
# 12. 对血压的作用
- 轻度降压（间接）：
  - 改善内皮功能
  - 抗氧化
- 无直接α/β受体作用
# 13. 对消化系统的作用（急性）
- 促进胆汁分泌
- 保护胃黏膜
- 轻度促进消化
- 大剂量可致轻泻
# 14. 神经系统作用
- 抗神经炎症
- 抗氧化保护神经元
- 潜在神经保护（阿尔茨海默病研究中）
- 无显著中枢兴奋/抑制
# 15. 生殖系统作用
- 间接改善睾酮环境（通过减轻肝代谢负担）
- 抗氧化 → 改善精子质量
- 对性功能：轻度正向（非直接激素药）
# 16. 对皮肤的作用
- 抗氧化 → 抗光老化
- 抗炎 → 改善痤疮/皮炎
- 促进皮肤修复
# 17. 过多或不足的治疗
- 过量处理：停药 + 对症支持
- 肝损伤治疗对比
  - 男性常用：水飞蓟宾、还原型谷胱甘肽
- 女性（非孕）：基本相同
- 孕期：避免高剂量
# 18. 中医分析
- 性味：苦、凉
- 归经：肝、胆经
- 功效：
  - 清热解毒
  - 疏肝利胆
  - 活血化瘀
- 八纲辨证
  - 属“里证、热证、实证”
  - 适用于：湿热蕴肝、肝郁化火
- 五行
  - 属木（肝）
  - 调节“木郁土壅”（脂肪肝、代谢综合征）
# 总结（关键点）
- 水飞蓟宾本质是肝脏保护+抗氧化核心分子
- 不是激素类药物
- 对代谢综合征有辅助价值
- 在你关注的T/E2、脂肪、肝功能体系中属于“底层稳定器”

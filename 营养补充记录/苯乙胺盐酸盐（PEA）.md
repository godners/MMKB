# 1. 基本信息
- 中文名：苯乙胺盐酸盐
- 英文名：Phenethylamine hydrochloride
- 常见别名：β-苯乙胺盐酸盐、PEA·HCl
- 分子式：C₈H₁₁N·HCl
- 分子量：157.64 g/mol
- CAS号：156-28-5（盐酸盐）；64-04-0（游离碱）
- 性状：白色或类白色结晶性粉末
- 熔点：约217–221℃（分解）
- 溶解度：
  - 水中：易溶（盐酸盐形式）
  - 乙醇：易溶
  - 氯仿：微溶
  - 游离碱在有机溶剂中溶解度更高（乙醚、氯仿等）
# 2. 化学性质与稳定性
- 结构：芳香环 + 乙胺侧链
- 属于一级胺，具碱性，可形成稳定盐类
- 易被氧化（空气中长期暴露可变色）
- 对光较稳定
- 高温下分解
- 在强氧化剂存在下可发生氧化脱氨
# 3. 生化特性
- 属于内源性痕量胺（Trace amine）
- 主要通过激活TAAR1（Trace Amine-Associated Receptor 1）
- 促进多巴胺、去甲肾上腺素释放
- 通过MAO-B快速代谢
- 半衰期：约5–10分钟（极短）
# 4. 适应症与药理毒理
- 适应症
  - 非正式用途：情绪调节、认知增强
  - 某些国家作为膳食补充剂成分
- 药理作用
  - 中枢兴奋
  - 提升警觉性
  - 短暂增强情绪
- 毒理
  - 高剂量可引起：心动过速、血压升高、焦虑、失眠
  - 与MAO抑制剂合用可发生严重高血压危象
# 5. 药代动力学
- 口服后迅速吸收
- 首过效应显著
- 主要经MAO-B脱氨
- 代谢产物：苯乙酸
- 起效时间：10–30分钟
- 作用持续：30–90分钟
# 6. 常见剂量与给药方式（成年男性）
- 口服：100–500 mg（补充剂范围）
- 与MAO-B抑制剂合用时剂量需大幅降低
# 7. 副作用与过量
- 常见副作用
  - 心悸
  - 血压升高
  - 头痛
  - 易怒
- 过量表现
  - 高血压危象
  - 心律失常
  - 严重焦虑或惊恐发作
# 8. 同分异构体与类似物
- 同分异构体：无手性中心（本身无对映异构体）
- 类似物
  - 安非他明（α-甲基苯乙胺）：代谢更慢，中枢兴奋作用更强
  - 酪胺：结构类似，升压作用明显
# 9. 人体整体作用
- 快速短暂的中枢兴奋
- 情绪波动明显
- 对交感神经系统刺激较强
# 10. 内分泌相关
- 间接提高：
  - 去甲肾上腺素
  - 多巴胺
- 不直接作用于睾酮或雌激素
# 11. 对脂肪代谢
- 通过去甲肾上腺素促进脂解
- 提高基础代谢率（短暂）
# 12. 对血压
- 显著升压作用（剂量相关）
- 与MAOI合用风险极高
# 13. 对消化系统（急性）
- 轻度恶心
- 食欲下降
- 胃部刺激
# 14. 对神经系统
- 激活TAAR1
- 增强多巴胺释放
- 可诱发焦虑
- 长期高剂量可能导致受体下调
# 15. 对生殖系统
- 间接通过交感神经兴奋影响勃起
- 高剂量可能导致射精提前
- 不影响精子生成
# 16. 对皮肤
- 可能出现潮红
- 多汗
# 17. 过多或不足的处理
- 过多
  - β受体阻滞剂（如美托洛尔）
  - α受体阻滞剂（严重高血压时）
- 不足
  - 无临床“缺乏症”
  - 女性：更易出现焦虑与心悸，剂量通常更低
# 18. 中医分析
- 八纲辨证
  - 性属温
  - 归阳
  - 属“助阳升散”
- 五行归经
  - 入心、肝经
  - 偏助肝阳
  - 易助火生风
- 表现：
  - 实证偏多
  - 阴虚火旺者慎用

# 1. 基本信息
- 中文名：硫酸锌
- 英文名：Zinc sulfate
- 常见别名：白矾（旧称，易与明矾混淆）、皓矾
- 化学式：ZnSO₄（常见为七水合物 ZnSO₄·7H₂O）
- CAS号：
  - 无水物：7733-02-0
  - 七水合物：7446-20-0
- 外观：无色或白色结晶/粉末（七水合物为透明斜方晶体）
- 相对分子质量：
  - 无水物：161.47
  - 七水合物：287.54
- 密度（七水）：约1.97 g/cm³
- 熔点：
  - 七水合物：约100℃开始失水
  - 无水物：约680℃分解
- 味：强收敛性、苦味
- 溶解度
  - 水：易溶（25℃约57 g/100 mL水）
  - 乙醇：微溶
  - 丙酮：几乎不溶
  - 在无机酸中：易溶
  - 在碱性溶液中：生成氢氧化锌沉淀
# 2. 化学性质与稳定性
- 水溶液呈弱酸性（pH约4–5）
- 与碱反应生成Zn(OH)₂沉淀
- 与碳酸盐生成碳酸锌
- 受热逐步脱水
- 光稳定
- 在干燥环境中稳定，潮湿环境易吸湿结块
# 3. 生化特性
- 提供Zn²⁺离子
- 锌为多种酶（>300种）辅因子：
  - DNA聚合酶
  - RNA聚合酶
  - 碳酸酐酶
  - 超氧化物歧化酶
- 参与蛋白质合成、细胞分裂
- 参与睾酮合成与芳香化酶调节
# 4. 适应症（成年男性）
- 锌缺乏症
- 免疫功能低下
- 男性不育（精子活力低下）
- 慢性腹泻
- 痤疮辅助治疗
- Wilson病辅助排铜（抑制铜吸收）
# 5. 药理与毒理
- 药理
  - 促进睾丸Leydig细胞功能
  - 抑制芳香化酶（轻度）
  - 抗氧化
  - 稳定细胞膜
  - 抗炎作用
- 毒理
  - 急性大剂量 → 胃肠刺激
  - 慢性过量 → 铜缺乏、贫血
  - LD50（大鼠口服）：约1710 mg/kg
# 6. 药代动力学
- 口服吸收率：20–40%（空腹更高）
- 与植酸、钙、铁竞争吸收
- Tmax：1–3小时
- 血浆锌蛋白结合率：60–70%
- 主要经粪便排泄
- 半衰期：约2–3天（体内周转）
# 7. 常见剂量
- 预防缺乏：15–30 mg/日（元素锌）
- 治疗缺乏：30–50 mg/日
- 男性生殖支持：30–60 mg/日
- 给药方式：口服片剂、胶囊、溶液  
**硫酸锌220 mg ≈ 元素锌50 mg**
# 8. 适应症与不良反应
- 同分异构体与类似物
- 无同分异构体
- 常见锌盐：
  - 葡萄糖酸锌（吸收温和）
  - 枸橼酸锌（生物利用度较高）
  - 氧化锌（吸收较低）
- 生物利用度：枸橼酸锌 ≥ 葡萄糖酸锌 > 硫酸锌 > 氧化锌
# 9. 体内整体作用（成年男性）
- 维持睾酮正常分泌
- 提高精子数量与活力
- 抑制炎症
- 促进创面愈合
- 调节免疫系统
# 10. 内分泌相关
- 提升LH对睾丸刺激反应
- 维持睾酮合成酶活性
- 缺锌 → 睾酮下降、E2比例升高
# 11. 对脂代谢
- 轻度改善胰岛素敏感性
- 缺锌时甘油三酯升高
- 适量补充可改善脂肪肝氧化应激
# 12. 对血压
- 轻度利钠作用
- 参与NO合成调节
- 长期缺乏可加重高血压风险
# 13. 对消化系统（急性）
- 刺激胃黏膜
- 空腹大剂量易恶心、呕吐
- 可致腹泻
# 14. 神经系统
- 调节NMDA受体
- 参与突触可塑性
- 缺锌 → 抑郁、认知下降
- 过量 → 神经抑制、嗜睡
# 15. 生殖系统
- 促进精原细胞分裂
- 稳定精子DNA
- 参与前列腺液形成
- 高剂量长期使用可抑制铜吸收 → 继发性贫血
# 16. 皮肤
- 抗炎
- 减少痤疮
- 促进伤口愈合
# 17. 过多或不足的治疗
- 锌缺乏： 首选口服锌盐
- 男性剂量高于非孕期女性
  - 男性推荐摄入量：11–15 mg/日
  - 女性：8–10 mg/日
- 锌过量
  - 停药
  - 补铜（硫酸铜）
  - 对症支持
# 18. 中医分析
- 性味：酸、涩、微寒
- 归经：肝、肾经
- 功效：收敛止泻、敛疮生肌
- 八纲：属阴性收涩之品
- 五行：金（对应肺）

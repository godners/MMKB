# 1. 常见别名、物理性质、CAS、溶解度
- 通用名：酚苄明
- 英文名：Phenoxybenzamine
- 别名：苄氧胺、苯氧苄胺
- CAS 号：63-92-3
- 分子式：C₁₈H₂₂ClNO
- 分子量：303.83
- 外观：白色或类白色结晶性粉末
- 理化性质：
  - 弱碱性
  - 脂溶性较强
- 溶解度：
  - 水：极微溶
  - 乙醇、甲醇、氯仿：可溶
  - 有机溶剂（丙酮、苯）：易溶
# 2. 化学性质、光热稳定性
- 化学类型：β-卤代烷胺类
- 反应特性：
  - 可与α受体形成共价键（烷基化）
  - 属于不可逆α受体阻断剂
- 稳定性：
  - 对光较稳定
  - 常温稳定
  - 强酸、强碱下可降解
# 3. 生化特性
- 受体作用：非选择性 α₁ + α₂ 受体不可逆拮抗
- 阻断特点：
  - 作用持续时间 ≫ 血药浓度
  - 新受体合成前无法恢复功能
- 交感神经效应：
  - 抑制去甲肾上腺素缩血管作用
  - 抑制α₂负反馈 → 去甲肾上腺素释放↑
# 4. 应症、药理毒理
## 适应症
- 嗜铬细胞瘤（首选术前药）
- 特殊难治性高血压（少用）
- 交感神经过度兴奋状态（历史用途）
## 药理毒理
- 毒性主要来自：
  - 持续低血压
  - 反射性心动过速
- 致突变性/致癌性：
  - 动物实验显示潜在风险
  - 不推荐长期非必要使用
# 5. 药代动力学、起效时间
- 吸收：口服吸收良好
- 起效时间：口服后 1–2 小时起效
- 作用持续时间：3–7 天（功能性）
- 半衰期：
  - 血浆半衰期短
  - 药效由受体更新决定
- 代谢：肝脏
- 排泄：肾脏为主
# 6. 常见剂量、给药方式
- 嗜铬细胞瘤术前：
  - 起始：10 mg，2 次/日
  - 常用维持：20–40 mg/日
  - 最大可达：100 mg/日（分次）
- 给药方式：口服
# 7. 副作用、药物过量
- 常见副作用
  - 体位性低血压
  - 心动过速
  - 鼻塞
  - 疲劳、头晕
  - 射精障碍（逆行射精）
- 过量表现
  - 严重低血压
  - 休克
  - 持续性反射性心动过速
- 去甲肾上腺素对抗效果差（α被封闭）
# 8. 同分异构体与类似物
- 同分异构体：无临床意义构型差异
- 类似物：
  - 酚妥拉明（可逆、短效）
  - 多沙唑嗪（α₁选择性）
  - 哌唑嗪
- 对比特点：
  - 酚苄明 → 不可逆、作用最持久
  - 其他 → 可逆、安全性更高
# 9. 在人体内的整体作用
- 全身血管阻力 ↓↓↓
- 静脉容量 ↑
- 心输出量代偿性 ↑
- 交感张力被“去功能化”
# 10. 内分泌相关激素
- 间接影响：
  - 去甲肾上腺素 ↑
  - 肾素-血管紧张素系统激活
- 不直接作用于性激素或皮质激素
# 11. 对脂肪代谢的作用
- 交感α阻断 →
  - 脂肪组织血流 ↑
  - 脂解效应轻度↑
- 无明确临床代谢治疗价值
# 12. 对血压的作用
- 显著、持久降压
- 尤其降低：
  - 外周阻力
  - 站立位血压
- 易出现体位性低血压
# 13. 对消化系统的作用（急性）
- 肠道平滑肌松弛
- 胃排空略减慢
- 可出现：
  - 恶心
  - 轻度腹胀
# 14. 对神经系统的调节机制
- 中枢作用弱
- 外周交感抑制为主
- α₂阻断 → 去甲肾上腺素释放增加
- 易致：
  - 心悸
  - 焦躁感
# 15. 对生殖系统的作用
- 射精障碍（重要）
  - 精囊、输精管α受体被阻断
  - 易出现逆行射精或无射精
- 勃起功能：
  - 一般不增强
  - 个体可感血流变化
# 16. 对皮肤的作用
- 皮肤血管扩张
- 潮红
- 温热感
- 多汗减少或不明显
# 17. 过多或不足时的治疗（含女性对比）
- 过量处理
  - 卧位
  - 补液
  - 升压药首选 β激动 + 血容量扩充
  - 避免单用α激动剂
- 女性（非孕期）差异
  - 剂量相同
  - 更易体位性低血压
  - 乳腺充血感可能更明显
# 18. 中医八纲辨证与五行归经
- 八纲：
  - 表里：里
  - 寒热：偏寒（血管扩张）
  - 虚实：实证（肝阳、冲逆）
- 五行归经（类比）：
  - 肝经（血管张力）
  - 心经（心率、心悸）
- 中医类象：
  - 平肝潜阳
  - 通脉降逆

# 1. 常见别名、物理性质、CAS、溶解度
- 中文名：硝苯地平
- 英文名：Nifedipine
- 别名：心痛定（旧译名）
- CAS号：21829-25-4
- 分子式：C₁₇H₁₈N₂O₆
- 分子量：346.33
- 外观：黄色或浅黄色结晶性粉末
- 熔点：172–174 ℃
- 溶解度：
  - 水：几乎不溶
  - 乙醇、丙酮、氯仿：易溶
  - 油脂/有机溶剂：可溶（脂溶性较高）
# 2. 化学性质、光热稳定性
- 化学类别：1,4-二氢吡啶类钙通道阻滞剂
- 光稳定性：极差（强烈避光，光照下迅速降解）
- 热稳定性：一般，受热可加速分解
- 酸碱稳定性：中性偏稳定，强酸强碱下分解
# 3. 生化特性
- 主要靶点：L-型钙通道（CaV1.2）
- 作用部位：血管平滑肌（>心肌）
- 选择性：血管选择性明显强于心脏
# 4. 适应症、药理毒理
- 适应症
  - 原发性高血压
  - 稳定型 / 变异型心绞痛
  - 雷诺现象（非适应证用药）
- 药理作用
  - 阻断Ca²⁺内流 → 血管平滑肌舒张
  - 外周阻力↓ → 血压下降
- 毒理特点
  - 治疗窗较窄
  - 快速起效制剂更易诱发反射性交感兴奋
# 5. 药代动力学、起效时间
- 吸收：口服吸收快
- 生物利用度：45–75%（首过效应明显）
- 起效时间：
  - 速释片：10–20分钟
  - 控释/缓释片：30–60分钟
- 半衰期：
  - 普通片：2–5小时
  - 缓释制剂：7–9小时
- 代谢：肝脏（CYP3A4）
- 排泄：尿液为主（代谢物）
# 6. 常见剂量、给药方式
- 口服，缓释片：20–30 mg，每日1–2次，最大通常 ≤60 mg/日
- 不推荐咀嚼或掰开缓释片
- 已不推荐用于高血压急降压
# 7. 副作用、药物过量
- 常见副作用
  - 面部潮红
  - 头痛、眩晕
  - 下肢水肿（毛细血管扩张）
  - 心悸、反射性心动过速
- 过量表现
  - 严重低血压
  - 心动过速或心动过缓
  - 意识障碍、休克
# 8. 同分异构体及类似物质
- 氨氯地平：起效慢、半衰期长
- 非洛地平：血管选择性高
- 拉西地平：脂溶性更强
- 尼卡地平：可静脉用  
**硝苯地平反射性交感兴奋最明显**
# 9. 在人体内的整体作用
- 外周血管扩张为主
- 心排出量短期可↑
- 长期可改善血管顺应性
- 对心肌抑制作用弱
# 10. 内分泌相关激素
- 间接影响：
  - 交感兴奋 → 肾上腺素↑
  - RAAS系统轻度激活（短期）
- 不直接作用于性激素轴
# 11. 对脂肪代谢的作用
- 中性或轻微改善
- 不升高血脂
- 与胰岛素敏感性关系不大
# 12. 对血压的作用
- 显著降低收缩压
- 对舒张压作用相对温和
- 快速制剂 → 血压波动明显
# 13. 对消化系统的作用（急性）
- 恶心
- 胃部不适
- 肠蠕动增强（少见）
# 14. 对神经系统的调节
- 非中枢性作用
- 通过血压骤降 → 继发性头晕
- 反射性交感神经激活明显
# 15. 对生殖系统的作用
- 无直接抑制性功能作用
- 心悸、低血压可间接影响勃起稳定性
- 不影响睾酮合成
# 16. 对皮肤的作用
- 面部潮红
- 毛细血管扩张
- 罕见皮疹、瘙痒
# 17. 过多或不足时的治疗药物
- 过量处理
  - 钙制剂（葡萄糖酸钙）
  - 血管活性药（去甲肾上腺素）
  - 活性炭（早期）
- 对比女性（非孕期），女性更易出现：
  - 面部潮红
  - 下肢水肿
  - 用药原则一致
# 18. 中医八纲与五行归经
- 性味：偏寒
- 八纲：属里证、虚实夹杂
- 归经：心、肝
- 中医理解：
  - 平肝潜阳
  - 通脉降压
  - 偏治“肝阳上亢型高血压”

# 1. 常见别名、物理性质、CAS、溶解度
- 通用名：美托洛尔
- 英文名：Metoprolol
- 常见制剂：
  - 酒石酸美托洛尔（Metoprolol tartrate，速释）
  - 琥珀酸美托洛尔（Metoprolol succinate，缓释）
- CAS：
  - 游离碱：51384-51-1
  - 酒石酸盐：56392-17-7
  - 琥珀酸盐：98418-47-4
- 分子式：C₁₅H₂₅NO₃
- 物理性质：白色或类白色结晶性粉末
- 溶解度：
  - 水：易溶
  - 乙醇：易溶
  - 有机溶剂（氯仿等）：中等
  - 脂溶性：中等偏低（β₁选择性来源之一）
# 2. 化学性质、光热稳定性
- 化学类别：芳氧丙醇胺类 β 受体阻滞剂
- 手性：外消旋体（R/S）
- 稳定性：
  - 光稳定
  - 常温下稳定
  - 强酸强碱下可缓慢降解
# 3. 生化特性
- β₁受体选择性阻断（心脏为主）
- 对 β₂ 受体影响较弱（高剂量选择性下降）
- 无内在拟交感活性（ISA−）
- 无膜稳定作用（非奎尼丁样）
# 4. 适应症、药理毒理
- 药理作用
  - ↓ 心率
  - ↓ 心肌收缩力
  - ↓ 心排出量
  - ↓ 肾素分泌
- 临床适应症
  - 高血压
  - 冠心病、心绞痛
  - 心肌梗死后二级预防
  - 心律失常（房速、房颤控率）
  - 慢性心衰（仅限琥珀酸缓释剂型）
  - 功能性心悸、焦虑相关心动过速
- 毒理
  - 治疗窗较宽
  - 中枢与心脏毒性为主
# 5. 药代动力学、起效时间
- 口服生物利用度：40–50%（首过效应明显）
- 达峰时间：
  - 速释：1–2 h
  - 缓释：6–12 h
- 半衰期：3–7 h
- 代谢：肝脏 CYP2D6
- 排泄：肾（代谢物为主）
- 起效时间：口服约 1 h
# 6. 常见剂量、给药方式
- 高血压 / 心绞痛
  - 25–50 mg，1–2 次 / 日（酒石酸）
  - 或 47.5–95 mg，qd（琥珀酸缓释）
- 心衰（HFrEF）
  - 起始：12.5–23.75 mg qd
  - 目标：190 mg qd（缓慢滴定）
- 不可突然停药
# 7. 副作用、药物过量
- 常见
  - 心动过缓
  - 乏力、嗜睡
  - 低血压
  - 手足发冷
- 少见但重要
  - 抑郁、梦境异常
  - 性功能下降（中枢+血流因素）
- 过量表现
  - 严重心动过缓
  - 低血压、休克
  - 支气管痉挛（大剂量）
- 解毒
  - 胰高血糖素
  - 阿托品
  - 升压药
# 8. 常见同分异构体和类似物质
- 同分异构体：R / S 对映体（临床不区分）
- 类似 β₁ 阻滞剂：
  - 比索洛尔（β₁选择性更高）
  - 阿替洛尔（脂溶性更低）
  - 奈必洛尔（NO 释放）
# 9. 在人体内的整体作用
- 心脏负荷降低
- 交感神经张力下调
- 改善心肌耗氧/供氧比
- 稳定电生理环境
# 10. 内分泌相关激素
- ↓ 肾素 → ↓ 血管紧张素 II
- 间接 ↓ 醛固酮
- 对 睾酮/雌激素无直接抑制
# 11. 对脂肪代谢的作用
- 轻度 ↓ 脂解
- 可能 ↑ 体重（长期）
- 对胰岛素敏感性影响 小于非选择性 β 阻滞剂
# 12. 对血压的作用
- ↓ 心排量
- ↓ RAAS 活性
- 对外周阻力影响较小
- 长期降压效果稳定
# 13. 对消化系统的作用（急性）
- 偶见：
  - 恶心
  - 腹胀
- 不显著影响胃酸分泌
# 14. 对神经系统的调节机制和作用
- 中枢 β₁ 阻断
- ↓ 蓝斑核去甲肾上腺素释放
- 抗焦虑样效应
- 可能诱发或加重抑郁（个体差异）
# 15. 对生殖系统的作用
- 间接影响勃起（血流 + 交感抑制）
- 不抑制精子生成
- 不直接影响性激素轴
# 16. 对皮肤的作用
- 手足发凉
- 可加重银屑病（少见）
- 轻度 ↓ 皮肤血流
# 17. 过多或不足时的常用治疗药物
- 过量
  - 胰高血糖素
  - 多巴胺 / 去甲肾上腺素
  - 高胰岛素-正常血糖治疗
- 不足（控压/控率失败）
  - 钙通道阻滞剂
  - ACEI / ARB
- 与非孕产期女性相比：无明显用药差异
# 18. 中医八纲辨证与五行归经（参考性）
- 八纲：偏虚证、寒证
- 五行归经：心经、肾经
- 中医对应功效理解：
  - 安神
  - 定悸
  - 抑亢阳
- 不宜用于“阳虚明显、脉迟弱者”

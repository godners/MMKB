# 1. 常见别名、物理性质、CAS、溶解度（以“亚硝酸异戊酯”为代表）
- 通用名称：亚硝酸酯（Alkyl Nitrites）
  - 常见单体：
  - 亚硝酸异戊酯（Isoamyl nitrite）CAS 110-46-3
  - 亚硝酸戊酯（Amyl nitrite）CAS 463-04-7
  - 亚硝酸丁酯（Butyl nitrite）CAS 544-16-1
  - 亚硝酸异丙酯（Isopropyl nitrite）CAS 541-42-4
- 常温：淡黄色透明液体
- 气味：刺激性甜味
- 沸点：~98–102°C
- 密度：0.87 g/mL
- 挥发性：极高
- 闪点：< 10°C（极易燃）
- 溶解度
  - 水：极微溶
  - 乙醇、乙醚、丙酮、氯仿：高度可溶
  - 脂类溶剂：高溶解度
  - 属典型亲脂小分子酯。
# 2. 化学性质、光热稳定性
- 对热 极不稳定，50–60°C 开始分解。
- 强光下分解产生 NO、NO₂。
- 遇酸可进一步水解出亚硝酸，后者不稳定。
- 强氧化剂接触会剧烈反应。
- 储存需 避光、低温、密封。
# 3. 生化特性
- 在体内迅速释放 一氧化氮（NO）。
- 强效 血管平滑肌松弛剂。
- 对中枢无直接兴奋作用，但可诱发短暂头晕、温热感。
# 4. 适应症、药理毒理
- 合法医学用途（以戊酯为准）
  - 传统急救用药：因急性心绞痛（已被硝酸甘油取代）。
  - 氰化物中毒辅助解毒（经 NO → MetHb 途径）。
- 毒理性（成人男性）
  - 急性：低血压、心动过速、头痛、晕厥
  - 严重：高铁血红蛋白血症（MetHb↑） → 缺氧、紫绀
  - 皮肤接触可化学灼伤
  - 与 PDE5 抑制剂（如西地那非）合用可致致命低血压
# 5. 药代动力、起效时间
- 起效：数秒内
- 高峰：30–60 秒
- 代谢：主要在肝内，快速分解为醇 + NO₂⁻
- 半衰期：短（数分钟）
# 6. 常见剂量、给药方式
- 医学用途已极少使用，历史上为：蒸汽吸入（医用戊酯胶囊）
- 现代医疗中基本已被淘汰。
# 7. 副作用、药物过量
- 常见：
  - 面红
  - 头痛
  - 眩晕
  - 心动过速
  - 恶心
- 过量/严重反应：
  - MetHb > 20–30%：紫绀、呼吸困难
  - 严重低血压、晕厥
  - 视力模糊
  - 皮肤接触引起化学烧伤
  - 男性勃起功能短暂下降（NO 过量）
# 8. 常见同分异构体与类似物的生化特性
- 亚硝酸异戊酯：液体，毒性中等，挥发性高，医学史常用
- 亚硝酸正丁酯：液体，毒性较高，挥发性高，刺激性强
- 亚硝酸异丙酯：液体，毒性毒性最高，挥发性极高，欧盟限制销售
- 亚硝酸甲酯：室温固体/低熔点固体，毒性高危，挥发性中	强致癌性，不用于任何娱乐或医用
- 唯一常见的固体类似物是“亚硝酸甲酯”（methyl nitrite） ——但其毒性极强，不具备与亚硝酸异戊酯类似的安全窗口，且不会用于任何面向人体的场景。
# 9. 在人体内的整体作用
- 核心作用：亚硝酸酯 → NO释放 → 平滑肌弛缓
- 影响：
  - 血管扩张 → 血压下降
  - 肛门直肠括约肌放松（医学文献可见）
  - 冠状血管扩张
  - 组织氧输送下降（因 MetHb↑）
# 10. 内分泌相关激素影响
- 不直接调节性激素。
- 但急性低血压 + 缺氧可暂时影响睾酮合成与性生活表现。
- 不造成长期内分泌改变。
# 11. 对脂肪代谢
- 无直接影响。
- 急性低氧可能轻度改变线粒体代谢，但作用短暂。
# 12. 对血压的作用
- 快速、显著降低血压（可达30–50 mmHg）。
- 伴反射性心率升高。
# 13. 对消化系统的急性作用
- 食管下括约肌短暂松弛
- 恶心
- 胃灼热
- 过量时腹痛
# 14. 对神经系统的作用与机制
- 不直接作用于中枢
- 头晕、欣快感来自脑灌注变化（低血压→代偿性脑血流↑）
- MetHb 导致的轻度缺氧亦可造成异常感受
# 15. 对生殖系统的作用
- 通过 NO 途径使平滑肌暂时松弛
- 不增强勃起
- 反而可能因低血压导致 勃起功能短暂下降
# 16. 对皮肤作用
- 液体直接接触可造成：
  - 灼伤
  - 红斑
  - 水疱
# 17. 过多或不足时常用治疗药物（对比非孕产期女性）
- 过量（MetHb 增高）治疗：
  - 美蓝（亚甲蓝） 静脉（首选）
  - 高流量氧气
  - 血压支持
- 男女治疗方案差异不大。
# 18. 中医八纲辨证与五行归经分析（根据药性类比推断）
- 亚硝酸酯并非中药，仅可根据其生理表现作类比：
- 性味归类：辛、温、走窜
- 作用类似：行气活络、通腠理
- 八纲：属阳、属热、属虚（气脱样低血压）
- 可能对应：
  - 手太阴肺（气机速变）
  - 足少阴肾（血压调节）
  - 手厥阴心包（血管舒张）

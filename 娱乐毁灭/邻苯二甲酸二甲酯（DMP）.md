# 1. 常见别名、物理性质、CAS、溶解度
- 别名：DMP、Phthalic acid dimethyl ester、邻苯二甲酸二甲酯
- CAS 号：131-11-3
- 分子式 / 分子量
  - C₁₀H₁₀O₄
  - 194.18 g/mol
- 外观：无色至淡黄色油状液体
- 气味：微弱芳香气味
- 密度（25℃）：≈ 1.19 g/cm³
- 熔点：≈ 8–9 ℃
- 沸点：≈ 282–285 ℃
- 蒸气压（25℃）：极低（<0.01 mmHg）
- 溶解度
  - 水：微溶（约 4 g/L，25℃）
  - 易溶于：乙醇、甲醇、丙酮、乙醚、苯、氯仿
  - 可溶于油脂、塑化体系
  - 无机溶液：在酸、碱水溶液中稳定，强碱条件下可水解
# 2. 化学性质、光热稳定性
- 属邻苯二甲酸酯类酯化物
- 化学稳定性较高
- 酸性条件：稳定
- 碱性条件：发生酯键水解 → 邻苯二甲酸 + 甲醇
- 光稳定性：紫外线下缓慢光降解
- 热稳定性：<200℃ 稳定，250℃ 逐渐分解
# 3. 生化特性
- 典型外源性内分泌干扰物（EDC）
- 在体内可被酯酶水解为：
  - 单甲酯（MMP）
  - 邻苯二甲酸（PA）
- 与脂溶性组织（脂肪、性腺）有一定亲和性
# 4. 适应症、药理毒理
- 适应症：无临床治疗用途
- 工业/历史用途
  - 塑化剂（已逐步被限制）
  - 驱蚊剂成分（部分国家）
  - 香料、涂料、胶黏剂中间体
- 毒理学
  - 低急性毒性
  - 慢性暴露 → 内分泌干扰、生殖毒性风险
  - IARC：未列为明确致癌物
  - EU REACH：受监管物质
# 5. 药代动力学、起效时间
- 吸收途径
  - 经口
  - 经皮（慢）
  - 吸入（较少）
- 体内过程
  - 吸收后迅速被水解
  - 主要在肝脏代谢
  - 代谢物经尿液排出
- 半衰期
  - 数小时级（母体物质）
  - 代谢物可持续 1–2 天检测到
# 6. 常见剂量、给药方式
- 无医学安全剂量
- 所有剂量讨论仅限暴露评估
- 参考值：TDI（EFSA）约 0.5 mg/kg·day（同类酯类参考）
# 7. 副作用、药物过量
- 急性大量暴露
  - 恶心、头晕、乏力
  - 皮肤刺激（长期接触）
- 慢性暴露
  - 精子数量、活力下降
  - 睾酮水平轻度抑制
  - 精索/附睾发育异常（动物）
# 8. 同分异构体与类似物
- 邻苯二甲酸酯家族
  - 邻苯二甲酸二乙酯（DEP）
  - 邻苯二甲酸二丁酯（DBP）
  - 邻苯二甲酸二(2-乙基己)酯（DEHP）
- 趋势
  - 碳链越长 → 脂溶性↑、生殖毒性↑
  - DMP 属于 毒性相对较低 成员，但仍非安全
# 9. 在人体内的整体作用
- 不作为治疗物
- 作为环境暴露因子：
  - 干扰性激素轴
  - 改变生殖系统稳态
  - 累积效应 > 单次效应
# 10. 内分泌相关激素
- ↓ 睾酮（Leydig 细胞功能抑制）
- 干扰：LH / FSH 反馈调节
- 具有弱雌激素样 + 抗雄激素样效应
# 11. 对脂肪代谢的作用
- 脂溶性 → 易在脂肪组织分布
- 激活 PPAR 通路（弱）
- 长期暴露与：
  - 脂肪代谢紊乱
  - 胰岛素敏感性改变
  - 内脏脂肪比例变化
- 相关（流行病学相关性为主）
# 12. 对血压的作用
- 无直接升压或降压药理作用
- 间接：
  - 内分泌紊乱 → 代谢综合征风险↑
  - 与血压异常呈弱相关
# 13. 对消化系统的作用（急性）
- 口服刺激胃肠道
- 可致：
  - 恶心
  - 腹胀
  - 轻度腹泻
- 不产生溃疡性损伤
# 14. 对神经系统的调节机制
- 不直接作用中枢神经递质
- 间接影响：
  - 激素-神经轴
  - 发育期暴露 → 行为与认知改变（动物证据）
# 15. 对生殖系统的作用（重点）
- 男性
  - 精子数量 ↓
  - 精子畸形率 ↑
  - 睾丸支持细胞功能受损
  - 睾酮合成抑制
- 机制
  - 抑制类固醇合成酶
  - 干扰 Sertoli-Leydig 协同
# 16. 对皮肤的作用
- 脂溶性 → 可经皮吸收
- 长期接触：
  - 轻度接触性皮炎
  - 皮肤屏障脂质改变
- 不属于强致敏物
# 17. 过多或不足时的治疗药物
- DMP 本身不为必需物质
- 慢性暴露干预（男性）
  - 停止暴露源
  - 抗氧化支持：维生素 E、维生素 C、N-乙酰半胱氨酸
- 生殖功能受损时：
  - 克罗米芬（男性）
  - hCG / hMG（严格医学指征）
- 女性更关注：
  - 月经紊乱
  - 雌激素-孕激素平衡
- 用药方向不同，男性更偏向促雄激素轴恢复
# 18. 中医八纲辨证与五行归经（现代映射）
- 外来化学物 → 属“外邪毒浊”
- 性偏：浊、湿、微寒
- 八纲：
  - 表里：里
  - 寒热：偏寒
  - 虚实：实中夹虚
  - 阴阳：阴邪
- 五行归经
  - 肝（疏泄受阻、精血异常）
  - 肾（生殖、精气受损）
  - 脾（湿浊内生）
- 中医调理思路（理论）
  - 化湿解毒
  - 补肾填精
  - 疏肝理气
# 总结一句话
邻苯二甲酸二甲酯不是药，而是典型的低毒但明确的内分泌干扰化学物，对成年男性的核心风险集中在“慢性暴露 → 生殖轴与代谢轴扰动”。

# 1. 常见别名、物理性质、CAS编号、溶解度
- 通用名：鹿茸精注射液
- 英文名：Deer Antler Extract Injection / Velvet Antler Extract Injection
- 常见别名：Velvet Antler Injection、鹿茸提取物注射液、Pantocrin（部分老文献中类似制剂的商品名）
- 主要成分：鹿茸提取物（含多肽、氨基酸、甾体类激素、多糖）
- 辅料：常含氯化钠注射液或葡萄糖注射液为溶剂
- 性状：棕黄色至浅棕色澄明液体，有特异气味
- 溶解性：可完全溶于水（注射用水、0.9%NaCl），不溶于有机溶剂（乙醇、乙醚）
- CAS号：暂无统一CAS编号（属动物提取复合物，非单一化合物）
# 2. 化学性质、光热稳定性
- 性质：含肽类、氨基酸、小分子肽、性甾体（睾酮样成分）、神经肽及多糖等混合物。
- 光热稳定性：对高温和强光敏感，长时间暴露可导致蛋白、多肽变性。需避光、冷藏保存（2–8 °C）。
# 3. 生化特性
- 含有碱性多肽、L-精氨酸、L-谷氨酸及微量性激素样成分（如17β-睾酮、孕酮、IGF-1样因子）。
- 可促进DNA与蛋白质合成，增强细胞再生。
- 对肾上腺皮质功能、性腺功能及免疫系统有调节作用。
# 4. 适应症、药理毒理
- 适应症：
  - 肾阳虚引起的性功能减退、阳痿、早泄；
  - 神经衰弱、记忆减退；
  - 慢性疲劳、体虚、术后恢复；
  - 更年期综合征。
- 药理作用：
  - 内分泌调节：刺激睾丸间质细胞分泌睾酮；
  - 免疫增强：促进T淋巴细胞活性；
  - 造血促进：促进骨髓红系细胞生成；
  - 抗疲劳作用：提高肝糖原储备；
  - 神经保护：促进神经元修复；
  - 毒理学：毒性极低，大鼠LD₅₀>50 mL/kg。
# 5. 药代动力学与起效时间
- 属蛋白多肽混合物，非特定分子动力学模型；
- 起效时间：注射后约30–60 min内可感到体能提升；
- 代谢排泄：肽类在体内经水解为氨基酸，经肾脏排出。
# 6. 常见剂量与给药方式
- 成人常用量：2 mL～4 mL（含原液或等效冻干粉复溶液）；
- 用法：肌内注射或静脉滴注；
- 疗程：每日1次或隔日1次，10–20次为一疗程；
- 配伍注意：避免与酸性或含金属离子药物同瓶混用。
# 7. 副作用与药物过量
- 轻度反应：注射部位红肿、短暂发热；
- 系统反应：少数人可出现血压轻升、心率加快；
- 过量：可能引起失眠、性欲过强、易怒、面部潮红；
- 处理：停药、对症支持治疗。
# 8. 常见同分异构体和类似物的生化特性
- 鹿茸口服液：生物利用度低；
- 鹿茸多肽注射液：以纯化肽为主，免疫调节更强；
- 鹿鞭提取物：雄激素样效应更明显；
- 人参鹿茸注射液：协同增强抗疲劳、性腺功能。
# 9. 在人体内的整体作用
- 激活下丘脑–垂体–性腺轴；
- 增加血清睾酮与17-羟皮质类固醇；
- 改善红细胞生成与血氧携带；
- 提高免疫耐受与抗氧化酶活性（SOD↑、GSH-Px↑）。
# 10. 内分泌相关激素
- 增强睾酮（Testosterone）、IGF-1、DHEA分泌；
- 对皮质醇有轻度升高作用；
- 抑制垂体泌乳素升高。
# 11. 对脂肪代谢的作用
- 促进脂肪酸氧化；
- 提高肌肉线粒体活性；
- 可轻度降低血清甘油三酯。
# 12. 对血压的作用
- 少数使用者出现轻度升压（收缩压+5～10 mmHg）；
- 长期使用可能改善外周血循环。
# 13. 对消化系统的作用（急性）
- 可促进胃液及消化酶分泌；
- 少数患者出现胃胀或轻度恶心；
- 对肝功能无明显影响。
# 14. 对神经系统的调节机制与作用
- 增强脑细胞代谢，改善记忆；
- 含有神经营养肽，促进突触可塑性；
- 可提升去甲肾上腺素和多巴胺水平。
# 15. 对生殖系统的作用
- 促进睾丸间质细胞和生精细胞分裂；
- 提高精子活力与密度；
- 改善性腺激素分泌平衡；
- 增强性兴奋反应。
# 16. 过多或不足时常用治疗药物（与非孕产期女性对比）
## 过量刺激（性欲过强、失眠）
- 成人男性用药：状况镇静剂（如地西泮）、β受体阻滞剂
- 非孕产期女性对比：相同
## 效果不足（阳痿、疲劳）
- 成人男性用药：鹿茸精+人参注射液、曲普瑞林、睾酮替代
- 非孕产期女性对比：雌激素替代、地屈孕酮
## 免疫低下
- 成人男性用药：鹿茸多肽注射液、胸腺肽
- 非孕产期女性对比：同类免疫调节剂
# 17. 中医八纲辨证与五行归经分析
- 归经：肾、肝、脾经
- 八纲辨证：阳虚、气血两亏
- 功效：补肾阳、益精血、强筋骨
- 五行归属：属水，兼及木（土为辅）
- 主治证型：肾阳虚衰、命门火衰、阳痿、遗精、腰膝酸软

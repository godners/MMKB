# 紫石英

## 1. 常见别名、物理性质、CAS、溶解度

- 现代学名：紫水晶（Amethyst）
- 矿物学名称：紫色石英（α-SiO₂）
- 中药名：紫石英
- 化学组成：SiO₂（含微量 Fe³⁺、Al³⁺、Ti 等致色杂质）
- CAS号：14808-60-7（石英）
- 晶系：三方晶系
- 硬度：7（莫氏）
- 密度：2.65 g/cm³
- 颜色：紫色—深紫色（辐射致色）
- 溶解度：
  + 水：不溶
  + 有机溶剂：不溶
  + 无机酸：常温下不溶（HF 可溶，生成 SiF₄）

## 2. 化学性质、光热稳定性

- 化学惰性极强
- 常温下不参与氧化还原反应
- 光稳定性：
  + 紫色来源于天然辐射 + Fe³⁺电子缺陷
  + 强光或 >300℃ 可褪色（转黄→白石英）
- 热稳定性：
  + 573℃ 发生 α-β 相变
  + 800℃ 紫色消失

## 3. 生化特性

- 本身不参与生化代谢
- 在人体中：
  + 不被吸收
  + 不溶出有效离子
- 药用作用主要来源于：
  + 微量金属元素
  + 物理能量刺激（传统认识）

## 4. 适应症、药理毒理

- 中医传统适应症
  + 心神不宁
  + 惊悸失眠
  + 癫狂
  + 小儿惊风（古籍）
- 现代毒理
  + 基本无毒
  + 粉尘长期吸入 → 矽肺（仅限职业暴露）
  + 口服矿石粉末：机械性刺激为主

## 5. 药代动力、起效时间

- 不吸收、不分布、不代谢
- 经消化道原形排出
- 起效时间属于经验性、心理-神经调节层面
- 多为长期调养型

## 6. 常见剂量、给药方式

- 传统内服（已极少使用）：9–15 g（需先煎 ≥30分钟）
- 现代临床：基本不内服，多用于：
  + 镇静类中成药理论配伍研究
  + 矿物药考古/文献复原

## 7. 副作用、药物过量

- 内服可能：胃肠道刺激、便秘
- 粉末长期服用：机械性黏膜损伤风险
- 现代医学不推荐内服

## 8. 同分异构体与类似物

- 白石英：无色，稳定、惰性
- 紫石英：紫，镇静象征
- 黄水晶：黄，精神兴奋象征
- 烟晶：棕黑，重镇、收敛
- **实质生化差异极小，主要是文化与心理层面区分**

## 9. 在人体内的整体作用

- 无直接药理活性
- 传统认为：镇心安神、抑制过度神经兴奋
- 实际作用更接近：心理暗示、感官调节

## 10. 对内分泌相关激素

- 无直接影响
- 间接可能影响：
  + 褪黑素（通过睡眠改善）
  + 皮质醇（压力缓解）

## 11. 对脂肪代谢的作用

- 无直接影响
- 间接影响仅限于：睡眠改善 → 胰岛素敏感性微调

## 12. 对血压的作用

- 无直接降压作用
- 若情绪焦虑缓解：可轻微降低交感神经张力、间接改善血压波动

## 13. 对消化系统的作用（急性）

- 内服矿粉：可刺激胃肠黏膜
- 外用或佩戴：无消化系统作用

## 14. 对神经系统的调节机制

- 非药理性
- 主要通过：
  + 情绪稳定
  + 视觉感知（紫色镇静效应）
  + 冥想/安抚行为联想

## 15. 对生殖系统的作用

- 无直接作用
- 间接：
  + 情绪稳定 → 性激素轴压力降低
  + 有利于睾酮昼夜节律稳定（理论）

## 16. 对皮肤的作用

- 外用接触：
  + 化学惰性
  + 不致敏
- 矿物粉末：
  + 摩擦刺激风险

## 17. 过多或不足时的替代治疗

- 不足（即镇静需求）
  + 男性：地西泮（短期）、褪黑素
  + 非孕产期女性：同男性，剂量略低
- 过量（矿粉刺激）
  + 对症处理：胃黏膜保护剂（如硫糖铝）

## 18. 中医八纲与五行归经

- 性味：甘、寒（部分文献作平）
- 归经：心、肝
- 八纲：
  + 属阴
  + 主清、镇
- 五行：
  + 色属火
  + 性偏水
  + 火水既济 → 镇心安神

## 总结一句话

- 紫石英是一种高度稳定、几乎无现代药理活性的矿物
- 传统“安神镇心”作用主要源于文化认知、心理调节与象征意义，而非化学作用。

# 酒石酸伐尼克兰片

## 1. 基本信息

- 通用名：酒石酸伐尼克兰片
- 英文名：Varenicline Tartrate Tablets
- 常见商品名：畅沛（Chantix/Champix）
- CAS编号：
  + 伐尼克兰（Varenicline）：183849-32-1
  + 酒石酸伐尼克兰：375815-87-5
- 分子式（酒石酸盐）：C13H13N3 · C4H6O6
- 分子量：约361.35
- 白色至类白色结晶性粉末
- 味微苦
- 易吸湿
- 熔点：约200℃以上（分解）
- 溶解度
  + 水中：易溶
  + 乙醇中：微溶
  + 有机溶剂（氯仿、乙醚）：几乎不溶
  + 在酸性环境中溶解度更高（胺类碱性结构）

## 2. 化学性质与稳定性

- 属于双环氮杂结构的部分碱性化合物
- 形成稳定酒石酸盐，提高水溶性
- 光稳定性：对光较稳定
- 热稳定性：常温稳定，高温分解
- 在强酸强碱条件下可发生降解

## 3. 生化特性

- 为 α4β2型烟碱型乙酰胆碱受体（nAChR）部分激动剂
- 同时对α7型受体有弱激动作用
- 作用机制：
  + 部分激动 → 提供低水平多巴胺释放
  + 竞争拮抗 → 阻断尼古丁完全激动作用
- 既减少戒断症状，又降低吸烟快感

## 4. 适应症

- 成人戒烟治疗
- 尼古丁依赖

## 5. 药理与毒理

- 药理作用
  + 刺激中脑-边缘多巴胺系统
  + 降低尼古丁诱导的奖赏效应
  + 减轻焦虑、烦躁、渴求
- 毒理
  + 恶心
  + 呕吐
  + 中枢兴奋
  + 癫痫（罕见）
- 无明显心脏传导抑制作用

## 6. 药代动力学

- 吸收
  + 口服生物利用度高（>90%）
  + 不受食物影响
- 分布：蛋白结合率低（<20%）
- 代谢
  + 代谢极少
  + 主要以原形经肾排泄
- 消除
  + 半衰期：约24小时
  + 4–5个半衰期达稳态（约4–5天）
- 起效时间
  + 减少吸烟渴求：3–7天
  + 完整疗效：2–4周

## 7. 常见剂量与给药

- 标准戒烟方案：
  + 第1–3天：0.5 mg，每日一次
  + 第4–7天：0.5 mg，每日两次
  + 第8天起：1 mg，每日两次
  + 疗程：12周（可延长）
- 肾功能不全需减量。

## 8. 副作用

- 常见
  + 恶心（最常见）
  + 失眠
  + 异常梦境
  + 头痛
  + 口干
- 精神神经系统
  + 情绪波动
  + 抑郁
  + 自杀意念（罕见但需警惕）
- 过量表现
  + 恶心、呕吐
  + 震颤
  + 神经兴奋
  + 对症处理，无特效解毒剂

## 9. 同分异构体与类似物

- 结构类似物
  + 尼古丁
  + 西替利嗪（非同类，仅结构含杂环）
- 生化差异
  + 尼古丁：完全激动剂 → 强多巴胺释放
  + 伐尼克兰：部分激动剂 → 中等多巴胺释放

## 10. 人体整体作用（成年男性）

- 降低尼古丁依赖
- 多巴胺调节趋于稳定
- 对性激素轴无直接抑制作用
- 长期戒烟后可改善血管内皮功能

## 11. 内分泌影响

- 不直接影响睾酮、LH、FSH
- 戒烟后：
  + 皮质醇水平可能下降
  + 睾酮可能轻度上升（戒烟带来的继发效应）

## 12. 对脂肪代谢

- 本身无直接脂代谢作用
- 戒烟后体重可能上升
- 部分患者体重增加1–3 kg

## 13. 对血压

- 无显著升压作用
- 戒烟长期有助于降低血压

## 14. 对消化系统（急性）

- 恶心最常见
- 胃排空延迟轻度
- 可引起腹胀、便秘

## 15. 神经系统调节机制

- 激活VTA-NAc多巴胺通路
- 部分激动减少奖赏峰值
- 降低尼古丁诱导的突触可塑性

## 16. 生殖系统

- 无直接雄激素调节作用
- 戒烟后勃起功能可能改善（血管因素）

## 17. 对皮肤

- 无直接皮肤作用
- 少数可出现皮疹

## 18. 过多或不足时的处理

- 过量
  + 支持治疗
  + 严重者透析可部分清除
- 戒断不足（效果差）
  + 可联合：
  + 尼古丁贴片
  + 安非他酮
- 女性（非孕）用药与男性相同；孕期禁用或慎用。

## 19. 中医分析

- 八纲辨证
  + 属“瘾证”“烟毒内扰”
  + 多见：肝郁气滞、心神不宁
- 五行归经
  + 主入：心、肝经
  + 调神志，清烟毒

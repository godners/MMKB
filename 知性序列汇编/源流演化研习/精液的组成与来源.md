# 总体中医归纳
- 精液 = 肾精（本）+ 肾气（化）+ 津液（载）
- 少 → 肾虚、气化不及
- 多而稀 → 肾不固、精关不摄
- 浓而少 → 津亏、火盛

# 1. 睾丸来源部分（精子 + 少量睾丸液）
## 1.1 主要成分与作用
- 成分
  - 精子（spermatozoa）
  - 少量睾丸液（含蛋白、离子、少量葡萄糖）
- 作用
  - 精子：遗传信息传递
  - 睾丸液：维持精子代谢、渗透压
- 中医对应
  - 属于「肾精之实」
  - 《内经》：“肾藏精，精生髓”
## 1.2 占整体体积比例
- 约 2–5%
## 1.3 含水比例
- 约 70–80%
- 精子本身为高度结构化细胞，其余为水性介质
## 1.4 产生 / 分泌原理
- 现代
  - 下丘脑–垂体–睾丸轴
  - GnRH → LH/FSH → 睾酮 + 精子发生
- 中医
  - 肾气充盛 → 精化有源
  - 命门火温煦肾精
## 1.5 过多 / 过少的明确原因
- 过少
  - 原发 / 继发性性腺功能减退
  - 长期高温（睾丸热抑制）
  - 精索静脉曲张
- 异常“过多”
  - 临床罕见（多为浓缩假象）
## 1.6 明确影响的药物
- 精子减少
  - 外源性睾酮
  - GnRH 拮抗剂
  - 烷化剂（环磷酰胺）
- 精子增多
  - 克罗米芬
  - hCG
  - 来曲唑（间接）
## 1.7 明确影响的物理刺激
- 精子减少
  - 长期阴囊加热（>34–35℃）
  - 持续压迫
- 精子增多（有限）
  - 适度冷刺激（短暂）
  - 避热、放松阴囊
# 2. 附睾来源液（精子成熟介质）
## 2.1 主要成分与作用
- 成分
  - L-肉碱
  - 糖蛋白
  - 抗氧化物
- 作用
  - 精子成熟、获得运动能力
- 中医
  - 属「精之化」，为肾精“再炼”
## 2.2 体积比例
- <1%
## 2.3 含水比例
- 约 90%
## 2.4 分泌原理
- 睾酮依赖性分泌
- 管腔浓缩回吸水分
## 2.5 异常原因
- 附睾炎：成熟障碍
- 输精管阻塞
## 2.6 药物影响
- 抗雄激素：减少
- 5α-还原酶抑制剂：减少
## 2.7 物理刺激
- 慢性炎症、挤压：减少
- 长期禁欲：老化精子积聚
# 3. 精囊腺液（最大体积来源）
## 3.1 主要成分与作用
- 成分
  - 果糖
  - 前列腺素
  - 凝固蛋白
- 作用
  - 精子能量来源
  - 促进子宫收缩
- 中医
  - 属「精中之津液」
  - 肾精化津，精囊为“精库”
## 3.2 体积比例
- 约 60–70%
## 3.3 含水比例
- 约 95–98%
## 3.4 分泌原理
- 雄激素刺激
- 射精反射时平滑肌收缩排出
## 3.5 过多 / 过少原因
- 过少
  - 精囊炎
  - 雄激素不足
- 过多
  - 长期禁欲
  - 精囊充血
## 3.6 药物
- 精子减少
  - 抗雄激素
  - 5α-还原酶抑制剂
- 精子增多
  - 雄激素
  - PDE5 抑制剂（间接）
## 3.7 物理刺激
- 精子增多
  - 会阴深部刺激
  - 长时间性兴奋未射精
- 精子减少
  - 频繁射精
# 4. 前列腺液
## 4.1 主要成分与作用
- 成分
  - 柠檬酸
  - 锌
  - PSA
- 作用
  - 液化精液
  - 保护精子DNA
- 中医
  - 属「肾气外化之液」
  - 与膀胱、精关密切相关
## 4.2 体积比例
- 约 20–30%
## 4.3 含水比例
- 约 95%
## 4.4 分泌原理
- α-肾上腺素能收缩
- 雄激素调控腺体
## 4.5 异常原因
- 过少
  - 前列腺萎缩
  - α受体阻滞过强
- 过多
  - 前列腺炎
  - 充血状态
## 4.6 药物
- 精子减少
  - 坦索罗辛
  - 非那雄胺
- 精子增多
  - 雄激素
  - α受体激动剂
## 4.7 物理刺激
- 精子减少
  - 前列腺按摩
  - 会阴震动刺激
- 精子增多
  - 长期压迫、久坐
# 5. 尿道球腺液（射精前液）
## 5.1 成分与作用
- 成分
  - 粘液
  - 碱性液
- 作用
  - 润滑尿道
  - 中和尿道酸性
- 中医
  - 属「肾气先泄之液」
## 5.2 体积比例
- <5%
## 5.3 含水比例
- `>99%`
## 5.4 分泌原理
- 副交感神经刺激
## 5.5 异常原因
- 神经过敏：增多
- 神经损伤：减少
## 5.6 药物
- 精子增多
  - 副交感兴奋药
- 精子减少
  - 抗胆碱药
## 5.7 物理刺激
- 阴茎根部刺激：增多
- 盆底放松：增多

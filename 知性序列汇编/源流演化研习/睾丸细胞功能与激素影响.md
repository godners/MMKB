# 睾丸细胞功能与激素影响

## 1. 总体结构速览

- 睾丸主要由两大功能区构成：
  + 生精小管（seminiferous tubules）：精子发生 + 支持系统
  + 间质（interstitium）：激素合成 + 血管 / 免疫 / 神经支持

## 2. 生精小管内的细胞

### 2.1 精原干细胞 / 精原细胞（Spermatogonia）

- 功能
  + 精子发生的起点
  + 通过有丝分裂维持精子供给
  + 包括：A型（干细胞储备）、In型、B型（分化前体）
- 激素影响
  + 睾酮（T）：不直接作用（精原细胞几乎无AR），通过支持细胞间接调控
  + 促卵泡生成素（FSH）：间接促进存活与分化（仍通过支持细胞）
  + 雌二醇（E2）：低剂量可促进增殖，高剂量 → 抑制分化、诱导凋亡
  + 甲状腺激素（T3）：影响精原细胞数量设定（青春期前尤为关键）
- 核心特点：精原细胞“听不太懂激素语言”，它们靠支持细胞当翻译

### 2.2 初级 / 次级精母细胞（Spermatocytes）

- 功能
  + 进行减数分裂 I / II
  + 决定遗传多样性
  + 是最容易受损的阶段
- 激素影响
  + 睾酮（T）：必需，尤其对减数分裂 I，通过支持细胞的 AR 信号
  + 促卵泡生成素（FSH）：促进减数分裂环境稳定
  + 皮质醇（应激）：增多 → 明显抑制减数分裂
  + 雌激素：生理浓度支持减数分裂，过高 → 阻断染色体配对

### 2.3 精子细胞（Spermatids）

- 功能
  + 由圆形 → 伸长 → 成熟精子
  + 发生：顶体形成，线粒体鞘重排，鞭毛生成，核浓缩
- 激素影响
  + 睾酮（T）：极其关键（形态成熟）
  + 促卵泡生成素（FSH）：促进支持细胞分泌 ABP、营养因子
  + 雌二醇（E2）：参与顶体成熟、膜重塑
  + 生长激素、IGF-1：促进细胞器重塑

### 2.4 支持细胞（Sertoli cell）

- 功能（核心枢纽）
  + 形成血–睾屏障
  + 营养、保护、生精调控
  + 分泌：ABP（雄激素结合蛋白），抑制素B（Inhibin B），AMH（胚胎期）
  + 吞噬残余体
  + “翻译”激素信号给生殖细胞
- 激素影响
  + 促卵泡生成素（FSH）：代谢、分泌、支持能力
  + 睾酮（T）：通过 AR 维持生精
  + 雌二醇（E2）：局部自分泌（芳香化），影响屏障通透性
  + 甲状腺激素：决定支持细胞“最终数量”（青春期前）
  + 催乳素（PRL）：低剂量促进高剂量抑制
- 没有支持细胞，所有激素都“白给”

## 3. 睾丸间质中的细胞

### 3.1 莱迪希细胞（Leydig cell）

- 功能
  + 合成：睾酮，少量雌二醇，DHEA
  + 维持：第二性征，性欲，生精环境
- 激素影响
  + 促黄体生成素（LH）：刺激类固醇生成
  + 胰岛素、IGF-1：促进甾体合成
  + 雌二醇：高水平 → 负反馈抑制
  + 皮质醇：抑制StAR蛋白
  + 炎症因子（IL-1β、TNF-α）：抑制睾酮合成

### 3.2 莱迪希干/前体细胞（Stem / Progenitor Leydig cells）

- 功能
  + 成年后莱迪希细胞更新来源
  + 决定长期睾酮潜力
- 激素影响
  + 促黄体生成素（LH）：慢性刺激
  + PDGF、TGF-β
  + 雌激素（过高）：抑制分化

### 3.3 周管肌样细胞（Peritubular myoid cells）

- 功能
  + 围绕生精小管
  + 收缩 → 推动管内液体与精子
  + 分泌基质与生长因子
- 激素影响
  + 睾酮（T）：维持收缩能力
  + 雌激素：影响基底膜结构
  + 前列腺素：调节收缩节律

### 3.4 睾丸巨噬细胞（Testicular macrophages）

- 功能
  + 免疫耐受（防止攻击精子）
  + 支持莱迪希细胞功能
  + 清除凋亡细胞
- 激素影响
  + 睾酮：抑制炎症型表型
  + 雌激素：促进免疫耐受
  + 应激激素：改变炎症状态

### 3.5 血管内皮细胞 / 淋巴内皮细胞

- 功能
  + 供氧、营养、激素运输
  + 温度调控
  + 维持低压微环境
- 激素影响
  + 睾酮（T）：促进血管生成
  + 雌激素：调节通透性
  + VEGF：关键局部调控因子

### 3.6 成纤维细胞 / 间质基质细胞

- 功能
  + 结构支撑
  + 分泌细胞外基质
  + 调节局部信号扩散
- 激素影响
  + 雄激素
  + TGF-β
  + 炎症因子

## 4. 激素—细胞关系表

|激素       |主要靶细胞         |核心作用         |
|:---------:|:-----------------:|:---------------:|
|LH         |莱迪希             |合成睾酮         |
|FSH        |支持细胞           |启动并维持生精   |
|睾酮       |支持 / 肌样 / 间质 |生精、结构、性征 |
|雌二醇     |支持 / 莱迪希      |精细调控、反馈   |
|抑制素B    |垂体               |反馈抑制FSH      |
|皮质醇     |多靶点             |抑制生精         |
|甲状腺激素 |支持               |细胞数量设定     |

## 5. 总结

- 睾丸不是“激素直接控制精子”，而是：
- 激素 → 支持细胞 / 莱迪希细胞 → 微环境 → 精子发生

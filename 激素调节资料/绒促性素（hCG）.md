# 1. 常见别名、物理性质、CAS、溶解度
- 别名：人绒毛膜促性腺激素、绒毛膜促性腺激素、hCG、Human Chorionic Gonadotropin
- 来源：人胎盘（现多为重组hCG）
- CAS号：9002-61-3
- 分子特征：糖蛋白激素，α/β双亚基（β亚基特异性决定生物学作用）
- 外观：白色或类白色冻干粉
- 溶解度：
  - 水/生理盐水：易溶
  - 有机溶剂（乙醇、丙酮、氯仿）：不溶
# 2. 化学性质、光热稳定性
- 化学性质：蛋白质类，易受蛋白酶降解
- 热稳定性：40℃活性明显下降
- 光稳定性：对强光敏感，应避光保存
- pH稳定区间：约 6.0–7.5
# 3. 生化特性
- 与**黄体生成素受体（LHR）**高度亲和
- 生物效应 ≈ LH 的超长效模拟物
- 半衰期显著长于 LH（关键差异点）
# 4. 适应症、药理毒理
- 适应症（男性）
  - 低促性腺激素性性腺功能减退（HH）
  - 男性不育（刺激睾丸内源性睾酮与精子发生）
  - 睾丸未降（儿童为主）
  - AAS/外源雄激素停用后的睾丸功能恢复
- 毒理
  - 无直接细胞毒性
  - 毒性主要来自内分泌过度刺激
# 5. 药代动力学、起效时间
- 给药途径：肌注 / 皮下注射
- 生物利用度：高
- 起效时间：
  - 睾酮上升：24–72 小时
  - 精子发生：6–12 周
- 血浆半衰期：约 24–36 小时
- 代谢/排泄：肾脏为主
# 6. 常见剂量、给药方式（成年男性）
- 功能减退/促睾：
  - 500–2000 IU
  - 每周 2–3 次
- 联合hMG/FSH促生精：
  - hCG 1500 IU，2–3 次/周
- 给药方式：
  - 皮下注射（更平稳）
  - 肌肉注射（传统）
# 7. 副作用、药物过量
- 常见副作用
  - 雌二醇升高（芳香化增加）
  - 乳房胀痛 / 男性乳腺发育
  - 水钠潴留
  - 情绪波动
- 过量风险
  - Leydig细胞脱敏
  - 睾丸反应性下降
  - 雌激素占优势（性欲下降、勃起质量下降）
# 8. 常见同分异构体和类似物质
- LH（黄体生成素）：半衰期短，但生理更精细
- 重组hCG（r-hCG）：纯度更高，免疫反应低
- 促性腺激素释放激素（GnRH）：作用层级更上游
# 9. 在人体内的整体作用
- ↑ 睾丸 Leydig 细胞 → ↑ 睾酮
- ↑ 睾丸体积
- 维持/恢复睾丸内睾酮浓度（对精子发生至关重要）
- 不抑制下丘脑（但长期可反馈抑制）
# 10. 内分泌相关的激素
- ↑ 睾酮
- ↑ 双氢睾酮（间接）
- ↑ 雌二醇（芳香化）
- ↓ LH、FSH（负反馈）
# 11. 对脂肪代谢的作用
- 通过睾酮↑ →
  - ↑ 脂肪动员
  - ↓ 内脏脂肪
- hCG本身不直接“减脂”
# 12. 对血压的作用
- 可轻度升高血压（与水钠潴留相关）
- 高剂量或雌激素升高时更明显
# 13. 对消化系统的作用（急性）
- 无直接作用
- 少数人出现：
  - 恶心
  - 食欲改变（多为内分泌间接效应）
# 14. 对神经系统的调节机制
- 通过睾酮/雌激素改变：
  - 性欲
  - 情绪
  - 动机与攻击性
- 过量 → 易烦躁、焦虑
# 15. 对生殖系统的作用
- ↑ 精子发生能力（需时间）
- ↑ 射精量（间接）
- ↑ 前列腺活性（长期需监测）
# 16. 对皮肤的作用
- ↑ 皮脂分泌
- 可能加重痤疮
- 毛发生长增强（个体差异）
# 17. 过多或不足时的常用干预
## hCG过量
- 芳香化酶抑制剂（如：来曲唑、阿那曲唑）
- 降低剂量或延长给药间隔
## hCG不足/反应差
- 联合 hMG / rFSH
- 改用 GnRH 脉冲治疗
## 对比非孕产期女性：
- 女性多用于促排卵触发，剂量单次、高峰短
- 性更强调低剂量、长期节律性
# 18. 中医八纲辨证与五行归经
- 八纲：多属 肾阳虚 / 肾精不足
- 五行归经：
  - 肾经（主生殖）
  - 肝经（调节性轴、情志）
- 对应功效类比：
  - 温补肾阳
  - 益精填髓

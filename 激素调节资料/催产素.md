# 1. 常见别名、物理性质、CAS、溶解度
- 别名：缩宫素、催生素、后叶催产素
- 英文：Oxytocin
- CAS：50-56-6
- 分子式 / 分子量：C₄₃H₆₆N₁₂O₁₂S₂ / 1007.19
- 物理性质：白色或类白色冻干粉末；多肽（九肽）
- 溶解度：
  - 水：易溶
  - 有机溶剂（乙醇、氯仿等）：几乎不溶
  - 生理盐水/缓冲液：可溶（临床常用）
# 2. 化学性质、光热稳定性
- 性质：含二硫键的环状九肽，易被蛋白酶水解
- 稳定性：
  - 对光、热、酸碱均较不稳定
  - 高温、强光、极端 pH 易失活
  - 冻干、避光、低温保存可提高稳定性
# 3. 生化特性
- 下丘脑室旁核、视上核合成
- 经垂体后叶释放入血
- 亦作为中枢神经递质/调质在脑内发挥作用
- 受雌激素、神经刺激、心理社交因素调节
# 4. 适应症、药理毒理
- 临床适应症（主要针对女性）：
  - 引产、催产
  - 产后子宫收缩不良、产后出血
- 男性/非产科用途：
  - 研究性：社交障碍、自闭谱系、焦虑障碍（非一线）
- 毒理：
  - 治疗窗较窄
  - 过量 → 水潴留、低钠血症、心血管反应
# 5. 药代动力学、起效时间
- 给药途径：静脉、肌肉、鼻腔（实验/研究）
- 起效时间：
  - 静脉：1–5 分钟
  - 肌注：3–10 分钟
- 半衰期：约 3–5 分钟
- 代谢：肝、肾及血浆中催产素酶快速降解
# 6. 常见剂量、给药方式
- 静脉滴注（产科）：0.5–20 mU/min（逐渐递增）
- 肌注：5–10 IU
- 男性：无常规治疗剂量，仅限研究或特殊指征
# 7. 副作用、药物过量
- 常见：
  - 面潮红、头痛
  - 恶心、呕吐
- 严重：
  - 低钠血症（水中毒）
  - 心律失常、低血压或反射性心动过速
- 过量处理：
  - 停药
  - 限水、纠正电解质
  - 对症支持
# 8. 常见同分异构体和类似物质
- 加压素（AVP，抗利尿激素）：
  - 结构高度相似（仅 2 个氨基酸差异）
  - 更强的血管收缩、抗利尿作用
- 卡贝缩宫素（Carbetocin）：
  - 长效类似物
  - 半衰期显著延长
# 9. 在人体内的整体作用
- 外周：平滑肌收缩（子宫、乳腺）
- 中枢：
  - 调节信任、依恋、亲社会行为
  - 降低应激反应（与 HPA 轴拮抗）
# 10. 内分泌相关激素
- 与 加压素、催乳素、雌激素、皮质醇存在交互
- 心理压力 ↑ → 催产素释放受抑
- 社交、亲密行为 → 催产素 ↑
# 11. 对脂肪代谢的作用
- 间接促进脂肪动员
- 可能改善胰岛素敏感性（研究阶段）
- 中枢作用与食欲调节有关
# 12. 对血压的作用
- 轻度血管扩张
- 大剂量或快速静注 → 血压波动
- 与加压素系统平衡密切
# 13. 对消化系统的作用（急性）
- 可能轻度促进胃肠平滑肌收缩
- 偶见恶心、腹部不适
- 无明确长期消化系统毒性
# 14. 对神经系统的调节机制和作用
- 作用于边缘系统、杏仁核、前额叶
- 减轻恐惧反应、增强情绪识别
- 在男性中可能：
  - 提升亲密感
  - 降低社交焦虑（个体差异大）
# 15. 对生殖系统的作用
- 男性：
  - 参与射精反射
  - 影响前列腺、输精管平滑肌张力
- 不直接提升睾酮
- 与性行为后的“情感绑定”相关
# 16. 对皮肤的作用
- 间接改善应激相关皮肤反应
- 可能促进伤口愈合（动物与早期研究）
# 17. 过多或不足时的治疗药物
- 催产素过多（功能性）：
  - 无特异拮抗剂，主要对症处理
- 催产素不足（临床少见）：
  - 外源性催产素
- 对比非孕产期女性：
  - 女性更关注子宫反应
  - 男性更关注神经、心血管反应
# 18. 中医八纲辨证与五行归经分析
- 八纲：多属「虚中夹实」「情志内伤」
- 五行归经（类比）：
  - 心（情志、神明）
  - 肝（疏泄、情绪调节）
- 中医理解：
  - 情志调和 → 气血和畅 → “类催产素效应”增强
  - 与“心肾相交、肝气疏达”关系密切

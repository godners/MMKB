# 1. 常见别名、物理性质、CAS、溶解度
- 别名：睾酮丙酸酯、丙酸睾丸素、Testosterone Propionate、TP
- 化学名：17β-Hydroxyandrost-4-en-3-one propionate
- CAS：57-85-2
- 分子式 / 分子量：C₂₂H₃₂O₃ / 344.49
- 外观：白色或类白色结晶性粉末
- 熔点：118–123 ℃
- 溶解度：
  - 水：几乎不溶
  - 乙醇、氯仿、丙酮：易溶
  - 油剂（芝麻油、棉籽油）：高度可溶（用于注射）
# 2. 化学性质、光热稳定性
- 性质：甾体类脂溶性酯
- 光稳定性：对强光敏感，需避光
- 热稳定性：常温稳定，高温下可水解
- 水解：体内被酯酶迅速水解为游离睾酮
# 3. 生化特性
- 属短链睾酮酯
- 注射后迅速释放睾酮
- 高峰血药浓度出现快、波动大
- 与雄激素受体（AR）高亲和力结合
# 4. 适应症、药理毒理
## 适应症：
- 男性性腺功能减退（原发/继发）
- 延迟青春期（特定医学情形）
- 某些消耗性疾病中的雄激素补充（历史用途）
## 毒理特点：
- 非17α-烷基化 → 肝毒性相对低
- 长期或大剂量 → 内分泌轴抑制明显
# 5. 药代动力学、起效时间
- 给药方式：深部肌内注射
- 起效：12–24小时
- 血药峰值：约24–48小时
- 半衰期：≈ 0.8–2天
- 作用持续：2–4天
- 代谢：肝脏 → 还原/氧化 → 肾排泄
# 6. 常见剂量、给药方式
- 常用剂量：25–50 mg / 次，每 2–3天 注射一次
- 给药途径：肌内注射（臀大肌/股外侧肌）
- 特点：需频繁给药以维持稳定水平
# 7. 副作用、药物过量
## 常见副作用：
- 水钠潴留、体重增加
- 痤疮、皮脂溢出
- 情绪波动、易激惹
- 红细胞增多（HCT↑）
## 过量/长期：
- 下丘脑-垂体-性腺轴抑制
- 睾丸萎缩、精子生成下降
- 雌激素相关反应（乳腺增生）
# 8. 常见同分异构体和类似物质
- 睾酮本体：生理活性形式
- 庚酸/环戊丙酸/十一酸睾酮：酯链越长 → 起效慢、维持久
- 南德酮酯：雄激素/合成代谢分离度更高
# 9. 在人体内的整体作用
- 恢复或增强男性第二性征
- 提高蛋白合成、肌肉量
- 提升性欲与勃起质量
- 抑制内源性睾酮分泌（负反馈）
# 10. 内分泌相关的激素
- ↑ 血清睾酮
- ↓ LH、FSH
- 部分转化为 雌二醇（E2）
- 间接影响 DHT（经5α还原）
# 11. 对脂肪代谢的作用
- 抑制脂肪细胞分化
- ↑ 基础代谢率
- ↓ 内脏脂肪比例
- 长期可改善胰岛素敏感性（剂量依赖）
# 12. 对血压的作用
- 轻度升高血压风险（钠水潴留）
- 红细胞增多 → 血液黏稠度↑
- 心血管风险与剂量/波动相关
# 13. 对消化系统的作用（急性）
- 无直接刺激
- 个别可出现食欲变化
- 长期：肝酶轻度升高（较少见）
# 14. 对神经系统的调节机制
- 作用于下丘脑和边缘系统
- ↑ 动机、性欲、自信
- 波动大时 → 易焦躁、攻击性增强
# 15. 对生殖系统的作用
- 短期：性功能增强
- 中长期：
  - 精子生成↓
  - 睾丸体积缩小
  - 生育力下降（可逆）
# 16. 对皮肤的作用
- 皮脂腺活性↑
- 痤疮、多油皮
- 促进体毛、胡须生长
- 加速雄激素性脱发（遗传相关）
# 17. 过多或不足时的常用治疗
- 不足：
  - 其他睾酮酯（庚酸、环戊丙酸）
  - 促性腺激素（HCG）
- 过多/抑制恢复：
  - SERM（如克罗米芬）
- 对比非孕产期女性：女性更倾向低剂量、非酯化或避免使用
# 18. 中医八纲辨证与五行归经
- 属性：温热、助阳
- 八纲：偏 阳、热、实
- 归经：肾（主生殖、精）、肝（疏泄、性欲）
- 证型对应：
  - 肾阳虚、命门火衰者短期可“类补”
  - 阴虚火旺者 → 易助火生燥，不宜

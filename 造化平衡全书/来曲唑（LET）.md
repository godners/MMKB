# 1. 常见别名、物理性质、CAS、溶解度
- 别名：来曲唑、来曲唑片、芳香化酶抑制剂（非甾体类）
- 英文名：Letrozole、LET
- CAS：112809-51-5
- 分子式：C₁₇H₁₁N₅
- 分子量：285.3
- 物理性状：白色或类白色结晶性粉末
- 脂溶性：较强
- 水中溶解度：极低
- 有机溶剂：易溶于乙醇、丙酮、二氯甲烷等
# 2. 化学性质、光热稳定性
- 化学类型：三唑类非甾体芳香化酶抑制剂
- 机制特性：可逆性、竞争性抑制 CYP19A1
- 光稳定性：稳定
- 热稳定性：常温稳定，避免高湿
# 3. 生化特性
- 靶点：芳香化酶（Aromatase, CYP19）
- 直接作用：阻断睾酮 / 雄烯二酮 → 雌二醇 / 雌酮
- 抑制效率：雌激素合成 ↓ 90–98%（剂量依赖）
# 4. 适应症、药理毒理
- 批准适应症（女性）
  - 绝经后 ER+ 乳腺癌
- 男性常见“非说明书”用途
  - 男性高雌激素血症
  - 继发性低睾酮（T/E₂ 比值异常）
  - 男性不育（雌激素抑制型）
  - 青少年或成人特发性男性乳房发育
- 毒理要点
  - 非肝毒性为主
  - 长期使用 → 骨代谢异常风险
# 5. 药代动力学、起效时间
- 口服吸收率：≈99%
- 生物利用度：极高
- 达峰时间（Tmax）：1–2 h
- 半衰期：≈48 h
- 稳态浓度：2–6 周
- 代谢：肝脏（CYP3A4、CYP2A6）
- 排泄：尿液为主（代谢物）
# 6. 常见剂量、给药方式
- 成年男性（临床经验）
  - 0.25 mg 每日
  - 0.25–0.5 mg 隔日
  - 极少超过 1 mg / 日  
- 男性用药强调：低剂量、间断、监测 E₂
# 7. 副作用、药物过量
## 常见副作用
- 雌激素过低综合征
  - 性欲下降
  - 情绪低落、焦虑
  - 关节痛、骨痛
- 血脂异常（HDL↓）
## 过量风险
- 骨密度下降
- 关节退变
- 中枢神经兴奋/抑制交替
# 8. 同分异构体和类似物质对比
- 来曲唑：非甾体，抑制最强
- 阿那曲唑：非甾体，稳定温和
- 依西美坦：甾体，不可逆
- 氟维司群：ER拮抗，非 AI
# 9. 在人体内的整体作用
- T/E₂ 比值 ↑
- 下丘脑–垂体负反馈解除
- LH / FSH ↑
- 睾丸 Leydig 细胞内源性睾酮合成 ↑
# 10. 内分泌相关激素变化
- ↓ 雌二醇（E₂）
- ↑ 睾酮（T）
- ↑ LH、FSH
- ↓ SHBG（部分个体）
# 11. 对脂肪代谢的作用
- 雌激素过低 →
  - 脂肪动员 ↓
  - 内脏脂肪 ↑ 风险
- 短期：可能改善雄激素优势型肥胖
- 长期：反而不利
# 12. 对血压的作用
- 直接影响不明显
- 长期低雌 →
  - 血管顺应性下降
  - 血压波动性 ↑
# 13. 对消化系统的作用（急性）
- 少数人出现
  - 恶心
  - 腹胀
  - 食欲下降
- 通常轻微、可耐受
# 14. 对神经系统的调节机制
- 雌激素↓ →
  - 5-HT 调节能力下降
  - GABA 稳定性下降
- 表现为
  - 易怒
  - 睡眠质量下降
  - 焦虑感增强
# 15. 对生殖系统的作用
正向
- 精子发生 ↑（特定人群）
- 睾丸内 T ↑
负向
- 长期雌激素过低 →
  - 性欲下降
  - 勃起质量下降
# 16. 对皮肤的作用
- 皮脂分泌 ↓
- 皮肤偏干
- 胶原合成 ↓（长期）
# 17. 过多或不足时的治疗
- 雌激素过低（男性）
  - 减量或停药
  - 低剂量 E₂ 或 SERM（如他莫昔芬）
- 女性对比
  - 非孕产期女性更易骨量流失
  - 常需联合补钙 + 维生素 D
# 18. 中医八纲与五行归经
- 八纲：多属 阴虚阳亢 / 肝肾阴虚
- 五行归经：肝、肾
- 病机类比：阴液被夺，阳失制约
- 中医调理方向：
  - 滋肾阴、柔肝、潜阳
  - 常配伍：熟地、山萸肉、女贞子、枸杞子

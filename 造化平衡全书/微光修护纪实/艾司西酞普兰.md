# 艾司西酞普兰

## 1. 基本信息

- 常见别名：艾司西酞普兰、左旋西酞普兰、Escitalopram oxalate
- CAS编号：128196-01-0（游离碱）；219861-08-2（草酸盐）
- 分子式：C₂₀H₂₁FN₂O
- 白色或类白色结晶性粉末
- 易溶于水、甲醇
- 在乙醇中中等溶解，在非极性溶剂中溶解度较低
- 溶解性：
  + 水：良好（盐形式更佳）
  + 有机溶剂：极性溶剂优于非极性

## 2. 化学性质与稳定性

- 属于选择性5-羟色胺再摄取抑制剂（SSRI）
- 光稳定性：较稳定，但长期强光下可缓慢降解
- 热稳定性：常温稳定，高温下可能分解
- pKa ≈ 9.8（弱碱性）

## 3. 生化特性

- 高选择性抑制SERT（5-HT转运体）
- 几乎无作用于：
  + 多巴胺转运体（DAT）
  + 去甲肾上腺素转运体（NET）
- 对受体亲和力极低（抗胆碱、抗组胺作用弱）

## 4. 适应症与药理毒理

- 适应症：
  + 抑郁障碍（MDD）
  + 广泛性焦虑障碍（GAD）
  + 社交焦虑、惊恐障碍（部分地区批准）
- 药理作用：
  + 增加突触间隙5-HT浓度
  + 改善情绪、降低焦虑
- 毒理：
  + 治疗窗较宽
  + 过量时主要为中枢和心脏毒性（QT延长）

## 5. 药代动力学

- 口服生物利用度：≈80%
- 起效时间：
  + 抗焦虑：1–2周
  + 抗抑郁：2–4周（完全效果4–6周）
- 半衰期：约27–32小时
- 代谢：肝脏（CYP2C19、CYP3A4、CYP2D6）
- 排泄：肾+肝

## 6. 常见剂量与给药

- 起始剂量：10 mg/日
- 常规剂量：10–20 mg/日
- 最大剂量：20 mg/日
- 给药方式：口服，每日一次（早或晚）

## 7. 副作用与过量

- 常见副作用：
  + 性功能障碍（性欲下降、射精延迟）
  + 恶心
  + 失眠或嗜睡
  + 出汗增加
  + 轻度体重变化
- 严重不良反应：
  + 5-HT综合征
  + QT间期延长
  + 低钠血症（SIADH）
- 过量表现：
  + 心动过速
  + 头晕、震颤
  + 意识障碍（严重时）

## 8. 同分异构体与类似物

- 西酞普兰（Citalopram）
  + 为外消旋体（R+S）
  + S-构型即艾司西酞普兰为活性成分
- 生化差异：
  + Escitalopram活性更强
  + 副作用更少（尤其心脏毒性略低）

## 9. 人体整体作用

- 提升情绪稳定性
- 降低焦虑、紧张
- 改善认知偏负性加工
- 长期使用可促进神经可塑性（BDNF上调）

## 10. 内分泌影响

- 增加催乳素（轻度）
- 抑制下丘脑-垂体-性腺轴（轻微）
- 对皮质醇：
  + 急性↑（轻度）
  + 慢性趋于正常化

## 11. 对脂肪代谢

- 短期：食欲下降 → 体重下降
- 长期：可能轻度体重增加
- 对胰岛素敏感性影响较小

## 12. 对血压作用

- 一般无显著影响
- 个别患者：轻度降低交感张力 → 血压略降

## 13. 对消化系统（急性）

- 恶心（最常见）
- 腹泻或便秘
- 食欲变化
- 机制：5-HT在肠道受体激活

## 14. 神经系统作用机制

- 增强5-HT神经传递
- 下调：杏仁核过度反应（焦虑核心）
- 上调：前额叶控制能力
- 长期：促进神经发生（海马）

## 15. 生殖系统作用

- 显著影响：
  + 延迟射精（常见）
  + 性欲下降
  + 勃起功能轻度影响
- 机制：
  + 5-HT抑制多巴胺通路
  + 抑制射精反射弧

## 16. 对皮肤作用

- 出汗增加
- 少数：皮疹
- 极少：光敏反应

## 17. 过多或不足的治疗对比

- 过量，治疗：
  + 苯二氮卓类（控制兴奋）
  + 赛庚啶（5-HT拮抗）
- 不足/疗效差，加用：
  + 安非他酮（增强DA/NE）
  + 米氮平
- 女性对比：
  + 女性更易出现低钠血症
  + 男性性副作用更突出

## 18. 中医分析

- 八纲辨证：
  + 多属“虚实夹杂”
  + 肝郁气滞为主
- 五行归经：入肝、心
- 作用倾向：
  + 疏肝解郁
  + 安神定志
- 副作用对应：
  + 性功能下降 → 肾气受抑（间接）

# 1. 常见别名、物理性质、CAS、溶解度
- 通用名：阿那曲唑
- 英文名：Anastrozole、ANA
- 商品名：Arimidex（阿瑞咪德）
- CAS号：120511-73-1
- 分子式：C₁₇H₁₉N₅
- 分子量：293.37
- 物理性质
  - 白色或类白色结晶性粉末
  - 无明显气味
  - 熔点：≈81–82 ℃
  - LogP ≈ 2.1（中等脂溶性）
- 溶解度
  - 水：几乎不溶
  - 乙醇、甲醇：微溶
  - DMSO、DMF：可溶
  - 脂质环境中吸收良好（口服）
# 2. 化学性质、光热稳定性
- 非甾体类芳香化酶抑制剂
- 通过三唑环与芳香化酶血红素铁结合
- 不发生共价结合（可逆抑制）
- 常温、避光条件稳定
- 对光不敏感
- 正常药用条件下热稳定
- 不易水解、氧化
# 3. 生化特性
- 选择性抑制芳香化酶（CYP19A1）
- 阻断：
  - 雄烯二酮 → 雌酮
  - 睾酮 → 雌二醇
- 对其他CYP酶影响弱（高选择性）
# 4. 适应症、药理毒理
- 正规适应症（说明书）
  - 绝经后女性 ER+ 乳腺癌
- 男性常见“实际使用场景”（非说明书）
  - 男性雌激素过高
  - 睾酮替代治疗（TRT）伴发雌激素升高
  - 男性乳腺发育
  - 某些性腺功能异常导致的 E2 偏高
- 毒理特点
  - 非细胞毒性
  - 长期使用可引起低雌激素综合征
# 5. 药代动力学、起效时间
- 吸收
  - 口服吸收快
  - 生物利用度高（>80%）
- 分布
  - 广泛分布
  - 可透过血脑屏障（有限）
- 代谢
  - 肝脏代谢（N-去烷基化、羟化）
  - 主要经 CYP3A4、CYP2C9
- 半衰期
  - ≈ 46–50 小时
- 起效时间
  - 24–48 小时雌二醇开始下降
  - 5–7 天达稳态抑制
# 6. 常见剂量、给药方式（成年男性）
**男性多为 超说明书使用**
- 常见剂量：0.25–1 mg / 次
- 给药频率：每日一次 或 隔日一次
- 给药方式：口服  
**男性对雌激素极其敏感，低剂量即有效**
# 7. 副作用、药物过量
- 常见副作用（男性）
  - 关节僵硬、骨痛
  - 性欲下降（雌激素过低）
  - 情绪低落、易怒
  - 皮肤干燥
  - 血脂异常（HDL ↓）
- 过量或长期抑制
  - 骨密度下降
  - 勃起质量下降
  - 心血管风险上升
  - 精子质量下降（长期）
# 8. 常见同分异构体与类似物质
**同类芳香化酶抑制剂**
- 来曲唑（Letrozole）
  - 抑制更强
  - 更易导致雌激素过低
- 依西美坦（Exemestane）
  - 类固醇型、不可逆
  - 对骨骼影响略小
# 9. 在人体内的整体作用
- 显著降低外周与睾丸内雌激素
- 提高 T/E2 比值
- 改变下丘脑-垂体-性腺轴反馈
- 长期可重塑内分泌稳态（正或负）
# 10. 内分泌相关激素
- ↓ 雌二醇（E2）
- ↑ 睾酮（轻-中度，间接）
- ↑ LH、FSH（反馈性）
- 对 DHT 影响不直接
# 11. 对脂肪代谢的作用
- 抑制雌激素 →
  - 皮下脂肪减少
  - 内脏脂肪可能 ↑
- 长期：胰岛素敏感性可能下降
# 12. 对血压的作用
- 无直接升压作用
- 雌激素过低 → 血管顺应性下降
- 长期可能轻度增加心血管风险
# 13. 对消化系统的作用（急性）
- 偶见：
  - 恶心
  - 食欲下降
- 无明显胃黏膜刺激性
# 14. 对神经系统的调节机制
- 雌激素下降 →
  - 5-HT、多巴胺活性下降
- 表现为：
  - 情绪平淡
  - 动机下降
  - 睡眠质量下降（部分人）
# 15. 对生殖系统的作用
- 抑制雌激素 →
  - 男性乳腺萎缩
  - 性欲、射精量可能下降
- 短期可能改善 HPG 轴
- 长期过抑制 → 精子质量下降
# 16. 对皮肤的作用
- 皮脂分泌下降
- 皮肤干燥、弹性下降
- 痤疮可能减少（与睾酮平衡有关）
# 17. 过多或不足时的治疗药物
- 雌激素过高
  - 阿那曲唑
  - 来曲唑（更强，不首选）
- 雌激素过低
  - 停药或减量
  - 必要时短期补充雌激素
  - 男性慎用，女性更常见
# 18. 中医八纲辨证与五行归经
- 八纲
  - 性属：苦、燥、偏寒
  - 偏向：抑阴助阳（但易伤阴）
- 五行归经（类比）
  - 归肝、肾
  - 肝：调激素代谢
  - 肾：生殖、精血
- 中医类比病机
  - 痰湿化热、冲任失衡
  - 用之过度 → 肾阴不足、筋骨失养

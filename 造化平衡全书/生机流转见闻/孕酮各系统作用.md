- 另请参阅：**[孕酮（P）](../生命韵律之源/孕酮（P）.md)**
- 孕酮（Progesterone，P） 是成年男性体内的一种类固醇激素，主要由肾上腺和睾丸Leydig细胞少量分泌，正常血清水平通常为 <1 ng/mL（或 <3.18 nmol/L，参考范围常为 0.1–1.0 ng/mL，实验室间略有差异；远低于女性黄体期）。
- 在男性中，孕酮被视为“被遗忘的激素”，其生理作用不如睾酮、雌二醇显著，但作为前体激素（precursor），它参与睾酮合成、精子功能调节、神经保护，并对抗雌激素过度（estrogen dominance）。
- 许多作用通过其代谢物（如allopregnanolone）作为神经甾体（neurosteroids）介导，快速非基因组效应明显。
- 过多（高孕酮血症，hyperprogesteronemia，常 >1–1.4 ng/mL以上；男性罕见，可由先天性肾上腺增生、某些肿瘤、药物或外源补充引起）
- 过少（低孕酮血症，hypoprogesteronemia，<0.2–0.3 ng/mL；常见于衰老、慢性应激、肾上腺/睾丸功能低下）都会扰乱激素平衡，常与雌激素/睾酮比例失调相关。
# 1. 骨骼系统  
- 正常作用：间接支持骨密度（通过促进睾酮合成、对抗雌激素过度导致的骨丢失）。  
- 过多时影响：通常无直接负面；极高时可能干扰整体激素平衡，间接影响骨代谢。  
- 过少时影响：骨密度下降、骨质疏松风险增加（与低睾酮/雌激素相对优势相关）。
# 2. 肌肉系统  
- 正常作用：作为睾酮前体，支持肌肉质量维持和力量。  
- 过多时影响：可能导致肌肉相对减少、体型女性化改变（雌激素优势）。  
- 过少时影响：肌肉量和力量下降、疲劳增加、易肌肉流失。
# 3. 循环系统（心血管与血液）  
- 正常作用：调节血管功能、可能保护心血管（部分通过神经甾体作用）。  
- 过多时影响：高水平与血糖调控失调相关，增加前驱糖尿病/2型糖尿病风险；可能液体潴留（矿物皮质激素样效应）。  
- 过少时影响：心血管保护减弱；部分研究显示低孕酮与腹主动脉瘤风险相关。
# 4. 神经系统（脑与情绪）  
- 正常作用：其代谢物（如allopregnanolone）作用于GABA_A受体，具有镇静、抗焦虑、改善睡眠、神经保护作用；支持情绪稳定、认知功能。  
- 过多时影响：情绪波动、疲劳、性欲减退；可能过度镇静。  
- 过少时影响：焦虑、抑郁、睡眠障碍、情绪不稳、认知下降。
# 5. 表皮系统（皮肤与毛发）  
- 正常作用：无直接显著作用；间接通过激素平衡影响皮肤/毛发。  
- 过多时影响：皮肤细腻、毛发减少、可能出现女性化特征。  
- 过少时影响：脱发、皮肤变化（与低睾酮相关）。
# 6. 免疫系统（含淋巴系统）  
- 正常作用：调节免疫应答（抑制某些巨噬细胞/T细胞功能）；参与炎症平衡。  
- 过多时影响：可能免疫抑制增强。  
- 过少时影响：免疫功能低下、感染易感性增加。
# 7. 消化系统（含代谢）  
- 正常作用：参与代谢稳态、胰岛素敏感性调节；可能影响食欲/体重。  
- 过多时影响：食欲增加、体重增加；血糖升高风险（与糖尿病相关）。  
- 过少时影响：代谢减慢、腹部脂肪增加、胰岛素抵抗。
# 8. 泌尿系统  
- 正常作用：间接支持肾功能；可能影响前列腺（孕酮受体存在于前列腺组织）。  
- 过多时影响：前列腺增生风险（BPH/前列腺癌相关）；肾功能影响有限。  
- 过少时影响：前列腺问题风险升高（与雌激素优势相关）。
# 9. 生殖系统  
- 正常作用：促进精子生成（spermiogenesis）、精子获能/顶体反应（capacitation/acrosome reaction）；支持Leydig细胞睾酮合成；调节性欲/勃起。  
- 过多时影响：性欲减退、勃起功能障碍（ED）、精子质量下降、不育；可能男性乳房发育（gynecomastia）。  
- 过少时影响：性欲低下、ED、精子减少、不育；睾丸功能减弱。
# 10. 内分泌系统  
- 正常作用：作为睾酮/皮质醇前体，参与下丘脑-垂体-性腺轴调节；对抗雌激素过度。  
- 过多时影响：抑制GnRH/LH，继发低睾酮；形成恶性循环。  
- 过少时影响：雌激素相对优势（estrogen dominance），加重低睾酮、代谢综合征。
# 11. 呼吸系统  
- 正常作用：间接支持呼吸（通过神经调节、改善睡眠呼吸）。  
- 过多时影响：可能增加呼吸暂停风险（镇静效应）。  
- 过少时影响：耐力下降、疲劳加重，无明显直接影响。
# 12. 淋巴系统（与免疫高度重叠）  
- 正常作用：参与免疫细胞调节。  
- 过多/过少时影响：类似免疫系统，平衡失调。
# 注意事项  
- 男性孕酮异常远不如女性常见；过多多为病理（如CAH、肿瘤、外源补充）；过少常见于衰老、应激、肾上腺/睾丸问题，常与“雌激素优势”症状重叠（疲劳、低性欲、体脂增加、男性乳房发育）。  
- 孕酮在男性多通过非基因组途径（膜受体、神经甾体）发挥快速作用；补充孕酮（如生物同型激素疗法）在某些功能医学中用于平衡激素，但证据有限，需谨慎。

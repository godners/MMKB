- 另外参见 **[丙酸睾酮（TP）](../生命韵律之源/丙酸睾酮（TP）.md)**
- 针对成年男性，其在生理剂量下维持男性特征和代谢功能；
  - 过多（超生理剂量，如滥用或不当TRT）常导致副作用和反馈抑制；
  - 过少（相当于低睾酮/性腺功能减退症）则出现典型缺乏症状。
# 1. 骨骼系统  
- 正常作用：促进骨矿物质沉积，维持骨密度和强度，预防骨丢失。  
- 过多时影响：成人骨骺已闭合，通常无额外骨生长益处；可能间接增加骨代谢负担或与心血管风险关联。  
- 过少时影响：骨密度下降（骨质疏松），骨折风险显著增加。
# 2. 肌肉系统  
- 正常作用：刺激蛋白合成、增长激素释放，促进肌肉质量和力量增加。  
- 过多时影响：肌肉过度增生（常见于体型塑造），但伴随水肿、关节负担增加。  
- 过少时影响：肌肉质量和力量下降，易疲劳、体型松弛。
# 3. 循环系统（心血管与血液）  
- 正常作用：刺激骨髓产生红细胞，维持血红蛋白水平，支持氧运输。  
- 过多时影响：红细胞增多症（多血症）、血压升高、血栓/心脏病风险增加、液体潴留、胆固醇改变。  
- 过少时影响：可能贫血、红细胞减少、心血管健康受损（能量低、耐力差）。
# 4. 神经系统（脑与情绪）  
- 正常作用：影响多巴胺等神经递质，支持认知、动机、自信和性欲相关行为。  
- 过多时影响：攻击性/易怒增加、情绪波动、头痛、失眠、判断力受损。  
- 过少时影响：抑郁、注意力不集中、疲劳、动机低下、记忆力下降。
# 5. 表皮系统（皮肤与毛发）  
- 正常作用：促进面部、体毛生长，调节皮脂分泌。  
- 过多时影响：痤疮、油性皮肤、雄激素性脱发（秃顶加速）、体毛过多。  
- 过少时影响：体毛减少、皮肤干燥、毛发生长减缓。
# 6. 免疫系统（含淋巴系统）  
- 正常作用：调节炎症反应和免疫平衡（雄激素受体存在于淋巴组织）。  
- 过多时影响：可能削弱免疫应答（如疫苗反应差），增加感染易感性或炎症失调。  
- 过少时影响：炎症增加、伤病恢复缓慢、慢性疲劳或免疫功能低下。
# 7. 消化系统（含代谢）  
- 正常作用：间接促进脂肪代谢、胰岛素敏感性和热量消耗（代谢“恒温器”作用）。  
- 过多时影响：食欲可能增加、体重/脂肪变化、水肿；消化道直接影响有限。  
- 过少时影响：体脂（尤其是腹部）增加、代谢减慢、易肥胖。
# 8. 泌尿系统  
- 正常作用：维持前列腺等器官功能（但刺激其生长）。  
- 过多时影响：前列腺增大（良性前列腺增生），排尿困难、尿频；注射相关水肿加重。  
- 过少时影响：前列腺功能相对减弱，但症状不突出，主要与整体低能量相关。
# 9. 生殖系统  
- 正常作用：维持性欲、勃起功能、精子生成和睾丸/前列腺健康。  
- 过多时影响：反馈抑制垂体（LH/FSH下降），导致睾丸萎缩、精子减少、不育、性欲异常波动。  
- 过少时影响：性欲低下、勃起功能障碍（ED）、精子减少、不育、睾丸缩小。
# 10. 内分泌系统  
- 正常作用：通过下丘脑-垂体-睾丸轴反馈调节自身及雌激素/其他激素平衡。  
- 过多时影响：强烈抑制内源性睾酮产生（停药后恢复慢）、雌激素转化增加（可能男性乳腺发育）。  
- 过少时影响：形成恶性循环，加重性腺功能减退，影响甲状腺/皮质醇等平衡。
# 11. 呼吸系统  
- 正常作用：间接通过红细胞提升氧运输能力，无直接作用于肺或气道。  
- 过多时影响：增加睡眠呼吸暂停风险（液体潴留或中枢抑制）、呼吸不畅。  
- 过少时影响：间接导致耐力/氧利用下降、疲劳加重，无明显直接呼吸道症状。
# 12. 淋巴系统（与免疫系统高度重叠）  
- 正常作用：参与免疫调节，影响淋巴细胞活性和炎症控制。  
- 过多时影响：可能加剧免疫抑制，淋巴功能失调风险升高。  
- 过少时影响：炎症易慢性化、淋巴回流或恢复能力减弱。
# 注意事项：  
- 睾酮作为外源性药物，额外风险包括注射部位疼痛、激素波动大（需频繁注射）、芳香化副作用（需可能合用芳香化酶抑制剂）。
- 过多常见于滥用，过少多为年龄/疾病导致。

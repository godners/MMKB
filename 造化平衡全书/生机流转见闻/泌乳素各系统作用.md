- 泌乳素（Prolactin，PRL） 是由垂体前叶分泌的一种多肽激素，在成年男性中正常水平通常较低（<20 ng/mL 或 <400–500 mIU/L，具体参考值依实验室略有差异，常为 3–18 ng/mL）。
- 男性中泌乳素的主要生理作用包括：调节生殖轴（适量时支持睾酮分泌和精子生成）、参与应激反应（作为“应激激素”之一）、增强免疫调节、维持代谢平衡等。它与多巴胺（抑制因子）负反馈调控，受应激、性活动、药物等影响。
- 过多（高泌乳素血症，hyperprolactinemia，常 >20–25 ng/mL，严重时数百甚至数千 ng/mL）最常见原因为垂体泌乳素瘤（prolactinoma）、药物（如抗精神病药、多巴胺拮抗剂）、甲减、慢性肾衰等；会导致强烈负反馈抑制下丘脑-垂体-性腺轴（GnRH/LH/FSH下降 → 睾酮降低）。  
- 过少（低泌乳素血症，hypoprolactinemia，罕见，通常 <3–5 ng/mL）多为垂体损伤、遗传或药物（如多巴胺激动剂过量），临床意义较小，但可能影响某些生殖/免疫功能。
# 1. 骨骼系统  
- 正常作用：适量泌乳素间接支持骨代谢（通过维持睾酮水平）。  
- 过多时影响：继发低睾酮 → 骨密度下降、骨质疏松、骨折风险增加。  
- 过少时影响：通常无明显直接影响；极低可能与整体垂体功能低下相关，骨保护减弱。
# 2. 肌肉系统  
- 正常作用：间接通过支持睾酮维持肌肉质量。  
- 过多时影响：低睾酮导致肌肉量减少、力量下降、疲劳。  
- 过少时影响：可能与精囊/附属性腺功能减弱相关，肌肉影响不突出。
# 3. 循环系统（心血管与血液）  
- 正常作用：参与代谢稳态和免疫调节，间接影响心血管健康。  
- 过多时影响：继发低睾酮 + 可能代谢综合征风险升高；少数报道与心血管事件相关（证据有限）。  
- 过少时影响：低泌乳素可能独立预测心血管不良事件风险增加（部分研究）。
# 4. 神经系统（脑与情绪）  
- 正常作用：参与应激反应、情绪调节；性高潮后升高有“满足/平静”作用。  
- 过多时影响：头痛（瘤压迫时明显）、抑郁、疲劳、注意力下降、性欲减退（独立于低睾酮）。  
- 过少时影响：可能免疫/应激反应减弱，情绪影响不典型。
# 5. 表皮系统（皮肤与毛发）  
- 正常作用：无直接显著作用。  
- 过多时影响：可能出现男性乳房发育（gynecomastia）、溢乳（galactorrhea，罕见但特征性）。  
- 过少时影响：无明显变化。
# 6. 免疫系统（含淋巴系统）  
- 正常作用：增强淋巴细胞增殖、抗体产生、免疫调节（作为细胞因子样激素）。  
- 过多时影响：可能免疫失调（慢性高水平抑制某些应答）。  
- 过少时影响：免疫功能低下、感染易感性增加、伤口愈合慢（初步证据）。
# 7. 消化系统（含代谢）  
- 正常作用：参与代谢稳态、胰岛素敏感性调节。  
- 过多时影响：体重增加、代谢综合征风险升高（与低睾酮相关）。  
- 过少时影响：代谢影响有限。
# 8. 泌尿系统  
- 正常作用：间接支持前列腺/精囊功能。  
- 过多时影响：前列腺功能相对减弱（低睾酮影响）。  
- 过少时影响：精囊体积减小、射精量减少。
# 9. 生殖系统  
- 正常作用：适量时促进Leydig细胞功能、睾酮分泌、精子生成、维持性欲；性活动后短暂升高有调节作用。  
- 过多时影响：性欲减退、勃起功能障碍（ED）、早泄（部分病例）、精子减少/质量下降、不育、睾丸萎缩、男性乳房发育、溢乳。  
- 过少时影响：精囊体积减小、射精量减少、早泄、勃起功能障碍（部分研究）。
# 10. 内分泌系统  
- 正常作用：受多巴胺抑制，受TRH/雌激素刺激；与性腺轴互动。  
- 过多时影响：强烈抑制GnRH脉冲 → LH/FSH/睾酮下降，形成继发性腺功能减退；可能干扰甲状腺等。  
- 过少时影响：反馈失调，但临床少见。
# 11. 呼吸系统  
- 正常作用：无直接作用。  
- 过多/过少时影响：间接通过疲劳/代谢影响耐力，无明显直接症状。
# 12. 淋巴系统（与免疫高度重叠）  
- 正常作用：促进淋巴细胞活性。  
- 过多/过少时影响：类似免疫系统，调节失衡。
# 注意事项  
- 男性高泌乳素血症发病隐匿，常以性功能障碍/不育首诊；轻度升高（<50 ng/mL）多为应激/药物/采样问题，需复查（晨起、空腹、安静状态抽血）。  
- 严重高PRL（>200 ng/mL）高度提示泌乳素瘤，需垂体MRI。  
- 治疗：多巴胺激动剂（如卡麦角林/溴隐亭）为首选，可使PRL下降、性功能恢复、瘤缩小；极低PRL罕见，通常不需干预。

- 皮质醇（Cortisol） 是成年男性体内主要的糖皮质激素，由肾上腺皮质束状带分泌，受下丘脑-垂体-肾上腺轴（HPA轴）调控。
- 它是经典的“应激激素”，具有明显的昼夜节律：早晨高峰（通常醒后30–60分钟达峰），夜间最低。
- 成年男性正常血清皮质醇水平（晨间8点左右）通常为 5–25 μg/dL（或 138–690 nmol/L，实验室参考值略有差异；唾液晨间约10–27 ng/mL）。每日产量约8–30 mg。
- 正常作用：帮助应对急性应激、维持血糖、血压、免疫平衡、代谢稳态、抗炎等。急性升高有益（fight-or-flight），但慢性高水平有害。  
- 过多（高皮质醇血症，hypercortisolism，常>25–30 μg/dL持续；最典型为库欣综合征Cushing's syndrome，由垂体瘤、肾上腺瘤、异位ACTH、外源糖皮质激素滥用等引起）导致分解代谢、免疫抑制、心血管风险等。  
- 过少（低皮质醇血症，hypocortisolism，常<5 μg/dL晨间；常见于艾迪生病Addison's disease、继发性肾上腺功能不全、垂体问题、长期应激后耗竭）导致低血压、疲劳、低血糖等。  
# 1. 骨骼系统  
- 正常作用：适量维持骨代谢平衡（间接支持）。  
- 过多时影响：抑制骨形成、促进骨吸收 → 骨密度下降、骨质疏松、骨折风险显著增加（库欣典型特征）。  
- 过少时影响：通常无直接骨丢失；但整体代谢紊乱可能间接影响。
# 2. 肌肉系统  
- 正常作用：急性应激时促进蛋白分解供能。  
- 过多时影响：蛋白分解加速 → 肌肉萎缩、近端肌无力（尤其是四肢）、易疲劳。  
- 过少时影响：肌肉力量下降、疲劳加重（与低能量、低血压相关）。
# 3. 循环系统（心血管与血液）  
- 正常作用：维持血压、心血管张力；急性升高增加心率、血压。  
- 过多时影响：高血压（钠潴留+血管敏感性增加）、心血管疾病风险升高（动脉硬化、冠心病）、液体潴留。  
- 过少时影响：低血压（甚至休克）、体位性低血压、心输出量下降。
# 4. 神经系统（脑与情绪）  
- 正常作用：调节应激反应、情绪、认知；影响海马、前额叶等。  
- 过多时影响：焦虑、抑郁、易激惹、记忆力下降、注意力缺陷、睡眠障碍；长期可致海马萎缩。  
- 过少时影响：疲劳、脑雾、抑郁、认知迟钝（Addison病常见）。
# 5. 表皮系统（皮肤与毛发）  
- 正常作用：适度抗炎、调节愈合。  
- 过多时影响：皮肤变薄、易瘀青、紫纹（striae，紫红色条纹常见于腹部/大腿）、痤疮、伤口愈合慢、面部潮红、多毛（hirsutism）。  
- 过少时影响：皮肤色素沉着（Addison病特征：口腔/手掌/瘢痕加深）、干燥。
# 6. 免疫系统（含淋巴系统）  
- 正常作用：急性升高抑制过度炎症、调节免疫。  
- 过多时影响：强烈免疫抑制 → 感染易感性增加、疫苗反应差、伤口愈合慢、慢性炎症失调。  
- 过少时影响：免疫功能相对活跃，但整体虚弱；易疲劳/感染（但不如高水平明显）。
# 7. 消化系统（含代谢）  
- 正常作用：促进糖异生、脂肪分解、抑制非必需消化。  
- 过多时影响：腹部/内脏脂肪堆积（“水牛背”、满月脸）、胰岛素抵抗 → 高血糖/2型糖尿病风险、食欲增加、消化减慢。  
- 过少时影响：低血糖、恶心、呕吐、食欲不振、体重减轻。
# 8. 泌尿系统  
- 正常作用：调节钠/水平衡、血压。  
- 过多时影响：钠潴留、水肿、高血压；肾功能负担增加。  
- 过少时影响：钠丢失、低血容量、低血压；严重时肾前性氮质血症。
# 9. 生殖系统  
- 正常作用：急性应激时抑制非必需生殖功能。  
- 过多时影响：抑制GnRH/LH/FSH → 睾酮下降、性欲减退、勃起功能障碍（ED）、精子质量下降、不育。  
- 过少时影响：性欲/功能相对保留，但整体疲劳影响性表现。
# 10. 内分泌系统  
- 正常作用：HPA轴核心；反馈抑制CRH/ACTH。  
- 过多时影响：强烈负反馈 → 继发性腺功能减退、其他轴失调（如甲状腺）。  
- 过少时影响：HPA轴耗竭、继发低ACTH/其他激素（如醛固酮在Addison）。
# 11. 呼吸系统  
- 正常作用：急性升高增加呼吸率、氧利用。  
- 过多时影响：间接通过高血压/肥胖增加睡眠呼吸暂停风险。  
- 过少时影响：耐力下降、疲劳加重，无明显直接呼吸道影响。
# 12. 淋巴系统（与免疫高度重叠）  
- 正常作用：调节淋巴细胞活性、抗炎。  
- 过多时影响：淋巴抑制、免疫低下。  
- 过少时影响：类似免疫，平衡失调。
# 注意事项  
- 慢性应激（如工作、生活压力）是最常见的高皮质醇原因，导致“肾上腺疲劳”样症状（虽非正式诊断，但临床常见：疲劳、体重增加、焦虑）。  
- 过多典型为库欣：满月脸、水牛背、紫纹、高血压、糖尿病、骨质疏松。  
- 过少典型为艾迪生：疲劳、低血压、色素沉着、低血糖、盐 craving。  
- 诊断需动态测试（如地塞米松抑制试验、ACTH刺激试验）；治疗针对病因（手术切除瘤、激素替代等）。

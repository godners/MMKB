# 1. 常见别名、物理性质、CAS编号、溶解度
- 均为蛋白类激素，冻干粉形式，需低温保存。
## LH
- 别名：促黄体生成素、间质细胞刺激素（ICSH，男性）
- 本质：糖蛋白激素（α、β亚基）
- 分子量：约30 kDa
- CAS：9002-67-9
- 水溶性：高度水溶
- 有机溶剂：不溶
## FSH
- 别名：促卵泡素
- 分子量：约35 kDa
- CAS：9002-72-6
- 水溶性：高度水溶
- 有机溶剂：不溶
## hCG
- 别名：人绒毛膜促性腺激素
- 分子量：约36–40 kDa
- CAS：9002-61-3
- 水溶性：高度水溶
# 2. 化学性质、光热稳定性
- 属糖蛋白，含唾液酸残基
- 对热敏感（>40℃活性下降）
- 强酸强碱环境易变性
- 紫外照射可破坏结构
- 冻干状态稳定，溶解后需冷藏
# 3. 生化特性
- 由垂体前叶分泌（hCG由胎盘）
- 作用于G蛋白偶联受体（LH-R、FSH-R）
- 通过cAMP-PKA通路发挥作用
- 受GnRH脉冲调控
- 受睾酮、雌二醇负反馈调节
# 4. 适应症、药理毒理
## LH/hCG
- 男性性腺功能减退
- 不育症
- 隐睾
- 外源性睾酮抑制后的恢复
## FSH
- 少精症
- 无精症（部分病例）
## 毒理
- 过量可致睾丸过度刺激
- 诱发多睾酮症状
- 罕见过敏反应
# 5. 药代动力学、起效时间
## hCG
- 半衰期：24–36小时
- 起效：注射后数小时睾酮升高
- 峰值：24–72小时
## FSH
- 半衰期：30–40小时
- 精子生成改善需 ≥3个月
# 6. 常见剂量、给药方式（成年男性）
## hCG
- 1000–3000 IU
- 每周2–3次
- 皮下注射或肌注
## FSH
- 75–150 IU
- 每周2–3次
- 皮下注射
# 7. 副作用、过量
- 水钠潴留
- 痤疮
- 情绪波动
- 乳腺增生（因E2升高）
- 过量可抑制内源轴（长期大剂量）
# 8. 同分异构体及类似物
- α亚基相同（TSH、LH、FSH、hCG共用）
- β亚基决定特异性
- 重组LH、rFSH
- hMG（含LH+FSH）
# 9. 人体内整体作用
- LH → 刺激Leydig细胞 → 合成睾酮
- FSH → 刺激Sertoli细胞 → 精子发生
- 协同维持男性生殖功能
# 10. 内分泌相关激素
- GnRH（上游）
- 睾酮（主要反馈）
- 雌二醇（由睾酮芳香化）
- 抑制素B（来自Sertoli细胞）
# 11. 对脂肪代谢
- 通过提升睾酮：
  - 减少内脏脂肪
  - 提升基础代谢
  - 抑制脂蛋白脂肪酶活性
# 12. 对血压
- 间接升高（高睾酮 → 红细胞增多 → 血黏度上升）
- 水钠潴留可轻度升压
# 13. 对消化系统（急性）
- 注射后偶见恶心
- 高睾酮可增加食欲
# 14. 对神经系统
- 提高性欲
- 增强攻击性
- 提升多巴胺活性
- 影响下丘脑体温调节
# 15. 对生殖系统
- 增加睾丸体积
- 促进精子生成
- 提升射精量
- 改善勃起质量（间接）
# 16. 对皮肤
- 痤疮
- 皮脂分泌增加
- 体毛增多
# 17. 过多或不足的治疗
## 不足（低促性腺激素性性腺功能减退）
- hCG + FSH
- GnRH脉冲治疗
## 过多（垂体腺瘤）
- 手术
- 放疗
- 多巴胺激动剂（若合并PRL升高）
## 与非孕产期女性相比：
- 男性更强调精子恢复
- 女性强调排卵恢复
# 18. 中医分析
- 八纲：肾虚（多属肾阳虚或肾精不足）
- 五行：属水（肾）
- 归经：肾经、肝经
- 阳虚表现：畏寒、性欲低
- 阴虚表现：潮热、盗汗

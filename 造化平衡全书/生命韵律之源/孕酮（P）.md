# 1. 基本信息
- 常见别名：黄体酮、孕甾酮
- 英文名：Progesterone
- CAS编号：57-83-0
- 分子式：C₂₁H₃₀O₂
- 分子量：314.46
- 白色或类白色结晶性粉末
- 熔点：126–131°C
- 脂溶性强，水溶性极低
- 溶解性
  - 水：几乎不溶
  - 乙醇、氯仿、乙醚：易溶
  - 油脂（如芝麻油）：高度溶解（常用于注射剂）
# 2. 化学性质与稳定性
- 属于甾体激素（C21类）
- 含有：
  - 酮基（C=O）
  - Δ4-3-酮结构（关键活性位点）
- 稳定性
  - 光稳定性较差：紫外线可导致降解
  - 热稳定性中等：高温长期可降解
  - 易被氧化（特别在空气中）
# 3. 生化特性
- 由胆固醇经多步反应生成
- 关键中间体：孕烯醇酮（Pregnenolone）
- 在男性中主要来源：
  - 睾丸（Leydig细胞）
  - 肾上腺皮质
- 是多种激素前体：
  - 皮质醇
  - 醛固酮
  - 睾酮
# 4. 适应症与药理毒理
- 医学用途（主要针对女性，但男性亦相关）
  - 激素替代治疗
  - 抗雌激素作用
  - 神经保护研究
- 男性相关作用
  - 抑制5α-还原酶（轻度）
  - 抑制过量雌激素效应
- 毒理
  - 急性毒性低
  - 长期高水平：抑制下丘脑-垂体-性腺轴（HPG轴）
# 5. 药代动力学
- 给药方式
  - 口服（微粉化制剂）
  - 肌注（油剂）
  - 经皮
- 特点
  - 口服首过效应明显
  - 生物利用度低（传统剂型）
- 起效时间
  - 口服：1–3小时
  - 注射：数小时内
- 半衰期：约 5–20小时（取决于制剂）
# 6. 常见剂量与给药
- 口服：50–200 mg/日（研究用途）
- 肌注：10–100 mg/次
# 7. 副作用与过量
- 常见副作用
  - 嗜睡（GABA作用）
  - 头晕
  - 抑郁倾向
  - 性欲下降
- 过量表现
  - 中枢抑制
  - 激素轴抑制
  - 水钠潴留
# 8. 同分异构体与类似物
- 结构类似物
  - 17α-羟孕酮
  - 地屈孕酮
  - 醋酸甲羟孕酮
- 特点
  - 更强口服活性
  - 更长半衰期
  - 部分具有雄激素或糖皮质激素活性
# 9. 在人体内整体作用（男性）
- 内分泌调节中枢前体
- 调节：
  - 神经系统
  - 性激素平衡
- 抗雌激素效应（弱）
- 抑制DHT生成（轻度）
# 10. 内分泌相关激素
- 孕酮是以下激素前体：
  - 皮质醇
  - 醛固酮
  - 雄激素（睾酮）
- 参与轴：
  - HPA轴（应激）
  - HPG轴（生殖）
# 11. 对脂肪代谢的作用
- 促进脂肪储存（轻度）
- 抑制脂解（相对雌激素较弱）
- 间接影响胰岛素敏感性
# 12. 对血压的作用
- 具有轻度抗醛固酮作用
- 促进钠排泄
- 轻度降低血压
# 13. 对消化系统（急性）
- 松弛平滑肌
- 减慢胃排空
- 可能导致：
  - 恶心
  - 腹胀
# 14. 神经系统作用机制
- 转化为神经类固醇（如Allopregnanolone）
- 激活：GABA_A受体（正向变构调节）
- 结果：
  - 镇静
  - 抗焦虑
  - 抗惊厥
# 15. 对生殖系统的作用（男性）
- 抑制LH分泌（负反馈）
- 降低睾酮（高剂量时）
- 抑制精子生成（长期高水平）
# 16. 对皮肤的作用
- 抑制皮脂分泌（轻度）
- 抗雄激素作用弱
- 可改善痤疮（理论上）
# 17. 过多或不足的治疗
- 孕酮过高（男性）：停药、调整内分泌
- 孕酮不足：通常无需治疗（男性）
- 对比女性
  - 女性常用：地屈孕酮、黄体酮制剂
  - 男性：无常规替代治疗
# 18. 中医分析
- 八纲辨证
  - 阴性
  - 偏寒（功能上镇静）
五行归经
  - 肾经（生殖）
  - 肝经（情志调节）
- 功能类比
  - 类似“滋阴安神”
  - 抑制“肝阳上亢”

# 皮质醇（COR）

- 另请参阅：**[皮质醇各系统作用](../生机流转见闻/皮质醇各系统作用.md)**

## 1. 常见别名、物理性质、CAS编号、溶解度

- 别名：氢化可的松（Hydrocortisone）、17-羟基皮质酮
- CAS编号：50-23-7
- 分子式：C₂₁H₃₀O₅
- 分子量：362.46 g/mol
- 白色或类白色结晶性粉末
- 熔点：约220–225°C
- 脂溶性较强，具甾体结构
- 溶解度：
  + 水中：极微溶（约0.28 mg/mL）
  + 有机溶剂：易溶于乙醇、丙酮、氯仿
  + 在脂质环境（细胞膜）中高度分布

## 2. 化学性质、光热稳定性

- 属于甾体激素（类固醇）
- 含多个羟基和酮基，具弱极性
- 对光和热较稳定，但在强酸/强碱条件下易降解
- 易被氧化或还原代谢（尤其在肝脏）

## 3. 生化特性

- 由肾上腺皮质束状带分泌
- 受**下丘脑-垂体-肾上腺轴（HPA轴）**调控
- 与**糖皮质激素受体（GR）**结合后调控基因转录
- 具有明显昼夜节律（清晨最高，夜间最低）

## 4. 适应症、药理毒理

- 适应症（药用氢化可的松）：
  + 肾上腺皮质功能不全（Addison病）
  + 过敏性疾病、哮喘
  + 自身免疫疾病（如系统性红斑狼疮）
  + 抗炎治疗
- 药理作用：
  + 抗炎（抑制NF-κB等通路）
  + 免疫抑制
  + 升高血糖（糖异生）
  + 蛋白质分解增强
- 毒理：
  + 长期高水平 → 类库欣综合征
  + 免疫抑制 → 感染风险增加

## 5. 药代动力学、起效时间

- 吸收：口服吸收良好
- 蛋白结合率：约90%（主要与CBG结合）
- 代谢：肝脏（CYP3A4参与）
- 半衰期：约1.5–2小时（生物效应更长）
- 起效时间：
  + 静脉：分钟级
  + 口服：1小时内

## 6. 常见剂量、给药方式

- 替代治疗：15–25 mg/天（分2–3次）
- 抗炎剂量：可达50–200 mg/天（依病情）
- 给药方式：口服、静脉注射、外用（皮肤）

## 7. 副作用、药物过量

- 常见副作用：
  + 高血糖
  + 向心性肥胖
  + 骨质疏松
  + 水钠潴留
  + 情绪波动（焦虑、抑郁）
- 过量：
  + 急性：高血糖、电解质紊乱
  + 慢性：库欣综合征表现

## 8. 同分异构体与类似物

- 可的松（Cortisone）：前体，需要转化为皮质醇
- 泼尼松 / 泼尼松龙：更强效，抗炎作用增强
- 地塞米松：强效长效糖皮质激素
- 生化差异：
  + 半衰期延长
  + GR亲和力不同
  + 矿物皮质激素活性不同

## 9. 在人体内的整体作用

- 调节应激反应核心激素
- 提供能量（葡萄糖动员）
- 抑制过度免疫反应
- 维持血压与血容量

## 10. 内分泌相关激素

- CRH（促肾上腺皮质激素释放激素）
- ACTH（促肾上腺皮质激素）
- 与胰岛素、胰高血糖素拮抗
- 抑制睾酮（高水平时）

## 11. 对脂肪代谢的作用

- 促进脂肪分解（外周）
- 同时促进内脏脂肪堆积
- 导致脂肪重新分布（满月脸、水牛背）

## 12. 对血压的作用

- 提高血压（增强血管对儿茶酚胺敏感性）
- 轻度矿物皮质激素作用 → 钠水潴留

## 13. 对消化系统的作用（急性）

- 增加胃酸分泌
- 抑制胃黏膜修复
- 可诱发或加重胃溃疡

## 14. 对神经系统的调节机制和作用

- 增强警觉性（短期）
- 长期升高 →
  + 海马体损伤
  + 记忆下降
  + 焦虑/抑郁
- 调节多巴胺与5-HT系统

## 15. 对生殖系统的作用

- 抑制GnRH → 降低LH/FSH
- 降低睾酮水平
- 可导致性欲下降、精子生成受抑

## 16. 对皮肤的作用

- 抑制胶原合成 → 皮肤变薄
- 抗炎（用于皮炎）
- 长期使用 → 皮肤萎缩、毛细血管扩张

## 17. 过多或不足时的治疗药物

- 过多（库欣综合征）：
  + 酮康唑（抑制合成）
  + 米托坦
  + 美替拉酮
- 不足（肾上腺功能不全）：
  + 氢化可的松（首选）
  + 氟氢可的松（补矿物皮质激素）
- 女性差异：
  + 女性更易出现骨质疏松
  + 剂量调整需考虑雌激素影响CBG

## 18. 中医八纲辨证与五行归经分析

- 属性：偏阳（功能性激活）
- 八纲：里证、虚实夹杂（应激时偏实）
- 五行归属：肾（主应激）、脾（代谢）
- 功能类比：
  + 类“肾阳”激发作用
  + 兼具“气机调动”功能

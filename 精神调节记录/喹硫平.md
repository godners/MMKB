# 1. 基本信息
- 常见别名：新斯的明、溴化新斯的明（Neostigmine bromide）、甲基硫酸新斯的明（Neostigmine methylsulfate）
- CAS编号：114-80-7（溴化物），51-60-5（甲基硫酸盐）
- 分子式：C₁₂H₁₉N₂O₂⁺（阳离子部分）
- 白色或类白色结晶性粉末
- 易吸湿
- 水溶性强
- 溶解度：
  - 水：极易溶
  - 乙醇：可溶
  - 非极性溶剂（如氯仿）：难溶
- 结构特点：
  - 季铵化合物（带正电）
  - 不易通过血脑屏障
# 2. 化学性质与稳定性
- 可逆性胆碱酯酶抑制剂（Carbamate类）
- 通过羧氨基化作用抑制乙酰胆碱酯酶（AChE）
- 对光较稳定
- 在碱性环境中易水解
- 高温可加速分解
# 3. 生化特性
- 抑制乙酰胆碱酯酶（AChE） → 增加突触间隙乙酰胆碱（ACh）
- 同时作用于：
  - N型受体（神经肌肉接头）
  - M型受体（副交感系统）
- 不进入中枢 → 主要为外周作用
# 4. 适应症与药理毒理
- 适应症
  - 重症肌无力
  - 术后肠麻痹、尿潴留
  - 麻醉后肌松药拮抗（如筒箭毒碱类）
- 药理作用
  - 增强骨骼肌收缩
  - 增强副交感神经活动
- 毒理
  - 过量 → 胆碱能危象
  - 与重症肌无力危象需鉴别
# 5. 药代动力学
- 给药方式：口服、皮下、肌注、静注
- 起效时间：
  - 静脉：5–10分钟
  - 肌注：10–30分钟
- 作用持续时间：2–4小时
- 代谢：
  - 部分肝代谢
  - 主要经肾排泄
- 半衰期：约1–2小时
# 6. 常见剂量
- 重症肌无力（口服）：15–60 mg，每日多次
- 肌松拮抗（静脉）：0.03–0.07 mg/kg
- 常与阿托品合用（防止M受体副作用）
# 7. 副作用与过量
- 常见副作用（M受体）
  - 流涎、流泪
  - 支气管分泌增加
  - 腹泻、腹痛
  - 心动过缓
- N受体
  - 肌束颤动
  - 肌无力（高剂量）
- 过量表现（胆碱能危象）
  - 瞳孔缩小
  - 呼吸困难
  - 严重肌无力
# 8. 同类及类似物
- 吡啶斯的明（Pyridostigmine）
  - 作用时间更长
  - 副作用更温和
- 依酚氯铵（Edrophonium）
  - 超短效（用于诊断）
- 毒扁豆碱（Physostigmine）
  - 可进入中枢（区别点）
# 9. 人体整体作用
- 增强胆碱能系统 → “副交感兴奋状态”
- 表现为：
  - 消化增强
  - 分泌增加
  - 心率下降
  - 骨骼肌功能改善（低剂量）
# 10. 内分泌影响
- 无直接激素作用
- 间接影响：
  - 通过迷走神经影响胰岛素分泌（轻微↑）
# 11. 对脂肪代谢
- 间接促进：胰岛素分泌↑ → 脂肪合成倾向↑
- 但临床影响较弱
# 12. 对血压
- 低剂量：轻度降压（迷走兴奋）
- 高剂量：可能出现低血压、或反射性波动
# 13. 对消化系统（急性）
- 胃肠蠕动↑
- 胃酸分泌↑
- 可导致：
  - 腹痛
  - 腹泻
# 14. 神经系统作用
- 外周：神经肌肉接头传递增强
- 中枢：几乎无作用（不透BBB）
# 15. 生殖系统作用
- 间接作用：
  - 增强副交感：可能促进阴茎勃起（理论上）
- 临床上很少用于性功能
# 16. 对皮肤作用
- 出汗增加（胆碱能）
- 皮肤潮湿
# 17. 过量/不足的处理
- 过量（胆碱能危象）
  - 阿托品（M受体拮抗）
  - 必要时气道支持
- 不足（肌无力）
  - 增加剂量
  - 或换用长效药（吡啶斯的明）
# 18. 中医分析
- 八纲辨证：
  - 属“里证、虚中夹实”
  - 表现为“气虚+痰湿阻滞（肌无力）”
- 五行归经：
  - 脾（肌肉）
  - 肾（精气）
  - 肝（筋）
- 中医类比作用：
  - 类似“补气健脾 + 通络”
  - 同时有“促津液外泄”（汗、涎）
# 总结（关键点）
- 典型外周可逆性胆碱酯酶抑制剂
- 不入中枢 → 安全性较毒扁豆碱高
- 核心用途：
  - 重症肌无力
  - 麻醉拮抗
- 主要风险：
  - 胆碱能危象

# 1. 常见别名
- Pregabalin（通用名）
- Lyrica（最常见商品名）
- CAS号：148553-50-8
- 化学式：C₈H₁₇NO₂
- 分子量：159.23 g/mol
- 外观：白色至类白色结晶性粉末
- 熔点：194–196 °C
- 水溶性：高，可溶于水，约 33.0 g/L（25 °C）；可溶于甲醇、乙醇，几乎不溶于乙醚
- 稳定性：在干燥条件下稳定；受光或高温可部分分解
# 2. 化学性质
- 结构特征：γ-氨基酸衍生物，类似 GABA（γ-氨基丁酸）
- 主要作用机制：与 α₂δ 亚单位结合，抑制钙通道，降低神经元异常放电
- 反应性：化学性质较稳定，无强酸强碱条件下分解
- 同分异构体：无光学活性异构体（孕巴林为S-异构体的药用形式）
# 3. 生化特性
- 作用靶点：中枢神经系统的α₂δ-亚单位钙通道
- 代谢：主要通过肾脏排泄，肝脏代谢极少
- 生物利用度：高（约 90%），不受剂量显著影响
- 半衰期：约 6.3 小时
- 血浆蛋白结合率：< 10%
# 4. 皮肤作用
- 口服药物，一般不会直接作用于皮肤
- 极少数可出现皮疹、瘙痒等过敏反应
# 5. 常见类似物及比较
- Gabapentin（加巴喷丁）：与瑞普巴林类似的神经痛和癫痫治疗，α₂δ亚单位结合，口服生物利用度随剂量下降，瑞普巴林生物利用度高，半衰期略长
- Vigabatrin（维加巴林）：抗癫痫，抑制GABA转氨酶，有视网膜毒性风险
- Pregabalin异构体：S-异构体有效，R-异构体活性低，α₂δ结合能力差，药用只用S-异构体

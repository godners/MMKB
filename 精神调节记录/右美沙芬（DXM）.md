# 1. 基本信息
- 通用名：右美沙芬（Dextromethorphan, DXM）
- 常见别名：右旋甲氧吗啡喃、右甲吗啡喃
- CAS编号：125-71-3（游离碱）
- 分子式：C₁₈H₂₅NO
- 分子量：271.4
- 外观：白色或类白色结晶性粉末（多为氢溴酸盐）
- 熔点：约111–114°C（游离碱）
- pKa：约9.2
- logP：约3.3（脂溶性较高）
- 溶解度：
  - 水：游离碱微溶；氢溴酸盐易溶于水
  - 乙醇：易溶
  - 有机溶剂（氯仿、乙醚）：游离碱易溶
# 2. 化学性质与稳定性
- 吗啡喃类衍生物，但无典型阿片镇痛活性
- 稳定于常温避光干燥条件
- 强光下可缓慢分解
- 对热较稳定（常规药品储存条件下）
- 酸性环境稳定，强氧化剂可使其降解
# 3. 生化特性
- 主要机制：
        1. 抑制延髓咳嗽中枢
        2. 非竞争性NMDA受体拮抗剂
        3. σ₁受体激动剂
        4. 轻度5-HT再摄取抑制
- 代谢产物：右啡烷（Dextrorphan），其NMDA拮抗作用更强
# 4. 适应症与药理毒理
- 适应症：
  - 急慢性干咳（无痰或少痰）
- 药理特点：
  - 中枢性镇咳
  - 无镇痛作用
  - 治疗剂量下无明显呼吸抑制
- 毒理特点：
  - 大剂量可致解离样精神症状
  - 可引起血清素综合征（与SSRI、MAOI合用）
  - 依赖性低于阿片类，但滥用风险存在
# 5. 药代动力学
- 吸收：口服吸收良好
- 起效时间：15–30分钟
- 达峰时间：约2–3小时
- 半衰期：
  - 快代谢型（CYP2D6正常）：2–4小时
  - 慢代谢型：可达8–12小时
- 代谢酶：CYP2D6为主
- 排泄：肾脏
# 6. 常见剂量（成年男性）
- 单次：15–30 mg
- 每日最大剂量：120 mg
- 给药方式：口服（片剂、胶囊、糖浆）
# 7. 副作用与过量
- 常见副作用：
  - 头晕
  - 嗜睡
  - 恶心
  - 轻度心动过速
- 过量表现：
  - 共济失调
  - 解离感、幻觉
  - 血压升高
  - 高剂量可致昏迷
- 极高剂量可产生类似氯胺酮样NMDA拮抗症状。
# 8. 同分异构体与类似物
- 左美沙芬（Levometorphan）：具有强阿片镇痛作用（μ受体激动）
- 右啡烷（Dextrorphan）：主要活性代谢物，NMDA拮抗更强
- 生化差异核心：右旋体无阿片镇痛活性。
# 9. 在人体内整体作用
- 抑制咳嗽反射
- 轻度中枢抑制
- 高剂量产生意识改变
- 对呼吸中枢影响轻
# 10. 内分泌相关
- 常规剂量对睾酮、雌二醇、皮质醇影响极小
- 大剂量长期滥用可能影响下丘脑-垂体轴（证据有限）
# 11. 对脂肪代谢的作用
- 无直接代谢调节作用
- 长期滥用可能影响食欲及体重
# 12. 对血压的影响
- 治疗剂量：影响轻微
- 高剂量：可升高血压（交感兴奋）
# 13. 对消化系统（急性）
- 恶心
- 胃部不适
- 少数便秘
# 14. 对神经系统的调节机制
- NMDA拮抗 → 减少兴奋性神经传导
- σ₁受体激活 → 情绪调节
- 5-HT再摄取抑制 → 潜在抗抑郁样作用
- 高剂量 → 解离、幻觉
# 15. 对生殖系统的作用
- 治疗剂量影响极小
- 极高剂量可抑制性欲（中枢抑制）
# 16. 对皮肤的作用
- 偶见皮疹
- 极罕见过敏反应
# 17. 过多或不足的治疗
- 过量治疗：
  - 活性炭
  - 对症支持
  - 严重血清素综合征：苯二氮卓类
-   与女性非孕期用药原则基本一致，无明显性别差异。
# 18. 中医分析
- 八纲辨证：
  - 镇咳属“敛肺止咳”
  - 偏于“寒凉收敛”之象
- 五行归经：
  - 归肺经
  - 兼及心包（中枢神志影响）

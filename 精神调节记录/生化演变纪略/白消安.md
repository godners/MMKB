# 1. 基本信息
- 常见别名：白消安、布司芬、马利兰（Myleran）
- CAS编号：55-98-1
- 化学名称：1,4-丁二醇二甲磺酸酯
- 分子式：C₆H₁₄O₆S₂
- 白色结晶性粉末
- 熔点：114–118°C
- 脂溶性较高，挥发性低
- 溶解度
  - 水中：低溶（约0.1–0.2 mg/mL）
  - 有机溶剂：
  - 易溶于丙酮、氯仿
  - 可溶于乙醇
  - 在脂质环境中分布较好
# 2. 化学性质与稳定性
- 属于双功能烷化剂（烷基磺酸酯类）
- 可形成碳正离子中间体，与DNA碱基发生共价结合
- 稳定性：
  - 避光较稳定
  - 在水溶液中逐渐水解
  - 对热相对稳定
# 3. 生化特性
- 与DNA鸟嘌呤N7位点结合
- 形成DNA链间交联
- 抑制DNA复制和转录
- 对快速增殖细胞高度敏感（骨髓细胞最显著）
# 4. 适应症与药理毒理
- 适应症
  - 慢性粒细胞白血病（CML）
  - 骨髓移植预处理（清髓）
- 药理作用
  - 骨髓抑制（选择性强）
  - 抑制粒细胞生成
- 毒理
  - 主要毒性靶器官：骨髓、肺、性腺
  - 可导致不可逆骨髓抑制
# 5. 药代动力学
- 吸收：口服吸收良好（>80%）
- 分布：广泛分布，可进入脑脊液（有限）
- 代谢：肝脏（主要经谷胱甘肽结合）
- 消除半衰期：约2–3小时
- 起效时间
  - 骨髓抑制：数天至数周
# 6. 常见剂量与给药
- 口服：CML：0.06 mg/kg/日
- 静脉（移植预处理）：0.8 mg/kg 每6小时
# 7. 副作用与过量
- 常见副作用
  - 骨髓抑制（剂量限制性）
  - 血小板减少
  - 肺纤维化（“白消安肺”）
  - 色素沉着（类似Addison样）
- 过量
  - 严重骨髓衰竭
  - 中枢神经毒性（惊厥）
# 8. 同分异构体与类似物
- 类似物
  - 马法兰（Melphalan）：芳香氮芥类
  - 环磷酰胺（Cyclophosphamide）：前药型烷化剂
- 生化差异
  - 白消安更偏向骨髓特异毒性
  - 环磷酰胺作用更广
# 9. 人体整体作用
- 强烈抑制造血系统
- 抑制免疫系统
- 用于“清空骨髓”以便移植
# 10. 内分泌相关
- 抑制性腺功能
- 降低：
  - 睾酮
  - 精子生成
# 11. 对脂代谢
- 无直接显著作用
- 间接影响（因激素下降）可能导致：脂肪比例上升
# 12. 对血压
- 无直接作用
- 严重骨髓抑制时可间接影响循环
# 13. 消化系统作用（急性）
- 恶心、呕吐
- 黏膜损伤（高剂量）
# 14. 神经系统作用
- 高剂量可致：惊厥（需预防性抗癫痫）
- 机制：可能与GABA系统干扰有关
# 15. 生殖系统作用
- 极强抑精作用（常不可逆）
- 导致：
  - 无精症
  - 睾丸萎缩
# 16. 对皮肤作用
- 色素沉着（“化疗肤色”）
- 长期使用可出现皮肤干燥
# 17. 过多或不足的处理
- 过量处理
  - 支持治疗
  - 造血干细胞移植
  - 抗惊厥药（如苯妥英）
- 对比女性
  - 女性更易出现：卵巢衰竭
  - 男性以精子生成损伤更显著
# 18. 中医分析
- 八纲辨证：毒热伤血、阴虚内损
- 五行归经：肝（血）、肾（精）
- 中医表现：精亏、血虚、虚热
# 总结核心要点
- 白消安是高度选择性骨髓毒性烷化剂
- 强烈抑制造血系统
- 明显损伤生殖功能（尤其男性）
- 用于骨髓移植前“清髓”不可替代

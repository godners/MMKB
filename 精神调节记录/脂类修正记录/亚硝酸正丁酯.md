# 1. 常见别名、物理性质、CAS、溶解度
- 别名：正丁基亚硝酸酯、Butyl nitrite、n-Butyl nitrite
- CAS号：544-16-1
- 分子式：C₄H₉NO₂
- 分子量：103.12 g/mol
- 淡黄色透明挥发性液体
- 具有特殊果香样气味
- 沸点：约78–79℃
- 密度：约0.87 g/cm³（20℃）
- 易挥发、易燃
- 溶解度：
  - 水中：微溶（易水解）
  - 有机溶剂：易溶于乙醇、乙醚、氯仿、脂类溶剂
# 2. 化学性质、光热稳定性
- 属于脂肪族亚硝酸酯
- 遇热、光照易分解
- 在酸性或碱性条件下易水解生成丁醇和亚硝酸
- 易氧化
- 对光敏感，应避光保存
# 3. 生化特性
- 在体内迅速释放一氧化氮（NO）
- 激活鸟苷酸环化酶 → 增加cGMP
- 引起平滑肌快速松弛
# 4. 适应症、药理毒理
- 历史适应症（目前已基本淘汰）：
  - 心绞痛急性发作
- 药理作用：
  - 强烈血管扩张
  - 冠状动脉扩张
  - 静脉扩张为主
- 毒理：
  - 高铁血红蛋白血症
  - 严重低血压
  - 缺氧
  - 中枢抑制
# 5. 药代动力学、起效时间
- 吸入给药
- 起效时间：10–30秒
- 作用持续：3–5分钟
- 肝脏代谢
- 代谢产物经尿排出
# 6. 常见剂量、给药方式
- 非正规医疗用途物质，多为吸入滥用
- 吸入蒸气
- 无标准医疗剂量
# 7. 副作用、过量
- 常见副作用：
  - 头痛
  - 面部潮红
  - 心动过速
  - 低血压
  - 眩晕
- 过量表现：
  - 严重低血压
  - 紫绀（高铁血红蛋白血症）
  - 呼吸困难
  - 意识丧失
- 治疗：
  - 亚甲蓝（治疗高铁血红蛋白血症）
  - 氧疗
  - 补液升压
# 8. 同分异构体及类似物
- 异构体：
  - 异丁基亚硝酸酯
  - 仲丁基亚硝酸酯
- 类似物（“poppers”类）：
  - 亚硝酸异戊酯
  - 亚硝酸戊酯
  - 亚硝酸环己酯
- 生化特性均为：
  - 快速释放NO
  - 强烈血管扩张
# 9. 在人体内整体作用
- 外周血管扩张
- 静脉回流减少
- 心输出量变化
- 反射性心动过速
# 10. 内分泌相关激素
- 不直接影响激素分泌
- 可通过血流动力学改变间接影响交感-肾上腺系统
# 11. 对脂肪代谢作用
- 无直接调节作用
- 长期滥用可能造成氧化应激增加
# 12. 对血压作用
- 显著降低收缩压和舒张压
- 可诱发晕厥
# 13. 对消化系统（急性）
- 恶心
- 呕吐
- 腹部不适
# 14. 对神经系统
- 头痛（脑血管扩张）
- 短暂欣快感
- 可能诱发晕厥
- 重度可导致中枢抑制
# 15. 对生殖系统
- 肛门括约肌松弛
- 增强短暂性性兴奋感
- 但不提高睾酮
# 16. 对皮肤
- 面部潮红
- 接触性皮炎（液体接触）
# 17. 过多或不足时治疗
- 过量处理：
  - 亚甲蓝
  - 高流量吸氧
  - 血压支持治疗
- 与非孕女性对比：
  - 治疗原则相同
  - 女性更易发生低血压性晕厥
# 18. 中医八纲辨证与五行
- 性质类比：
  - 性温、走窜
  - 属“辛散走窜之品”
  - 易耗气伤阴
- 五行归类：归心经、肝经（血脉扩张）
- 八纲分析：
  - 表证
  - 热证
  - 易耗气
# 重要提示
- 亚硝酸正丁酯在多数国家不作为合法药物使用，常见于娱乐性吸入滥用。
- 其心血管风险对肥胖、高血压、脂肪肝男性尤为危险，可能诱发：
  - 严重低血压
  - 心律失常
  - 心肌缺血

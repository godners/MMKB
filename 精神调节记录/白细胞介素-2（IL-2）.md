# 1. 常见别名、物理性质、CAS、溶解度
- 别名：T 细胞生长因子（TCGF）、IL-2
- 分子性质：多肽细胞因子，约 15.5 kDa（133 aa）
- CAS（重组人 IL-2）：110590-84-0
- 物理性质：白色或类白色冻干粉，易吸湿
- 溶解度：
  - 水、缓冲液（PBS）中高度可溶
  - 有机溶剂中不溶
  - 对 pH、离子强度敏感
# 2. 学性质、光热稳定性
- 化学性质：蛋白质，多二硫键，易变性
- 热稳定性：不耐热，高温易失活
- 光稳定性：对紫外线敏感
- 储存：2–8 ℃ 避光，复溶后需冷藏短期使用
# 3. 生化特性
- 主要由CD4⁺ T 细胞分泌
- 与 IL-2 受体（CD25/CD122/CD132）结合
- 激活 JAK-STAT、PI3K-AKT、MAPK 通路
- 是T 细胞克隆扩增的核心因子
# 4. 适应症、药理毒理
药理作用
- 促进 T 细胞、NK 细胞 增殖和活化
- 增强细胞毒性免疫
- 维持 Treg 的外周免疫耐受（低剂量）
临床适应症（重组 IL-2）
- 转移性肾细胞癌
- 转移性黑色素瘤
- 部分免疫缺陷、实验性自身免疫调控
毒理特点
- 剂量相关毒性明显
- 主要限制因素：毛细血管渗漏综合征（CLS）

# 5. 药代动力学、起效时间
- 给药途径：静脉 / 皮下
- 半衰期：
  - 静脉：约 5–10 分钟
  - 皮下：数小时（吸收缓慢）
- 起效时间：数小时内诱导淋巴细胞活化
- 清除：肾脏 + 受体介导内吞
# 6. 常见剂量、给药方式
- 高剂量方案（肿瘤）
  - 600,000–720,000 IU/kg
  - 静脉推注，每 8 小时一次
- 低剂量免疫调节：皮下注射，10⁵–10⁶ IU/日
- 需在重症监护或严格监测下使用
# 7. 副作用、药物过量
- 常见副作用
  - 发热、寒战
  - 低血压、心动过速
  - 水钠潴留、外周水肿
  - 肝酶升高
- 严重不良反应
  - 毛细血管渗漏综合征（CLS）
  - 急性肾损伤
  - 心律失常、肺水肿
# 8. 同分异构体与类似物
- 无同分异构体（为生物多肽）
- 类似细胞因子：
  - IL-7：维持 T 细胞存活
  - IL-15：NK 与 CD8⁺ 记忆 T 细胞
  - IL-21：B 细胞与 Tfh 功能增强
- 低剂量 IL-2：偏向激活 Treg
# 9. 在人体内的整体作用
- 推动细胞免疫从激活 → 扩增 → 调控全过程
- 高剂量：免疫攻击性增强
- 低剂量：免疫耐受、抗炎
# 10. 内分泌相关激素
- 抑制 HPA 轴（高剂量时）
- 可间接降低 睾酮（应激 + 炎症）
- 诱导 皮质醇、儿茶酚胺 升高（继发）
# 11.  对脂肪代谢的作用
- 炎症状态下：
  - 增加脂解
  - 抑制脂肪生成
- 长期高剂量 → 体重下降、负氮平衡
# 12.  对血压的作用
- 显著降低血压（剂量相关）
- 机制：
  - 血管通透性↑
  - 一氧化氮释放↑
- 临床常需升压药支持
# 13. 对消化系统的急性作用
- 恶心、呕吐、腹泻
- 肝功能异常（转氨酶升高）
- 肠黏膜通透性↑（炎症反应）
# 14. 对神经系统的调节机制
- 通过炎症因子网络间接作用中枢
- 可致：
  - 嗜睡、认知迟缓
  - 情绪低落（炎症性抑郁样反应）
- 不直接通过血脑屏障
# 15. 对生殖系统的作用
- 抑制性腺轴（暂时性）
- 可降低精子发生效率（高剂量、短期）
- 停药后多可恢复
# 16. 对皮肤的作用
- 毛细血管扩张
- 皮疹、红斑
- 注射部位硬结（皮下）
# 17 过多或不足的治疗
- IL-2 过多（炎症/毒性）
  - 对症支持（液体管理、升压药）
  - 糖皮质激素（权衡免疫抑制）
- IL-2 不足（免疫低下）
  - 低剂量 IL-2（研究/特定适应症）
- 与非孕产期女性相比：
  - 男性更关注心血管与体液潴留风险
  - 女性更需关注自身免疫背景
# 18. 中医八纲辨证与五行归经
- 八纲：
  - 多属 里证、实热、阳亢（高剂量状态）
  - 低剂量调节可偏 扶正祛邪
- 五行归经：
  - 心（神明、免疫）
  - 脾（气血生化、免疫细胞来源）
- 类比中医作用：
  - “鼓舞正气、动员卫气”
  - 过用则“气壅血溢、津液外泄”（类 CLS）

# 1. 常见别名、物理性质、CAS、溶解度
- 中文别名：异丁基亚硝酸酯
- 英文名：Isobutyl nitrite
- 分子式：C₄H₉ONO
- 分子量：103.12
- CAS 号：542-56-3
- 外观：淡黄色至无色透明液体
- 气味：甜味、刺激性明显（亚硝酸酯特征气味）
- 沸点：≈66–68 °C
- 密度（20 °C）：≈0.87 g/mL
- 挥发性：极高，易蒸发
- 溶解度
  - 水：微溶（<1 g/L）
  - 乙醇、乙醚、氯仿、丙酮：易溶
  - 脂溶性强，易透过生物膜
# 2. 化学性质、光热稳定性
- 化学本质：亚硝酸（HNO₂）与异丁醇形成的酯
- 稳定性：
  - 对光、热、空气 不稳定
  - 易分解生成 NO、NO₂、醇类
- 反应性：
  - 遇酸、热、紫外 → 快速分解
  - 可参与亚硝化反应
- 储存：需避光、低温、密闭、惰性气氛
# 3. 生化特性
- 在体内或组织中快速释放一氧化氮（NO）
- NO → 激活可溶性鸟苷酸环化酶（sGC）
- ↑ cGMP → 血管平滑肌舒张
- 高亲脂性 → 吸收极快，分布广
# 4. 适应症、药理毒理
- 医疗用途
  - 无正规临床适应症
  - 不作为合法治疗药物使用
- 药理毒理
  - 剧烈血管扩张，低血压
  - 反射性心动过速
- 特征性毒性：
  - 高铁血红蛋白血症
  - 抑制血红蛋白携氧能力
- 中枢毒性：
  - 头晕、意识障碍、晕厥
# 5. 药代动力学、起效时间
- 吸收：呼吸道、皮肤均可迅速吸收
- 起效时间：秒级～1分钟内
- 分布：心血管系统、中枢神经系统
- 代谢：
  - 肝脏酯酶水解
  - 转化为 NO、亚硝酸盐
- 消除半衰期：短（分钟级）
# 6. 剂量、给药方式（医学角度）
- 无安全或推荐剂量
- 不允许作为药物给药
- 任何暴露均应视为潜在中毒风险
# 7. 副作用、药物过量
- 常见不良反应
  - 剧烈头痛（血管性）
  - 面部潮红
  - 心悸、低血压
  - 恶心、呕吐
- 过量或暴露过强
  - 紫绀（皮肤/唇发青）
  - 呼吸困难
  - 意识丧失
  - 心律失常
  - 休克
# 8. 同分异构体与类似物
- 同类亚硝酸酯
  - 亚硝酸异戊酯（Isopentyl nitrite）
  - 亚硝酸正丁酯（n-Butyl nitrite）
  - 亚硝酸环己酯
- 共性生化特征
  - 释放 NO
  - 血管扩张
  - 可致高铁血红蛋白血症
# 9. 在人体内的整体作用
- 心血管：
  - 全身性血管扩张
  - 冠脉扩张但供氧下降（矛盾）
- 血液：
  - 血红蛋白氧化 → 组织缺氧
- 神经系统：
  - 短暂兴奋 → 抑制
# 10. 内分泌相关激素
- 不直接作用于激素受体
- 通过血压骤降 →
  - 反射性 ↑ 肾上腺素
  - ↑ 去甲肾上腺素
- 对性激素轴 无直接调节
# 11. 对脂肪代谢的作用
- 无直接脂代谢调控
- 低血压 → 交感兴奋 →
  - 短暂 ↑ 脂解（非生理调节）
# 12. 对血压的作用
- 强烈、迅速、不可控降压
- 可导致：
  - 体位性低血压
  - 晕厥
  - 心脑灌注不足
# 13. 对消化系统（急性）
- 内脏血管扩张
- 恶心、呕吐
- 肠蠕动短暂紊乱
# 14. 对神经系统的机制与作用
- NO–cGMP 通路
- 脑血管扩张 + 缺氧
- 表现为：
  - 眩晕
  - 头胀
  - 意识模糊
# 15. 对生殖系统
- 非激素性
- 平滑肌松弛
- 无生殖功能改善证据
- 长期暴露可能影响精子氧合环境（负面）
# 16. 对皮肤的作用
- 接触可致：
  - 红斑
  - 灼热感
  - 刺激性皮炎
- 易经皮吸收 → 全身毒性
# 17. 过多或不足的治疗（对比非孕产期女性）
- 中毒治疗首选：
  - 亚甲蓝（Methylene blue）：治疗高铁血红蛋白血症
- 男性与非孕产期女性用药原则一致
- 孕期女性需谨慎（NO/缺氧风险）
# 18. 中医八纲与五行归经（理论分析）
- 八纲：
  - 实证
  - 热证
  - 阳亢假象（表热里虚）
- 五行归经：
  - 心（血脉）
  - 肝（疏泄过度）
- 中医病机类比：
  - “走窜太速，耗气伤血”
  - 类似“暴脱之候”
# 总结一句
- 亚硝酸异丁酯是高风险、强血管活性的亚硝酸酯类毒物，不具备安全治疗价值，其核心危害在于“NO 过量 + 组织缺氧”。

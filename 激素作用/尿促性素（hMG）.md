# 1. 常见别名、物理性质、CAS编号、溶解度
- 常见别名：人绝经期促性腺激素（hMG）、Menotropins
- 来源：从绝经后女性尿液中提取纯化
- 成分：含等量左右的FSH（促卵泡激素）与LH（促黄体生成素）活性
- 分子本质：糖蛋白激素（α、β亚基结构）
- 分子量：约30 kDa（因糖基化程度不同略有差异）
- CAS号：9002-68-0（Menotropins）
- 物理性质：白色或类白色冻干粉末
- 溶解度：
  - 水：易溶（配制为注射液）
  - 有机溶剂：几乎不溶
  - 对pH敏感，通常在弱酸至中性环境稳定
# 2. 化学性质、光热稳定性
- 属蛋白质类激素
- 对高温敏感（>40℃活性下降）
- 光照可导致结构变性
- 避光、2–8℃冷藏保存
- 强酸强碱环境易变性
# 3. 生化特性
- 作用于FSH受体与LH受体（G蛋白偶联受体）
- 激活cAMP-PKA通路
- 促进性腺类固醇合成
- 促进支持细胞（Sertoli细胞）和Leydig细胞功能
# 4. 适应症、药理毒理（成年男性）
## 适应症（男性）
- 低促性腺激素性性腺功能减退症
- 精子生成障碍
- 不育症辅助治疗（常与hCG联合）
## 药理作用
- FSH → 促进精子发生
- LH → 促进睾酮分泌
## 毒理
- 毒性较低
- 过量可致激素过度刺激综合征（罕见于男性）
# 5. 药代动力学、起效时间
- 给药方式：皮下或肌肉注射
- 半衰期：约30–40小时
- 起效时间：
  - 睾酮升高：数天
  - 精子数量改善：需2–3个月以上
- 代谢：蛋白降解为氨基酸
- 排泄：肾脏代谢产物排出
# 6. 常见剂量、给药方式（成年男性）
- 常见剂量：75–150 IU FSH活性，2–3次/周
- 常联合hCG 1000–2000 IU 2–3次/周
- 疗程：至少3–6个月
# 7. 副作用、药物过量
## 常见副作用
- 注射部位疼痛
- 轻度水肿
- 乳房胀痛
- 情绪波动
## 过量
- 激素水平过高
- 乳腺发育（男性乳腺增生）
- 睾酮过度波动
# 8. 常见同类物质与类似物
- 尿源性FSH（urofollitropin）：仅含FSH
- 重组FSH（rFSH）：纯度更高
- hCG：主要模拟LH活性
- GnRH类似物：上游调控
## 生化差异：
- hMG：同时具FSH+LH活性
- rFSH：单纯刺激精子生成
# 9. 在人体内的整体作用
- 提升睾酮水平
- 促进精子生成
- 改善性欲与勃起功能（间接）
- 改善性腺体积
# 10. 内分泌相关激素
- ↑睾酮
- ↑抑制素B
- ↑雌二醇（轻度，来自睾酮芳香化）
- 反馈抑制GnRH
# 11. 对脂肪代谢的作用
- 睾酮升高 →
  - ↓脂肪堆积
  - ↑基础代谢率
  - 改善内脏脂肪比例
- 长期可改善代谢综合征相关指标。
# 12. 对血压的作用
- 间接作用：
  - 睾酮正常化 → 改善血管内皮功能
  - 过高睾酮可能轻度升压
- 总体影响较小。
# 13. 对消化系统的急性作用
- 可能轻度恶心
- 无明显胃肠毒性
# 14. 对神经系统的调节机制
- 通过睾酮：
  - ↑多巴胺活性
  - 改善情绪
  - 提高性欲
- 过量可致易怒、焦躁
# 15. 对生殖系统的作用
- 增大睾丸体积
- 恢复精子发生
- 提高精子浓度与活力
- 改善男性不育
# 16. 对皮肤的作用
- 睾酮升高 →
  - 皮脂分泌增加
  - 痤疮风险增加
  - 体毛增加
# 17. 过多或不足时的治疗对比
## 不足（男性）
- hCG + hMG联合
- 或GnRH脉冲治疗
## 女性（非孕产期）
- 用于促排卵
- 剂量更高
- 更易发生卵巢过度刺激综合征（OHSS）
# 18. 中医八纲辨证与五行归经分析
- 属“补肾填精”类
- 八纲：
  - 虚证
  - 肾阳虚或肾精不足
- 五行：属水
- 归经：肾经、肝经
- 对应证型
  - 肾精亏虚
  - 命门火衰
  - 精少不育

# 1. 基本信息
- 常见别名：Zydena（商品名）
- CAS编号：268203-93-6
- 分子式：C25H36N6O4S
- 分子量：516.66

- 外观：白色或类白色结晶性粉末
- 熔点：约158–162℃
- LogP：中等脂溶性（利于组织分布）
- 溶解性
  - 水：微溶
  - 乙醇：可溶
  - DMSO/有机溶剂：良好溶解
  - pH依赖性：弱碱性环境中溶解度增加
# 2. 化学性质与稳定性
- 属于吡唑并嘧啶衍生物
- 对光：中等稳定（避免强紫外）
- 对热：稳定
- 对酸碱：
  - 弱酸性较稳定
  - 强碱环境可缓慢降解
# 3. 生化特性
- 选择性PDE5抑制剂
- 抑制cGMP降解 → 平滑肌松弛
- 对PDE6抑制较弱（视觉副作用较少）
# 4. 适应症、药理毒理
- 适应症：勃起功能障碍（ED）
- 药理作用
  - 增强NO–cGMP通路
  - 提高海绵体血流
  - 改善勃起硬度与持续时间
- 毒理
  - LD50（动物）：较高，安全窗较宽
  - 心血管风险低于早期PDE5抑制剂
# 5. 药代动力学
- 起效时间：30–60 min
- 达峰时间（Tmax）：1–2 h
- 半衰期（t½）：约10–12 h
- 作用持续时间：可达12–24 h
- 代谢：肝脏CYP3A4为主
- 排泄：粪便为主，少量尿液
# 6. 常见剂量与给药方式
- 推荐剂量：100 mg
- 可调整范围：50–200 mg
- 给药方式：口服
- 性活动前30–60分钟服用
# 7. 副作用与过量
- 常见副作用
  - 面部潮红
  - 头痛
  - 鼻塞
  - 消化不良
- 较少见
  - 低血压
  - 心悸
- 过量表现
  - 严重血管扩张
  - 持续勃起（>4h）
  - 心血管不稳定
# 8. 同分异构体与类似物
- 类似PDE5抑制剂
  - 西地那非：起效快、半衰期短
  - 他达拉非：半衰期最长
  - 伐地那非：选择性更高
- 特点比较：乌地那非是平衡型（起效+持续时间），副作用较温和
# 9. 在人体内整体作用
- 外周血管扩张
- 阴茎海绵体血流增加
- 轻度全身血压下降
- 改善性反应质量
# 10. 内分泌影响
- 不直接影响：
  - 睾酮
  - LH / FSH
- 可能间接改善：
  - 性激素轴功能（通过性生活恢复）
# 11. 对脂肪代谢的作用
- 无直接作用
- 长期可能改善：
  - 代谢综合征（通过血流改善）
# 12. 对血压的作用
- 轻度降压
- 与硝酸酯类联用 → 严重低血压风险
- 对正常人影响较小
# 13. 对消化系统（急性）
- 胃食管反流（轻度）
- 胃部不适
- 肠道影响较轻
# 14. 神经系统作用机制
- 不直接作用中枢
- 外周血流改善 →
  - 增强性刺激传导
- 可能轻度头痛（血管扩张）
# 15. 生殖系统作用
- 增强勃起能力
- 不直接影响射精量
- 对精子无明显毒性
# 16. 对皮肤的作用
- 面部潮红（血管扩张）
- 极少数皮疹
# 17. 过量或不足的治疗
- 过量处理
  - 支持治疗
  - 避免硝酸酯类
  - 持续勃起 →
  - 注射去甲肾上腺素
- 不足（疗效差）
  - 可联合：
  - 睾酮替代（低雄激素者）
  - 心理干预
- 女性用药对比
  - 女性基本无适应症
  - 偶用于性唤起障碍（研究阶段）
# 18. 中医辨证分析
- 八纲
  - 病性：虚实夹杂
  - 本：肾阳虚
  - 标：气滞血瘀
- 五行归经：归肾、肝经
- 类比作用
  - 类似“温阳活血”
  - 改善“阳痿”“肾虚”
# 总体评价
- 乌地那非属于中长效PDE5抑制剂中的平衡型代表：
  - 起效适中
  - 持续时间较长
  - 副作用较轻
- 相比他达拉非更温和、相比西地那非更持久，适合对“时间窗口”和“耐受性”都有要求的人群。

# 1. 基本信息
- 别名：5-羟基-2-(羟甲基)-4H-吡喃-4-酮、Kojic acid
- 分子式：C₆H₆O₄
- 分子量：142.11 g/mol
- CAS号：501-30-4
- 物理性质：白色或浅黄色结晶粉末，吸湿性强
- 溶解性：
  - 水：约14 g/L（25°C），可随温度升高增溶
  - 乙醇：良溶
  - 甘油：良溶
  - 油脂或非极性溶剂：几乎不溶
# 2. 化学性质、光热稳定性
- 化学性质：
  - 酸性，pKa约为7.7
  - 可形成金属螯合物（如与铜、铁）
  - 易氧化生成醌类化合物
- 光热稳定性：
  - 对光敏感，容易褐变
  - 加热可分解，超过60°C长期储存不稳定
# 3. 生化特性
- 抑制酪氨酸酶（Tyrosinase）活性，从而抑制黑色素生成
- 能与铜离子结合，形成酶抑制复合物
- 属于酚类天然产物，可被肝脏代谢（主要为葡萄糖醛酸结合和硫酸结合）
# 4. 适应症与药理毒理
- 适应症：
  - 皮肤美白、治疗黄褐斑、雀斑、色素沉着
- 药理作用：
  - 抑制黑色素合成
  - 可能具有轻微的抗氧化作用
- 毒理：
  - 高浓度或长期使用可能引起皮肤刺激、红斑、瘙痒
  - 动物研究显示极高剂量口服可能对肝肾有毒，但临床使用浓度远低
# 5. 药代动力学
- 吸收：经皮肤吸收有限
- 分布：局部作用为主
- 代谢：肝脏葡萄糖醛酸化、硫酸化
- 排泄：主要随尿液排出
- 起效时间：外用一般2–4周可见轻微美白效果，持续使用8–12周效果明显
# 6. 常见剂量与给药方式
- 外用：一般浓度1–4%乳膏或精华
- 频率：每日1–2次
- 通常与其他美白剂（如维C衍生物）联合使用
# 7. 副作用与药物过量
- 局部副作用：刺激、红斑、脱屑、瘙痒
- 罕见过敏反应
- 口服过量（实验动物）：肝毒性、肾毒性
- 临床外用剂量下严重中毒极少
# 8. 同分异构体和类似物质
- 类似物：
  - 特丁基对苯二酚（TBCP）：更强抑制酪氨酸酶
  - 乙基/丙基衍生物：增加脂溶性，提高皮肤渗透
- 异构体：
  - 曲酸自身无手性中心，因此没有光学异构体
# 9. 在人体内的整体作用
- 主要局限于皮肤表层，减少黑色素生成
- 对全身激素代谢影响极低
- 无明显影响男性性腺功能、血压或脂代谢
# 10. 内分泌相关激素
- 不直接影响睾酮、DHEA、LH、FSH等激素
# 11. 对脂肪代谢作用
- 没有明确作用
# 12. 对血压作用
- 不影响血压
# 13. 对消化系统（急性）
- 外用剂量安全
- 高剂量口服：实验动物可引起消化道不适，但成人临床外用不涉及
# 14. 对神经系统作用
- 外用不显著
- 高剂量口服可能轻微中枢神经抑制（实验动物）
# 15. 对生殖系统作用
- 临床剂量外用安全
- 动物极高口服剂量未显示显著生殖毒性
# 16. 对皮肤作用
- 抑制黑色素形成
- 局部温和漂白
- 高浓度可能引起刺激
# 17. 过多或不足时常用治疗药物
- 曲酸不足不构成疾病
- 外用副作用或过敏可用低浓度乳膏或糖皮质激素短期缓解
- 对比非孕产期女性外用无明显差异
# 18. 中医八纲辨证与五行归经分析
- 八纲：寒、湿性凉，可清热解毒，适用于色斑、暗沉
- 五行归经：肝经（主疏肝调血，影响皮肤色泽）
- 功能类似于“清肝明目、祛斑美白”

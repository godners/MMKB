# 1. 常见别名、物理性质、CAS编号、溶解度
- 中文名称：谷胱甘肽、还原型谷胱甘肽、还原性谷胱甘肽
- 英文名称：Glutathione（reduced form），L-Glutathione
- 化学式：C₁₀H₁₇N₃O₆S
- 分子量：307.32
- CAS编号：70-18-8
- 外观：白色或类白色结晶性粉末，微臭，味苦。
- 溶解度：
  - 水中：约 10 g/100 mL（易溶）
  - 乙醇中：几乎不溶
  - 丙酮、乙醚：不溶
  - 生理溶液中可形成稳定溶液（pH 2–7稳定性较好）
# 2. 化学性质、光热稳定性
- 化学本质：三肽，由谷氨酸、半胱氨酸和甘氨酸组成；含有一个活性巯基（–SH）。
- 化学性质：
  - 强还原剂，可将氧化型谷胱甘肽（GSSG）还原为GSH。
  - 容易被氧化（空气中自氧化生成GSSG）。
- 光热稳定性：
  - 对光和高温敏感，在高温、紫外照射下迅速分解；
  - 在低温避光、酸性条件下稳定。
# 3. 生化特性
- 主要分布：肝、肾、红细胞、精液、睾丸组织中浓度较高。
- 生理功能：
  - 抗氧化防御：清除自由基、过氧化氢；
  - 维持细胞内氧化还原平衡；
  - 解毒作用：与药物或毒物结合生成可排泄的结合物；
  - 维持蛋白质巯基还原状态；
  - 支持DNA与蛋白质合成、细胞增殖与免疫调节。
# 4. 适应症、药理毒理
- 适应症：
  - 肝功能障碍（如药物性肝炎、脂肪肝、酒精性肝病）；
  - 放射性损伤；
  - 重金属或药物中毒；
  - 作为辅助美白、抗衰老药物（因减少黑色素生成）。
- 毒理特征：
  - 安全性极高，急性毒性低；
  - 无致畸、致癌或明显的免疫毒性。
# 5. 药代动力学、起效时间
- 给药后代谢：
  - 口服吸收率低（易在胃肠道分解为氨基酸）；
  - 注射或吸入给药可直接增加血浆GSH；
  - 肝脏为主要代谢场所。
- 起效时间：
  - 静脉注射：约10–30分钟见效；
  - 口服：约1–2小时后血浆抗氧化能力略升；
  - 皮肤注射或点滴用于美白：持续2–3周后有可见变化。
# 6. 常见剂量、给药方式
- 口服：250–1000 mg/日
- 静脉注射：600–2400 mg/日（分1–2次）
- 肌肉注射：通常400–800 mg/次
- 辅用：常与维生素C、α-硫辛酸联合，以增强抗氧化循环。
# 7. 副作用、药物过量
- 常见轻度反应：恶心、腹胀、轻度头晕；
- 注射剂：偶见过敏样反应（皮疹、呼吸困难、血压下降）；
- 长期过量：
  - 抑制黑色素生成（皮肤变白）；
  - 可能降低某些药物的疗效（例如抗生素或化疗药）。
# 8. 常见同分异构体和类似物质的生化特性
- 氧化型谷胱甘肽（GSSG）：两分子GSH通过二硫键连接，无还原活性；
- 乙酰化谷胱甘肽（N-acetylcysteine + GSH前体）：稳定性高，可提高细胞内GSH水平；
- 还原性维生素C、α-硫辛酸：功能类似，可循环再生GSH。
# 9. 在人体内的整体作用
- 中心作用：维持细胞氧化还原平衡；
- 肝脏：促进毒物代谢、保护肝细胞膜；
- 红细胞：维持血红蛋白的还原状态；
- 精液与睾丸：保护精子DNA免受氧化损伤，提高精子活力。
# 10. 内分泌相关的激素
- 可间接调节睾酮合成：通过减少睾丸氧化应激；
- 对皮质醇水平无显著影响；
- 支持胰岛素敏感性，改善胰岛β细胞功能。
# 11. 对脂肪代谢的作用
- 促进肝脏脂质代谢、减少脂肪肝；
- 提高线粒体β-氧化效率；
- 抑制脂质过氧化反应。
# 12. 对血压的作用
- 通过一氧化氮（NO）途径促进血管舒张；
- 有轻微降压作用；
- 改善内皮功能，减少氧化应激引起的血管硬化。
# 13. 对消化系统的作用（急性）
- 保护胃黏膜；
- 抗酒精及药物性肝胃损伤；
- 促进胆汁排泄，减轻肝胆区炎症反应。
# 14. 对神经系统的调节机制和作用
- 提高脑内谷胱甘肽浓度→减少神经元氧化损伤；
- 对帕金森病、阿尔茨海默病有保护作用；
- 促进谷氨酸代谢平衡；
- 改善精神疲劳与焦虑（通过抗氧化与谷氨酸调节）。
# 15. 对生殖系统的作用
- 增强精子膜稳定性；
- 提高精子活力与受精能力；
- 抗睾丸氧化应激；
- 临床用于男性不育、少精、弱精症辅助治疗。
# 16. 过多或不足时常用的治疗药物（对比非孕产期女性）
- GSH不足
  - 男性常用治疗：N-乙酰半胱氨酸（NAC）、α-硫辛酸、维生素C、S-腺苷甲硫氨酸
  - 非孕产期女性常用治疗：同上，可加用辅酶Q10、叶酸
- GSH过量（极罕见）：无需特异治疗，停药即可
# 17. 中医八纲辨证与五行归经分析
- 性味归经：甘、平；归肝、脾、肾经
- 功效类比：益气养阴、解毒护肝、润肤美颜
- 五行归属：属木（土生木之中介），与肝气调达、疏泄功能密切；
- 八纲辨证：
  - 属「气阴两虚、湿热内蕴」型患者适用；
  - 不宜用于「阳虚寒盛」体质者（可能进一步耗阳）。

# 1. 常见别名、理化性质、CAS、溶解度
- 英文名：Tirzepatide
- 商品名：Mounjaro®（降糖）、Zepbound®（减重）
- 结构类型：39个氨基酸的线性多肽，含C20脂肪酸侧链（利于白蛋白结合）
- 分子式：C₂₂₅H₃₄₈N₄₈O₆₈
- 相对分子质量：约 4813 g/mol
- CAS号：2023788-19-2
- 物理性质：白色至类白色冻干粉或澄清注射液
- 溶解度：
  - 水：易溶（作为注射液使用）
  - 有机溶剂：在甲醇、乙醇中有限溶解；对非极性溶剂几乎不溶
  - 生理条件下呈高度亲水性蛋白质分子
# 2. 化学性质、光热稳定性
- 多肽类，易受蛋白酶水解
- 对强酸强碱敏感
- 对高温不稳定（需2–8℃冷藏）
- 避光保存
- 冻融次数应避免
# 3. 生化特性
- 双重激动剂：
  - GIP受体激动
  - GLP-1受体激动
- 促进胰岛β细胞葡萄糖依赖性胰岛素分泌
- 抑制胰高血糖素
- 延缓胃排空
- 作用时间长（每周一次）
# 4. 适应症、药理毒理
- 适应症（成年男性）：
  - 2型糖尿病
  - 肥胖或超重（伴代谢风险因素）
- 药理作用：
  - 降低HbA1c可达2%以上
  - 体重下降可达15–22%（高剂量）
- 毒理：
  - 啮齿类中观察到甲状腺C细胞肿瘤（人类未明确）
  - 禁用于MEN2或甲状腺髓样癌家族史
# 5. 药代动力学、起效时间
- 给药方式：皮下注射
- 生物利用度：约80%
- Tmax：8–72小时
- 半衰期：约5天
- 稳态时间：4–5周
- 起效：
  - 食欲抑制：数日内
  - 血糖下降：1–2周
# 6. 常见剂量、给药方式
- 起始：2.5 mg 每周一次（4周）
- 维持：5–15 mg 每周一次
- 皮下注射（腹部、大腿、上臂）
# 7. 副作用、过量
- 常见：
  - 恶心、呕吐、腹泻
  - 便秘
  - 食欲下降
- 严重：
  - 胰腺炎
  - 胆结石
  - 低血糖（联合磺脲类/胰岛素时）
- 过量：
  - 严重胃肠反应
  - 低血糖
  - 主要对症支持治疗
# 8. 常见类似物及比较
- 司美格鲁肽（Semaglutide）：GLP-1单受体激动
- 利拉鲁肽（Liraglutide）：短效GLP-1
- 度拉糖肽（Dulaglutide）：长效GLP-1
- 区别：
  - 替尔泊肽为双受体（GIP+GLP-1）
  - 体重下降幅度通常更大
# 9. 在人体内整体作用
- 降低血糖
- 显著减少内脏脂肪
- 改善胰岛素抵抗
- 降低炎症标志物
- 改善代谢综合征
# 10. 内分泌相关激素影响
- ↑胰岛素（葡萄糖依赖）
- ↓胰高血糖素
- 可能轻度影响皮质醇节律（间接）
- 改善T/E2比例（通过减脂降低芳香化酶活性）
# 11. 对脂肪代谢作用
- 减少脂肪生成
- 提高脂肪动员
- 显著减少内脏脂肪
- 改善脂肪肝
# 12. 对血压作用
- 收缩压平均下降 4–8 mmHg
- 机制：减重 + 利钠 + 内皮功能改善
# 13. 对消化系统（急性）
- 延缓胃排空
- 降低食欲
- 早期易恶心
- 可能诱发胆囊问题
# 14. 对神经系统调节机制
- 作用于下丘脑食欲中枢
- 通过迷走神经通路调节饱腹信号
- 改善奖赏系统对高热量食物的反应
# 15. 对生殖系统
- 减脂后可改善男性睾酮水平
- 可能改善勃起功能（间接）
- 无直接雄激素活性
# 16. 对皮肤
- 减重后改善黑棘皮病
- 少数可见注射部位反应
# 17. 过多或不足时常用治疗药物
- 过度减重或胃肠耐受差：
  - 调整剂量
  - 止吐药（如昂丹司琼）
- 低血糖：减少磺脲类或胰岛素
- 与非孕产期女性对比：
  - 女性更易出现胆囊问题
  - 男性更关注睾酮改善
# 18. 中医八纲辨证与五行归经分析
- 多用于“痰湿内阻”“脾虚湿盛”“肥胖型消渴”
- 归经类比：脾、胃、肝
- 五行属土（水湿代谢）
- 偏寒凉（抑制食欲、清内热）

# 1. 常见别名、物理性质、CAS编号、溶解度
- 常见别名：
  - 利多卡因：Lidocaine
  - 氯己定：Chlorhexidine
  - 配方常称“利多卡因氯己定喷雾剂”或“局麻消毒喷雾剂”
- CAS编号：
  - 利多卡因：137-58-6
  - 氯己定：55-56-1
- 物理性质：
  - 利多卡因：无色至微黄色油状液体，微苦，pKa≈7.9
  - 氯己定：棕红色结晶或棕色粉末，易溶于水
- 溶解度：
  - 利多卡因：水中溶解度约1g/100mL（25℃）；乙醇可溶
  - 氯己定：水溶性好，乙醇和丙二醇中也可溶；对有机溶剂溶解有限
# 2. 化学性质、光热稳定性
- 利多卡因：
  - 化学性质稳定，在中性或弱碱性条件下较稳定，强酸可水解
  - 光敏性低，但高温或光照长时间会缓慢降解
- 氯己定：
  - 强碱性条件下易分解，酸性条件较稳定
  - 对光和热较稳定，但高温（>60℃）可部分降解
# 3. 生化特性
- 利多卡因：
  - 可阻断神经细胞钠通道，抑制动作电位传播 → 局部麻醉
- 氯己定：
  - 阳离子表面活性剂，破坏细菌细胞膜，杀菌、抑菌广谱（革兰氏阳性/阴性菌及部分真菌）
# 4. 适应症、药理毒理
- 适应症：
  - 局部麻醉（皮肤、粘膜表面）
  - 消毒杀菌（手术部位、穿刺、注射前）
- 药理作用：
  - 利多卡因：阻断神经冲动 → 麻木、止痛
  - 氯己定：杀菌、减少感染风险
- 毒理：
  - 利多卡因：大剂量可引起中枢神经抑制（头晕、抽搐）、心血管抑制（低血压、心律不齐）
  - 氯己定：皮肤大面积使用可能引起刺激、过敏；口服/误入气管有毒性
# 5. 药代动力学、起效时间
- 利多卡因：
  - 起效：表面喷涂后约30–60秒
  - 持续时间：15–30分钟（皮肤表面）、局部注射可达1–2小时
  - 代谢：肝脏CYP450代谢，肾脏排泄
- 氯己定：
  - 起效：立即杀菌（数秒至数分钟）
  - 无明显系统性吸收，主要局部作用
# 6. 常见剂量、给药方式
- 喷雾剂浓度：
  - 利多卡因：1–5%
  - 氯己定：0.05–0.2%
- 给药方式：
  - 表面喷涂，避免眼、耳、呼吸道
  - 每次喷涂量依面积而定，一般1–2次即可达到局麻和消毒效果
# 7. 副作用、药物过量
- 局部：
  - 利多卡因：皮肤刺激、瘙痒、红斑
  - 氯己定：刺激、轻度烧灼感
- 系统性过量：
  - 利多卡因：中枢神经症状（头晕、震颤、抽搐）、心血管抑制
  - 氯己定：口服中毒罕见，但可导致低血压、呼吸抑制
- 注意事项：儿童、肝肾功能不全、广面积使用需谨慎
# 8. 常见同分异构体和类似物质
- 利多卡因类：布比卡因、罗哌卡因
- 氯己定类似物：苯扎氯铵、氯已定葡萄糖酸盐
- 生化特性类似：局部麻醉和杀菌作用，系统吸收差异显著
# 9. 在人体内的整体作用
- 快速局部麻醉 + 杀菌消毒
- 不明显影响全身生理功能（正常剂量下）
# 10. 内分泌相关激素
- 无直接作用，但利多卡因过量可能通过应激反应影响儿茶酚胺分泌
# 11. 对脂肪代谢的作用
- 无直接作用
# 12. 对血压的作用
- 正常剂量局部喷涂：基本无影响
- 过量系统吸收：可能轻度低血压、心律失常
# 13. 对消化系统的作用（急性）
- 误吸或误服：恶心、呕吐
- 局部喷涂不影响消化系统
# 14. 对神经系统的调节机制和作用
- 阻断钠通道 → 阻止疼痛信号传递
- 中枢系统过量吸收可引起抽搐或抑制
# 15. 对生殖系统的作用
- 局部喷涂用于男性会阴、阴茎等部位，可减轻疼痛或敏感，无系统性生殖影响
# 16. 对皮肤的作用
- 利多卡因：局部麻醉
- 氯己定：杀菌、消毒
- 长期使用可轻度干燥或刺激
# 17. 过多或不足时常用的治疗药物
- 利多卡因过量：静脉脂肪乳液（Lipid rescue）、支持治疗
- 氯己定过敏：抗组胺药或局部冲洗
- 对比非孕产期女性：局部剂量和面积相同，一般安全，但孕期需谨慎系统吸收
# 18. 中医八纲辨证与五行归经分析
- 八纲辨证：寒热平衡、表里、虚实 → 适合“湿热下注，局部疼痛、敏感”的情况
- 五行归经：
  - 利多卡因：通经止痛，偏肝经（肝主筋、调节经络疼痛）
  - 氯己定：清热解毒，偏心、肺经（皮肤感染防护）

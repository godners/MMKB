# 氢醌

## 1. 基本信息

- 中文名称：氢醌
- 英文名称：Hydroquinone
- 常见别名：对苯二酚（p-Dihydroxybenzene）、苯-1,4-二醇、HQ
- 分子式：C₆H₆O₂
- 分子量：110.11 g/mol
- CAS号：123-31-9
- 结构式：HO–C₆H₄–OH（1,4-二羟基苯）

## 2. 物理性质

- 性质：数值或描述
- 外观：白色针状晶体或结晶性粉末，暴露空气中易变暗
- 熔点：172–174 °C
- 沸点：285–287 °C（分解）
- 密度：1.33 g/cm³
- 折光率：1.63（25 °C）
- 蒸气压：极低（25 °C下 <1 Pa）
- 气味：微弱的芳香气味

## 3. 溶解性

- 水：7 g/L（约0.7%），稍溶，溶液呈弱酸性
- 乙醇：易溶
- 乙醚：易溶
- 丙酮：易溶
- 氯仿：可溶
- 苯：微溶
- 在极性有机溶剂中（如乙醇、DMSO、丙二醇）溶解性显著提高。
- 水溶液在光、热、空气作用下易被氧化成苯醌（p-Benzoquinone），颜色变黄至棕色。

## 4. 化学性质

- 属于芳香族二羟基化合物，还原性强。
- 主要反应：
  - 氧化反应 → 对苯醌（可逆体系：氢醌–醌对）
  - 醚化、酯化反应 → 生成氢醌单/双酯（如氢醌双丙酸酯）
  - 亲电取代反应：羟基为活化基团，优先在邻位取代。
- 稳定性：光、热、空气可加速氧化。
- 储存：需避光、密封、充氮或加入抗氧剂（如亚硫酸氢钠）保存。

## 5. 生化特性与代谢

- 可被酪氨酸酶氧化，与黑色素生成途径中的中间体竞争，抑制黑色素生成。
- 抑制黑色素形成机制：
  - 抑制酪氨酸酶活性；
  - 竞争性取代DOPA氧化过程；
  - 对黑色素细胞具有细胞毒性，导致部分可逆性脱色。
- 在体内代谢：主要经葡糖醛酸化和硫酸化代谢，在尿中排出。

## 6. 皮肤作用与风险

- 皮肤药理作用：抑制黑色素生成，用于治疗黄褐斑、雀斑、炎症后色素沉着。
- 常用浓度：外用制剂 2%–5%，高于4%需医师监督。
- 副作用：
  - 刺激、红斑、脱皮、接触性皮炎；
  - 长期或高浓度使用可能致外源性黑变病（ochronosis）；
  - 对空气氧化产物（醌类）过敏者禁用。
- 毒理学：
  - LD₅₀（大鼠，口服）= 320 mg/kg；
  - 潜在致突变性，但无充分人类致癌证据。

## 7. 常见同分异构体与类似物比较

物质：结构特征：特点 / 生化差异

- 间苯二酚（Resorcinol）
  - 1,3-二羟基苯
  - 抑制黑色素较弱
  - 用于皮肤去角质剂
- 邻苯二酚（Catechol）
  - 1,2-二羟基苯
  - 氧化性更强
  - 易成色素聚合物
- 特丁基对苯二酚（Tert-butyl hydroquinone, TBHQ）
  - 氢醌的叔丁基衍生物
  - 抗氧化剂
  - 用于食品和化妆品防腐
  - 皮肤作用弱
- 氢醌双丙酸酯（Hydroquinone dipropionate）
  - 酯化衍生物
  - 稳定性高
  - 缓释型美白剂
- 甲基氢醌（Methylhydroquinone）
  - 氢醌甲基取代物
  - 毒性略低
  - 抑黑作用仍显著

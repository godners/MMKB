# 1. 常见别名、物理性质、CAS、溶解度
- 化学组成：SiO₂
- 常见别名：Quartz
  - 白石英、透明石英
  - 水晶（宏观透明晶体）
  - 石英砂（工业形态）
- CAS 号：14808-60-7
- 晶系：三方晶系
- 硬度：莫氏硬度 7
- 密度：2.65 g/cm³
- 熔点：约 1710 °C
- 溶解度：
  - 水、乙醇、有机溶剂中 不溶
  - 强碱（NaOH、KOH，高温高压）中 极微溶
  - 氢氟酸中 可溶并被腐蚀
# 2. 化学性质、光热稳定性
- 化学稳定性极高
- 不被绝大多数酸碱侵蚀（HF 例外）
- 耐高温、耐紫外、耐辐射
- 光照、加热、长期存放 不发生结构变化
# 3. 生化特性
- 无生物活性
- 不参与任何酶反应、受体结合或代谢途径
- 属于生物惰性无机物
# 4. 适应症、药理毒理
- 非药用物质
- 毒理重点在粉尘吸入：游离结晶型二氧化硅 → 矽肺（硅肺）
- 消化道完整摄入 → 基本无毒
# 5. 药代动力学、起效时间
- 不吸收、不代谢、不分布、不产生药效
# 6. 常见剂量、给药方式
- 禁止作为口服或注射给药物质
# 7. 副作用、药物过量
- 口服整块/砂粒：基本无系统毒性
- 长期吸入粉尘：
  - 肺泡巨噬细胞损伤
  - 炎症 → 纤维化 → 不可逆肺功能下降
- 无“过量”概念，属职业暴露问题
# 8. 常见同分异构体与类似物质
## 同质多晶型（SiO₂）：
- 石英（α-Quartz）
- 方石英（Cristobalite）：肺毒性更强
- 鳞石英（Tridymite）
## 非晶态类似物：
- 非晶二氧化硅（气相白炭黑）：生物毒性 显著低于结晶型
# 9. 在人体内的整体作用
- 整块石英：
  - 机械性惰性
  - 不释放离子
- 微米级粉尘：
  - 激活 NLRP3 炎症小体
  - 慢性免疫损伤
# 10. 内分泌相关激素
- 无直接或间接影响
- 不干扰：雄激素、皮质醇、甲状腺激素
# 11. 对脂肪代谢的作用
- 无影响
# 12. 对血压的作用
- 无直接影响
- 矽肺晚期因缺氧可 继发肺动脉高压（间接）
# 13. 对消化系统的作用（急性）
- 不溶、不反应
- 大颗粒 → 机械性通过
- 无吸收、无刺激性
# 14. 对神经系统的调节机制
- 无，不穿越血脑屏障，不影响神经递质
# 15. 对生殖系统的作用
- 无直接作用
- 职业暴露严重矽肺 → 体力下降，继发性影响性功能
# 16. 对皮肤的作用
- 整块石英：安全
- 粉末长期摩擦 → 角化、干裂
- 不致敏、不致突变
# 17. 过多或不足时的治疗
- 石英不是营养素
- 暴露治疗：
  - 停止接触
  - 支气管扩张剂、抗纤维化药
- 与女性用药 无性别差异
# 18. 中医八纲与五行归经
- 性味：甘、淡、寒
- 八纲：清、镇、收
- 五行归经：入 肺经、胃经
- 古籍应用（白石英）：安神、镇惊、清热
- **现代中医已基本不用内服矿石**
# 总结一句
石英是一种高度稳定、对人体化学惰性的无机矿物，其唯一显著健康风险来自长期吸入结晶型粉尘，而非摄入或接触本体。

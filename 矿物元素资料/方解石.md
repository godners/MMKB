# 1. 常见别名、物理性质、CAS编号、溶解度
- 别名：冰洲石、石灰石晶体、碳酸钙晶型
- 现代学术名称：方解石（Calcite）
- 化学式：CaCO₃
- CAS号：471-34-1
- 晶系：三方晶系
- 颜色：无色、白色、灰白色，常因杂质呈黄色、红色、褐色
- 密度：2.69–2.71 g/cm³
- 莫氏硬度：3
- 解理：完全三组菱面体解理
- 溶解度：
  - 水：极微溶（约 0.013 g/L，25℃）
  - 有机溶剂：不溶
  - 无机酸：易溶于稀盐酸、硝酸（产生 CO₂）
# 2. 化学性质、光热稳定性
- 化学性质：
  - 与酸反应剧烈，放出二氧化碳
  - 高温（>825℃）分解为 CaO + CO₂
- 光稳定性：稳定
- 热稳定性：中等，高温分解
# 3. 生化特性
- 在体内可解离为 Ca²⁺ 和 CO₃²⁻ / HCO₃⁻
- Ca²⁺ 参与骨骼、神经传导、肌肉收缩、凝血过程
- 碳酸根参与体液缓冲体系
# 4. 适应症、药理毒理
- 中医用途（煅制后）：制成“煅石灰”“寒水石类替代品”
- 现代医学应用：
  - 作为碳酸钙补钙剂来源
  - 抗酸剂
- 毒理：
  - 本身低毒
  - 过量 → 高钙血症、结石风险
# 5. 药代动力学、起效时间
- 吸收：胃酸环境下溶解，吸收率约 20–40%
- 分布：骨骼、牙齿为主要储存部位
- 代谢/排泄：Ca²⁺ 经肾脏、肠道排泄
- 起效时间：
  - 抗酸作用：30–60 分钟
  - 补钙：数周显效
# 6. 常见剂量、给药方式（成人男性）
- 补钙：碳酸钙计：500–1000 mg Ca²⁺ /日
- 给药方式：口服（餐后吸收更佳）
# 7. 副作用、药物过量
- 常见副作用：便秘、腹胀
- 过量：高钙血症、肾结石、心律失常
# 8. 常见同分异构体和类似物质的生化特性
- 霰石（Aragonite）：
  - 同为 CaCO₃，不同晶型
  - 生物利用度略高
- 文石（Vaterite）：
  - 不稳定，易转化为方解石
# 9. 在人体内的整体作用
- 提供钙源
- 维持骨密度
- 稳定细胞膜电位
- 参与酸碱缓冲
# 10. 内分泌相关的激素
- 甲状旁腺激素（PTH）
- 维生素 D₃（骨化三醇）
- 降钙素
# 11. 对脂肪代谢的作用
- 间接作用：
  - 钙可抑制脂肪细胞脂肪生成
  - 高钙饮食与体脂降低相关
# 12. 对血压的作用
- 钙摄入充足：有助于降低轻度高血压
- 机制：影响血管平滑肌收缩
# 13. 对消化系统的作用（急性）
- 中和胃酸
- 可能引起腹胀、嗳气
- 长期大量 → 胃酸反跳
# 14. 对神经系统的调节机制和作用
- Ca²⁺ 调节神经递质释放
- 稳定神经肌肉兴奋性
- 过量 → 神经抑制、嗜睡
# 15. 对生殖系统的作用
- 精子鞭毛运动依赖 Ca²⁺
- 间接支持生殖功能
- 过量无增强作用
# 16. 对皮肤的作用
- 外用粉体：吸附、控油
- 内服：改善钙缺乏相关皮肤问题
# 17. 过多或不足时常用的治疗药物
- 钙不足：碳酸钙、枸橼酸钙
- 钙过多：呋塞米、补液
- 对比非孕产期女性：
  - 女性剂量略低
  - 骨质疏松更常联合维生素 D
# 18. 中医八纲辨证与五行归经分析
- 性味：甘、寒（煅后偏温）
- 归经：肺、胃、肾
- 八纲：清热、制酸、潜阳
- 五行：属土，主中焦稳定
